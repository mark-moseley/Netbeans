About PHP Development in the NetBeans IDE

The NetBeans IDE for PHP provides tools for programming in the PHP language.

The Project Creation wizard enables setting up new PHP projects and importing previously developed source files.

The PHP editor provides the following features:

  • Syntax highlighting
  • Showing occurrences
  • Code completion
  • Navigation

You can also run and debug PHP applications in the IDE in three modes:

  • Local web server
  • Remote web server with FTP connection
  • Script

Find more information about PHP development in the NetBeans IDE on the PHP Learning Trail.
