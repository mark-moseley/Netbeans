Recapturing Schema From a Database

See Also

When you modify your database, such as by adding or deleting fields or changing relationships, you may need to recapture the database schema so that the changes are also available to your project.

To recapture the database schema from the database:

  1. Make sure the database server is running.
  2. Locate the database schema node (  ). The current database schema can be found in the following locations:
    • in the Configuration Files node in the Projects window
    • in the conf folder in the src folder in the Files window
  3. Right-click the database schema node and select Recapture Schema from Database from the contextual menu.
  4. Enter the username and passwod, if necessary.
  5. Click OK.

The Recapturing Database Schema progress box opens and displays the recapturing progress, including the table and view information it is capturing. After recapturing the database schema, you can explore the structure of the new database schema by expanding the schema in the Projects or Files windows. You can also explore the recaptured schema and view the data in the Runtime window.
