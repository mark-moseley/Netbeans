Working with Maven Artifacts

See Also 

The project dependencies that are displayed under a project's Libraries node are Maven artifacts that are specified in the POM. Each Maven artifact has a unique coordinate (groupId, artifactId and version) that is used to identify the artifact in the POM. An artifact can also have dependencies that are specified in the artifact's POM.

You can see the details about individual artifacts in the Artifact Viewer. The Artifact Viewer displays details in the following tabs.

  • Basic. The Basic tab displays the artifacts coordinates and version details.
  • Project. The Project tab displays metadata contained in the artifact's POM such as the description of the artifact, the URL for the project and issue tracking and version control details.
  • Classpath. The Classpath tab displays the artifact's compilation, runtime and test dependencies.
  • Graph. The Graph tab displays a visualization of the artifact's primary and secondary dependencies.

The buttons in the toolbar of the Artifact Viewer enable you to perform the following tasks.

Button Description
Add as Dependency. Click this to open the Add Dependency dialog box and choose a project.
Checkout Sources. If the sources of the artifact are available in a repository, click this to open the Checkout Sources wizard and specify a location for a local repository for the sources.
Create Library. Click this to create a NetBeans library from the artifact. The new library will be available in the Library Manager.

To open the Artifact Viewer.

  • In the Projects window, right-click an artifact under the Libraries node and choose View Artifact Details.
  • In the Maven Repository Browser, right-click an artifact and choose View Details.

Visualizing Dependency Conflicts

The Graph tab of the Artifact Viewer enables you to easily visualize dependencies. The visualizer also provides tools to help you find and resolve potential problems resulting from dependency version conflicts. In the Graphs tab, artifacts with a version conflict are indicated by a red background. You can click on the suggestion icon to open a dialog box with a description of the conflict and options for resolving the conflict.

To open the Graphs tab of the Artifact Viewer.

  • Right-click the Libraries node and choose Show Dependency Graph.
  • Right-click in the pom.xml editor and choose Show Dependency Graph.
