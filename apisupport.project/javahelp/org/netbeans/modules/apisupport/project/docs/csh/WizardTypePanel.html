Wizard Wizard: Wizard Type

See Also 

The New Wizard wizard creates a new Wizard. You open the New Wizard wizard by right-clicking a module project, choosing New > File/Folder and selecting Wizard from the NetBeans Plug-in Module category.

In the Wizard Type page, you have to set the following:

  • Registration Type. Specifies where the user will be able to find the wizard in the IDE.
  • Wizard Steps. Specifies the number of panels that the wizard will contain.
