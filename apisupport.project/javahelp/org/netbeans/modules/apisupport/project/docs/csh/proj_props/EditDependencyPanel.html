Edit Module Dependency Dialog Box

See Also 

The Edit Module Dependency dialog box lets you customize a module dependency. New module dependencies will initially use a major release version and specification version as taken from the module’s manifest and be included in the classpath if there are any available packages. The new dependency will be selected so that you may use Edit to customize it immediately. If multiple dependencies were added, just one of them will be selected automatically.
