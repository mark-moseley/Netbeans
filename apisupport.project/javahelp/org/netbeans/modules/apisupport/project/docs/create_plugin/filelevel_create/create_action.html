Creating An Action

See Also

A J2SE library descriptor adds a new class library to the Library Manager of the user's IDE. Whenever the module containing the library descriptor is enabled, the library is present in the Library Manager.

To create an action:

  1. Create a module project.
  2. From the Projects window or Files window, right-click the project's node and choose New > File/Folder from the pop-up menu.
  3. Under Categories, select NetBeans Plug-in Modules. Under File Types, select J2SE Library Descriptor. Click Next.
  4. Select the library that you want to make available to the user's IDE. Alternatively, click Manage Libraries and use the Library Manager to add a new library that you want to make available in the module. Click Next.
  5. Type the library name in the Libraray Name edit box.
  6. Type the display name in the Display Name edit box.
  7. Select a package to house the library descriptor from the Package drop-down list.
  8. Click Finish.

    The IDE adds the library to the module. To see the library's source files, open the Files window and look in the release folder.
