Creating a J2SE Library Descriptor

See Also

A J2SE library descriptor is an XML file that, when registered in the layer.xml file, adds a new class library to the IDE's Library Manager. Whenever the module containing the library descriptor is enabled, the library is present in the Library Manager.

To create a J2SE Library descriptor:

  1. Create a module project.
  2. From the Projects window or Files window, right-click the project's node and choose New > File/Folder from the pop-up menu.
  3. Under Categories, select NetBeans Plug-in Modules. Under File Types, select J2SE Library Descriptor. Click Next.
  4. Select the library that you want to make available to the user's IDE. It can be a JAR file or a ZIP file. Alternatively, if the library is not available in the IDE, click Manage Libraries and use the Library Manager to add a new library that you want to make available in the module. Click Next.
  5. Type the library name in the Libraray Name edit box.
  6. Type the display name in the Display Name edit box.
  7. Select or type a package name to house the library descriptor in the Package drop-down list.
  8. Click Finish.

    The IDE adds the new class library to the module. A library descriptor (an XML file) is added to the package specified in the wizard. The layer.xml file registers the library descriptor. To see the library's archive files, open the Files window and look in the release/libs folder.
