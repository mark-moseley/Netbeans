Module Suite Project Properties Dialog Box: Basic Branding

See Also 

You open the Module Suite Project Properties dialog box by right-clicking a module suite project and choosing Properties.

The Basic Branding page specifies the look-and-feel of an application built on top of the NetBeans Platform. Set the following properties:

  • Use basic branding information for building. Specifies that you want to brand the application. When you select this checkbox, a folder called branding is added to the suite's folder structure.
  • Name. Specifies the name of the application's executable.
  • Title. Specifies the title that will appear in the application's titlebar.
  • Icon. Specifies an icon with a dimension of 48x48 pixels, that will be displayed in the application's About dialog box, which will be available under the Help menu. Note that this icon does not set the executable's icon.
