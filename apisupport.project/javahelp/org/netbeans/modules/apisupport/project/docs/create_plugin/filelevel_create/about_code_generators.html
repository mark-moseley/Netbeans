About Code Generators

See Also

A code generator is an action that is added to the Insert Code popup, which the user invokes when pressing Alt-Insert. Traditionally, code generators are only found in Java documents but, from NetBeans 6.1 onwards, they can also be found in all other types of documents.

Via the Code Generator SPI and the accompanying Code Generator wizard, you can quickly and easily extend the code generator popup for a specific MIME type with new entries.
