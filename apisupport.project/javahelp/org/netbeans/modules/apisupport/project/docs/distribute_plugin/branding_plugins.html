Branding an Application

See Also

Once the functionality of an application built on top of the NetBeans platform is complete, the appearance of the application still closely resembles the default NetBeans Platform. You can use the IDE to personalize your application. In particular, you can do the following:

  • Replace the launcher's name.
  • Replace the application's splash screen.
  • Define a progress bar.
  • Define the application's title bar.
  • Hide unwanted menu bars, menu items, toolbars, and toolbar buttons.

To brand an application:

  1. Right-click the module suite project in the Projects window and choose Properties.
  2. In the Project Properties dialog box, click Basic Branding. Do the following:
    • Click Use basic branding information for building.
    • Type the name of the launcher.
    • Type the title to be displayed in the application's title bar.
    • Choose the icon that will be displayed in the application's Help > About dialog box.
  3. In the Project Properties dialog box, click Splash Branding. Do the following:
    • Define the progress bar.
    • Choose a splash screen.
  4. Click OK to confirm your choices and exit the Project Properties dialog box.
