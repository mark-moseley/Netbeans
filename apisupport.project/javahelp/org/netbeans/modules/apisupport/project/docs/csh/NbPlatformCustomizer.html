NetBeans Platform Manager Dialog Box

See Also 

The NetBeans Platform Manager is a tool for registering different NetBeans platforms with the IDE. The dialog box lists all your registered NetBeans platforms in the left pane and lists the platform that the IDE currently deploys to as the Default.

You can open the dialog box by choosing Tools > NetBeans Platform Manager.

Using the NetBeans Platform Manager, you can:

  • Register a new platform.
  • View registered platform's available modules.
  • Register source code for a platform in the Sources tab.
  • Register Javadoc documentation for a platform in the Javadoc tab.

Once you have registered a platform, you can configure a module project to use that platform for deployment. Go to the project's Project Properties dialog box, select the Libraries node, and choose the appropriate platform in the NetBeans Platform drop-down list.
