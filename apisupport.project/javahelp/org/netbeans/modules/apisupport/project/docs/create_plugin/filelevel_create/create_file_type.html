Creating A File Type

See Also

To create a new file type:

  1. Right-click a module project and choose New > File Type.
  2. Follow the steps in the New File Type wizard.
  3. After completing the wizard, you can do one or more of the following:
    • Tweak the layer.xml file to change the icon or rearrange the default actions added by the New File Type wizard.
    • Add properties to the default property sheet used by the new file type.
    • Use the New Action wizard to add actions to the new file type's pop-up menu. When you do so, select Conditionally Enabled in the first page of the New Action wizard. In the second page, set the file type context menu item.
    • Use the New Wizard wizard to create a New File wizard that the user will use to create new files of the new type defined in the New File Type wizard.
    • Use the New Project Template wizard to add a sample project containing example files of the new file type to the New Project wizard.
    • Add advanced support for the file type, such as syntax highlighting, code completion, and a Visual Editor.
