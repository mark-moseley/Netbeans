Bundling a Private Update Center's URL

See Also

The New Update Center wizard lets you create a plug-in module that registers a private update center in the user's Update Center wizard. In effect, the plug-in module bundles the URL to the private update center so that the user does not need to register the URL manually.

To bundle a private update center's URL:

  1. Right-click a module project and choose New > File/Folder. In the New File wizard, choose Update Center from the NetBeans Module Development category. Click Next.
  2. In the Update Center Declaration page, you have to set the following:
    • URL to Update Descriptor. Specifies the URL to the autoupdate descriptor.

      You can let the IDE generate the autoupdate descriptor for you; for details, see Generating an Autoupdate Descriptor.

    • Display Name. Specifies the name that you would like the user to see in their Update Center wizard.

    Click Finish.
