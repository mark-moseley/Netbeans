Module Project Properties Dialog Box: Packaging

See Also 

You open the Module Project Properties dialog box by right-clicking a module project and choosing Properties.

The Packaging page lets you specify the NetBeans Platform, the module's dependencies, and the required tokens:

  • JAR File: Gives the actual JAR file that will be created for the module, in the form ${cluster}/${module.jar}.
  • Needs Restart on Install. Sets nbm.needs.restart in the nbproject/project.properties file.
  • Must be Installed Globally. Installs the NBM file in the NetBeans installation directory as well as in the NetBeans user directory. If not selected, the NBM file is installed only in the NetBeans user directory. Sets nbm.is.global in the nbproject/project.properties file.
  • License. Points to an optional license file for the NBM file. Sets license.file in the nbproject/project.properties file. May be an absolute or project-relative path. .
  • Home Page. Sets nbm.homepage in the nbproject/project.properties file.
  • Author. Sets nbm.module.author in the nbproject/project.properties file.
