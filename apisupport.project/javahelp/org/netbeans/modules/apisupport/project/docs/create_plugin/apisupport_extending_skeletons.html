About Extending Skeleton API Implementations

See Also

Once you have used the NetBeans API wizards, you extend the generated code by using the NetBeans APIs. Several tools are provided specifically to help you at this stage of your development cycle. Amongst them are the following:

  • Access to NetBeans sources and Javadoc. After you register the NetBeans sources and Javadoc, you can access them inside the IDE. This gives you a quick and easy reference to the API classes and methods that you are implementing.
  • Search facility for NetBeans APIs. If you know the class that you need to use, but not the module (API or non-API) to which it belongs, a search facility is provided to help you. Once you have identified the module, the IDE registers it in the module project's projext.xml file.
