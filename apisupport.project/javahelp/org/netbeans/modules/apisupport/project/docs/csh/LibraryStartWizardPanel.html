Library Wrapper Module Project: Select Library

See Also 

You use a library wrapper module project to put a library JAR file on a module's classpath and to export some or all of the JAR file's packages from the module as public packages.

The New Library Wrapper Module Project wizard creates a new library wrapper module project. You open the New Library Wrapper Module Project wizard by opening the New Project wizard and selecting the Library Wrapper Module Project template from the NetBeans Plug-in Modules category.

In the Select Library page, you have to set the following properties:

  • Library. Specifies an archive file that you want to wrap in the module.
  • License. Specifies a text file that is the license of the archive file.
