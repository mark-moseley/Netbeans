Module Suite Project Properties Dialog Box: Libraries

See Also 

You open the Module Suite Project Properties dialog box by right-clicking any module suite project and choosing Properties.

The Libraries page lets you specify the NetBeans Platform against which the module suite will be compiled and run. It also lets you specify the modules that belong to the suite project. You need to set the following properties:

  • Java Platform: Specifies the target JDK for building module projects or module suite projects.
  • NetBeans Platform: Specifies the target NetBeans platform against which the module suite is compiled and run. Available NetBeans platforms are those that have been registered in the NetBeans Platform Manager.
  • Platform Modules. Specifies the clusters that will be deployed together with the module suite project.
