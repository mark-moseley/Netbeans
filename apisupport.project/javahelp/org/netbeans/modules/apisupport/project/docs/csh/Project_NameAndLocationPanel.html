Project Template Wizard: Name and Location

See Also 

The New Project Template wizard creates a new Project Template. You open the New Project Template wizard by right-clicking a module project, choosing New > Other and selecting Project Template from the Module Development category.

In the Name and Location page, you have to set the following:

  • Template Name. Specifies the name of the new project template.
  • Display Name. Specifies the template's label. For example, you will see this label in the Plugin Manager. (Optional)
  • Category. Specifies the template's category. For example, this will enable the user to find the template more easily in the New Project wizard. (Optional)
  • Package. Specifies the name of the package where the classes will be housed.
