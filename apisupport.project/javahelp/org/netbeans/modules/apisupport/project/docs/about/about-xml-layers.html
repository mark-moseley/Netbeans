About XML Filesystem Layers

See Also

You use an XML filesystem layer as part of the installation system for your plug-in. The contents of the XML layer will be automatically included in the system filesystem when your plug-in is installed.

When you use a wizard to create a plug-in module, an XML layer file is automatically created for you. When you use a wizard to create an IDE extension, such as a new file type or action, the related entries in the XML layer file are automatically generated for you.
