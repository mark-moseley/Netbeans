J2SE Library Descriptor Wizard: Name and Location

See Also 

The New J2SE Library Descriptor wizard creates a new J2SE library descriptor. You open the New J2SE Library Descriptor wizard by right-clicking a module project, choosing New > File/Folder and selecting J2SE Library Descriptor from the NetBeans Plug-in Module category.

In the Name, Icon, and Location page, you have to set the following:

  • Library Name. Specifies the name of the new library descriptor.
  • Display Name. Specifies the descriptor's label.
  • Package. Specifies the name of the package where the library will be housed.
