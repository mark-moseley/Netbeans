Java SE Library Descriptor Wizard: Name and Location

See Also 

The New Java SE Library Descriptor wizard creates a new Java SE library descriptor. You open the New Java SE Library Descriptor wizard by right-clicking a module project, choosing New > Other and selecting Java SE Library Descriptor from the Module Development category.

In the Name, Icon, and Location page, you have to set the following:

  • Library Name. Specifies the name of the new library descriptor.
  • Display Name. Specifies the descriptor's label.
