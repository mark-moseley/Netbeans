Wizards are defined in the . This API lets you create wizard panels, with steps, graphics, left side-bar text, and a user panel on the right.

The IDE provides templates that you can use to create a variety of wizards:

  • New Project wizard. Use the New Project Template wizard.
  • New File wizard. Use the New Wizard wizard.
  • Any other type of wizard. Use the New Wizard wizard.
