Module Project Properties Dialog Box: Sources

See Also 

You open the Module Project Properties dialog box by right-clicking a module project and choosing Properties.

The Sources page lets you specify the project folder, the module suite to which the project belongs, and the Java source level:

  • Project Folder: Displays the location of the project directory.
  • Module Suite. Displays the location of the containing suite, if this is a suite component module. If it is a standalone module, the field is not present.
  • Source Level. Sets the javac.source property in the nbproject/project.properties file. The options are 1.4 (default) or 1.5.
