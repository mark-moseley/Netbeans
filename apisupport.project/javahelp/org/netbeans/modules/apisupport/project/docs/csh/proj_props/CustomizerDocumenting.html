Module Project Properties Dialog Box: Documenting

See Also 

You open the Module Project Properties dialog box by right-clicking a module project and choosing Properties.

The Documenting page lets you specify the title of the Javadoc browser:

  • Javadoc Title. Sets javadoc.title in the project.properties file. Note that the IDE cannot build Javadoc unless you mark at least one package as public in the API Versioning page.
