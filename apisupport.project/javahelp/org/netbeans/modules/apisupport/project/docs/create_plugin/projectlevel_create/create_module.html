Creating a Module Project

See Also

In the IDE, you create modules in projects.

To create a module project:

  1. Choose File > New Project (Ctrl-Shift-N).
  2. From the NetBeans Plug-in Module category, select one of the following project templates:
    • Module Project.
    • Library Wrapper Module Project.
    • Module Suite Project.
  3. Follow the steps in the rest of the wizard.
