Module Installer Wizard: Module Installer

See Also 

The New Module Installer wizard creates a new module installer. You open the New Module Installer wizard by right-clicking a module project, choosing New > File/Folder and selecting Module Installer from the NetBeans Plug-in Module category.

In the Module Installer page, you have to set the following:

  • Package. Specifies the name of the package where the module installer will be housed. The module installer will always be named Installer.
