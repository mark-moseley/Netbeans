Module Suite Project Properties Dialog Box: Sources

See Also 

You open the Module Suite Project Properties dialog box by right-clicking a module suite project and choosing Properties. Here you can see or set the following properties:

  • Project Folder: Shows you the location of the module suite project folder in your filesystem.
  • Suite Modules. Specifies the modules that will be deployed together with the module suite project. Normally, a module is added to the suite's Suite Modules list automatically by the IDE when you use the New Project wizard to create the module as part of the module suite. However, you can use the Modules list to manually add or remove modules from the list, should you need to do so.
