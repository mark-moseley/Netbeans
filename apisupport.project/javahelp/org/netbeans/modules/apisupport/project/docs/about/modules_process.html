Working with Modules

The following table shows how to work with modules.

Create module project
  • Create the project using the best module project template for your programming needs.

Create file templates
  1. Right-click a module project in the Projects or Files window. Choose New > File/Folder.
  2. In the New File wizard, choose the best module file template for your programming needs.

Develop the module
  1. In the Projects window or Files window, double-click the file that you would like to edit.
  2. Use the Source Editor to edit the file.

Build the module
  • Choose Build > Build Main Project or right-click any project and choose Build Project. In addition, you can use the IDE to build the following:
    • NetBeans Module (.nbm) file
    • ZIP distribution
    • Java Web Start (.jnlp) file

Deploy, test, and debug
  • Choose Run > Run Main Project or right-click any project and choose Run Project.

Distribute
  1. If you have a suite project, right-click its node in the Projects window, choose Properties and use the Splash Screen panel to brand the application.
  2. Do one of the following:
    • If you do not have a suite project, right-click the module project and choose Create NBM. An NBM file is created. You can distribute and install this via the Update Center.
    • If you have a suite project, right-click it and choose Build ZIP Distribution. A ZIP distribution is built. Distribute the ZIP file.
