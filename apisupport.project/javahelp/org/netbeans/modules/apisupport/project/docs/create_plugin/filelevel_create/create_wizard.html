Creating A Wizard

See Also

Depending on the type of wizard you want to create, do the following:

To create a new Project wizard:

  1. Right-click a module project and choose New > Project Template.
  2. Follow the steps in the New Project Template wizard.

To create a new File wizard:

  1. Right-click a module project and choose New > Wizard.
  2. Follow the steps in the New Wizard wizard.

To create any other wizard:

  1. Right-click a module project and choose New > Wizard.
  2. Follow the steps in the New Wizard wizard.
