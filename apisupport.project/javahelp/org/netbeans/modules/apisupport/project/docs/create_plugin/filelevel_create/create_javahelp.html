Bundling a JavaHelp Help Set

See Also

A JavaHelp help set provides help files that explain the features and functionality of your plg-in module.

To create a new JavaHelp help set:

  1. Right-click a module project and choose New > File/Folder. In the New File wizard, choose JavaHelp Help Set under the NetBeans Module Development category.
  2. In the Location page, note the files that will be created and also note where they will be housed:
    • a. Specifies...
    • b. Specifies...
  3. Click Finish.
