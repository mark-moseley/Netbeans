Update Center Wizard: Update Center Declaration

See Also 

The New Update Center wizard lets you create a module that registers a update center in the user's Update Center wizard. In effect, the module bundles the URL to the update center so that the user does not need to register the URL manually. You open the Update Center wizard by right-clicking a module project, choosing New > Other and selecting Update Center from the Module Development category.

In the Update Center Declaration page, you have to set the following:

  • URL to Update Descriptor. Specifies the URL to the autoupdate descriptor.

    You can let the IDE generate the autoupdate descriptor for you; for details, see Generating an Autoupdate Descriptor.

  • Display Name. Specifies the name that you would like the user to see in their Update Center wizard.
