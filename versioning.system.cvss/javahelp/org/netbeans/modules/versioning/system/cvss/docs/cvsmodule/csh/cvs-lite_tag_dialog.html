CVS: Tag Dialog Box

Use this dialog box to attach a tag to a file or folder. You can attach an existing tag or create a new tag.

You access this dialog box by right-clicking a file or folder and choosing CVS > Tag.

To attach a tag to a file or folder, enter the tag name or click Browse to choose from a list of existing tags. You can also specify the following options:

  • Avoid Tagging Locally Modified Files. If selected, tags are not attached to local revisions that are not yet committed to the repository.
  • Move Existing Tags. If selected, tags attached to a revision
