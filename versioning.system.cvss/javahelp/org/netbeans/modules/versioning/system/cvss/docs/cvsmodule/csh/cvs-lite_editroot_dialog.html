CVS: CVS Root Dialog

See Also

Use this dialog box to specify the CVS root for the repository when checking out files from the repository or importing files into the repository.

You can access this dialog box by clicking the Edit button in the CVS Root panel of the CVS Checkout wizard or the CVS Import Project wizard.

  • Access Method. Select one of the following access methods from the drop-down list according to your type of CVS access.
    • pserv Select this type if you are connecting to CVS pserver repositories.
    • ext
      • Select this type if you are connecting to a CVS repository using SSH, etc.
    • local
      • Select this type for protocol access to a local CVS repository. This type requires an external CVS executable.
    • fork
      • Select this type for direct access to a local CVS repository. This type requires an external CVS executable.
  • User. Enter the username for the repository.
  • Host. Enter the host name of the repository.
  • Port. Enter the port number, if necessary.
  • Repository Path. Enter the path to the repository.
