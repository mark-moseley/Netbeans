CVS: Change CVS Root Dialog Box

See Also

The Change CVS Root dialog box enables you to modify the path to the CVS repository which you are working in. You can access this dialog box by choosing Versioning > CVS > Change CVS Root from the main menu.

For reference, the Change CVS Root dialog box displays the working copy on your system, as well as the CVS Root you are currently registered with.

To modify the CVS Root:

  • In the New CVS Root field, enter a new root type and location. You can select a root location from the drop-down list (if previous entries have been made), otherwise type in a new root type and location. Alternatively, click the Edit button to open the Edit CVS Root dialog box.
  • Click Change. The new CVS Root is registered in the IDE, and will be applied the next time you attempt to connect to the repository.
