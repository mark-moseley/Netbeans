Version control systems (VCS) aid developer groups working with shared source files in common repositories by managing file revision history information. The IDE's various VCS tools provide a means of effectively managing changes to version-controlled files.

In the IDE, you can call VCS commands on both files and directories in the Files window, Versioning window and Favorites window. To work with VCS in the IDE, all you need to do is register a VCS local working directory. For more information, see Setting Up VCS.

CVS Lite Tools

The IDE also provides several tools to simplify the process of working with version controlled files including:

  • Versioning Window. Enables you to view the evolution of changes in version-controlled files.
  • VCS Output Window. Provides automatic monitoring of active VCS operations.
  • Graphical Diff Viewer. Enbles you to compare differences in file revisions graphically
  • Merge Conflict Resolver. Assists you in resolving conflicts that occur when merging file revisions.
