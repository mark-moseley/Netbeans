CVS Wizard: Folder to Import

You see this panel of the CVS Import Project Wizard after you have specified the CVS Root for the repository in the first panel. In this panel you choose the folder to import and the target location in the repository.

When importing into the repository, you specify the following:

  • Folder to Import.
  • Import Message.
  • Repository Folder.
  • Checkout After Import.
