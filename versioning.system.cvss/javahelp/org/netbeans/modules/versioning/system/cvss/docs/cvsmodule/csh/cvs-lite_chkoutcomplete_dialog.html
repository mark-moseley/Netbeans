After you complete a checkout from the repository, you can choose to create a project from the checked out files. If you choose to create a project from the files, the IDE opens the New Project wizard to help you create the project. In the New Project wizard, you will be asked to select the project type and other project details for the new project.

If you do not want to create a project immediately, click Close. You can create a project later by opening the New Project wizard (Ctrl-Shift-N).
