CVS: Branch Dialog

Use this dialog box to specify the branch when creating a new branch or adding a file or folder to an existing branch.

You access this dialog box by right-clicking a file or folder and selecting CVS > Branch.

Specify the following in the Branch dialog box:

  • Branch name. Enter a new Branch name to create a new branch, or click Browse to see a list of available branches.
  • Tag Before Branching. If selected, the specified tag name is attached to the file or folder. The branch is created after the tag is attached to the file or folder. Enter a tag name in the text field.
  • Switch to This Branch Afterwards. If selected, the file or folder is switched to the new branch after creating it.
