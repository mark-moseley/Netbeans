CVS: View Revision Dialog Box

See Also

The CVS View Revision dialog box enables you to choose a specific revision for a versioned file and view that revision in the IDE's main window.

You access this dialog box by selecting a versioned file (e.g. in the Projects, Files, Favorites, or Versioning window) and choosing either CVS > View Revision from the right-click menu, or Versioning > View Revision from the main menu.

In the View Revision dialog box, choose from the following two options:

  • Tag/Branch/Revision. Select this option to specify the name of the tag, branch, or revision you want to view. In the adjacent text box, either type in the keyword that identifies the tag, branch, or revision, or click Browse to view items maintained in the repository.
  • Working Branch HEAD. Select this option to view the most current revision of the selected file.
