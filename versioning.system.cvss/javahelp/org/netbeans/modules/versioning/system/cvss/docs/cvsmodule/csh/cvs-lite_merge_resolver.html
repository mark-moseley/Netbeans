CVS: Merge Conflicts Resolver

Use the Merge Conflicts Resolver to resolve conflicts between version-controlled files before committing files to the repository.

When merge conflicts occur, a Merge Conflict badge () appears on the file's node. To resolve merge conflicts in the Merge Conflicts Resolver, right-click the file with the conflict and choose CVS > Resolve Conflicts.

The Merge Conflicts Resolver displays the two conflicting revisions side by side, with the conflicting areas highlighted. You resolve the conflicts by accepting one of the revisions. Once all conflicts are resolved, click OK.

For more, see Resolving Merge Conflicts.
