Setting Up CVS

See Also

The IDE's CVS setup is accomplished in the process of checking out files located in a remote repository to a local working directory on your system. The IDE automatically registers the working directory as it creates the necessary status information in which local copies of version-controlled files along with their status information will be stored.

See Checking Out Files from a Remote Repository for more information.
