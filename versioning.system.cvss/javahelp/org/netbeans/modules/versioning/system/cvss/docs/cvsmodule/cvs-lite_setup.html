Setting Up CVS

See Also

To work with version control in the IDE, you must register a local working directory in which local copies of version-controlled files and their status information will be stored. The IDE's CVS support enables you to accomplish this by simply checking out files located in a remote repository to a local working directory on your system. The IDE automatically registers the working directory as it creates the necessary status information.

See Checking Out Files from a Remote Repository for more information.
