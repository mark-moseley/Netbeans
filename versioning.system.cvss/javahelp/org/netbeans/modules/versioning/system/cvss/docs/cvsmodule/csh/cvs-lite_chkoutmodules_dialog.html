CVS: Browse Modules Dialog

You use this dialog box to select a module in the repository and determine its path. Expand the tree and select a module from the list. You can select more than one module by holding down the Ctrl key and selecting modules. The path to the module appears in the Module pane of the dialog box.

You can access this dialog box in the following ways:

  • Right-clicking a file or folder in the Versioning window and choosing List Modules.
  • Clicking the Select button in the CVS Export or CVS Checkout dialog boxes to determine the path to a module when exporting version-controlled files.
