CVS: Browse Modules Dialog Box

You use this dialog box to select a module in the repository and determine its path when checking out a module from the repository.

You can access this dialog box by clicking the Browse button next to the Module field in the CVS Checkout wizard.

The Browse Modules dialog box displays the contents of the repository in a tree view. Expand the nodes to view and select modules in the repository.

When you click OK, the path to the selected module appears in the Module field of the Checkout wizard.

  • You can select more than one module by holding down the Ctrl key when selecting modules.
