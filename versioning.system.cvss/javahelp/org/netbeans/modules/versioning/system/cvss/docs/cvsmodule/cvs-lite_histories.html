Working with Version Histories

See Also

Searching the histories of files can be helpful when you need to find specific commits, when fixing bugs for example.

Viewing File's Version History

The IDE's Search Histories window enables you to view a summary of a file's evolution over time by revision number. You can open the Search History window by choosing CVS > Search History.

You can compare a file with previous versions from within the Search Histories window using the Diff button.

To view revision differences in the Search History window:
  1. Click the Diff button in the toolbar.
  2. Select the revision against which you want to compare your local working copy.
  3. The IDE displays a diff of the file's revisions within the Search History window. Note that you can navigate between the differences using the Previous Difference and Next Difference buttons.

Searching for Specific Revisions

Using the Search History window, you can search a file's history for versions based on several criteria. 

The Search Histories window enables you to search for changes in version controlled files by:

  • Message. The description submitted with the commit.
  • Username. The author of the commit.
  • From Tag. The tag, branch, or date from which the commit was made.
  • To Tag. The tag, branch, or date before which the commit was made.

To search for a revision:

  1. Select the file for which you want to find a specific revision.
  2. Choose CVS > Search filename History from the main menu.

    3. The IDE displays the file's revision history in the Search History window.

  3. Enter search criteria information in the Message, Name, From, and To fields. You can use the Browse buttons to designate any tag, branch, revision, or date you want to limit your search to.
  4. Click Search.

    5. The IDE displays the file's revision history in the Search History window.

If you want to view other files that were included in a specific commit, you can broaden your search using the "your_project" and Open Projects links to the right of each revision.
