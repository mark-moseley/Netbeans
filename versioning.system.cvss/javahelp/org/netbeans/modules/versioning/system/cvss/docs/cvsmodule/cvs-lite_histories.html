Searching the histories of files can be helpful when you need to find specific . The IDE's Search Histories window provides a summary pane that displays _____. It also ______. You can diff files with previous versions. You can also display a list of files included in a particular commit using the Open Projects link.

The Search Histories window enables you to search for changes in version controlled files by:

  • Message. Indicates lines that have been changed since the earlier revision.
  • Username. Indicates lines that have been added since the earlier revision.
  • From Tag. Indicates lines that have been removed since the earlier revision.
