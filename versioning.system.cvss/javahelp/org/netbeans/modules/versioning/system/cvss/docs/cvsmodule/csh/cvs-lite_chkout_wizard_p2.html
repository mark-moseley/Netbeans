CVS Wizard: Module to Checkout

See Also

The Module to Checkout page of the CVS Checkout wizard enables you to choose the modules and branches you want to checkout and specify the location of your local working directory.

When checking out files from the repository, you specify the following:

  • Module. Enter the name of the module to checkout. You can click the Browse button to open the Browse Modules dialog box to search for available modules. If you do not specify a module, all modules in the repository are checked out.
  • Branch. Enter the name of the branch to checkout. You can click the Browse button to open the Browse Branches dialog box to search for available branches and tags. If you do not specify a branch or tag, the trunk is checked out.
  • Local Folder. Enter the location of your local working directory or click the Browse button to locate the directory.
