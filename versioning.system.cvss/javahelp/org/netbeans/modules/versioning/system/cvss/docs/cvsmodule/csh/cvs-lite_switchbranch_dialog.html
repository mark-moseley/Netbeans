CVS: Switch to Branch Dialog Box

The CVS Switch to Branch dialog box enables you to switch the selected files or folders in your local working directory to either the trunk or another branch. Revisions are not merged when you switch from a branch to the trunk or from the trunk to a branch.

  • CVS Status Labels in the Projects window indicate if the local working files are in a branch or the trunk. To activate CVS Status Labels, choose View > Show CVS Status Labels from the main menu.

You access this dialog box by right-clicking a file or folder and choosing CVS > Switch to Branch.

Select one of the following actions:

  • Switch to Trunk. If selected, your local working files are switched to the trunk.
  • Switch to Branch. If selected, your local working files are switched to the specified branch. Enter the name of the branch in the text field or click Browse to see a list of the available branches.
