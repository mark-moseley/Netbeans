Getting Started with NetBeans IDE 5.0

Welcome to NetBeans IDE 5.0! Use these guides to quickly get started.

Creating New Java Applications

These guides describe how to work with various technologies in the IDE. Each guide takes you through the process of creating, building, and running a simple example project.

What do you want to create?

  • General Java applications or libraries
  • Web applications
  • Enterprise applications
  • Web services
  • Web service clients
  • J2ME Mobile Information Device Profile (MIDP) applications
  • Java Graphical User Interfaces
  • NetBeans plug-in modules

Importing Existing Java Applications

These guides describe how to get your existing source code set up and working in the IDE. The import guides show you how to set up NetBeans projects from your sources, configure project dependencies, and use your own Ant scripts to build and run your projects.

What do you want to import?

  • General Java applications or libraries
  • Web applications
  • Enterprise applications or EJB modules
  • J2ME Mobile Information Device Profile (MIDP) application
  • Eclipse projects
  • JBuilder projects
