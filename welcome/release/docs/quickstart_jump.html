Add Functionality

The standard distributions of NetBeans IDE gives you everything you need to develop Java SE desktop applications, web applications, and Java EE enterprise applications. You can also download and install add-on packs that give you even more functionality.

  • Mobility Pack - Adds everything needed to immediately start writing, testing and debugging Java applications for mobile phones and other Java Micro Edition (Java ME) technology-enabled devices.
  • Enterprise Pack - Introduces tools for UML modeling, XML tools, and Web services orchestration to NetBeans IDE.
  • Profiler - A profiling tool that provides information about the runtime behavior of your applications.
