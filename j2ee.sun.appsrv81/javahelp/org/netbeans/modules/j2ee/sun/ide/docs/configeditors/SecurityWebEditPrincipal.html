Visual Editor: Edit Principal Dialog Box

See Also

This dialog box enables you to edit the name of a principal assigned to a security role. You open this dialog box by selecting a principal name in the Security Role Mappings section of the Security tab of sun-web.xml, sun-ejb-jar.xml or sun-application.xmland clicking Edit Principal.

This dialog box enables you to edit the following properties:

  • Principal Name. Use this field to modify the name of the principal assigned to a security role. The name of the principal must match a name specified on the Sun Java System Application Server/GlassFish.
  • Class Name. Use this field to modify the implementation class.
