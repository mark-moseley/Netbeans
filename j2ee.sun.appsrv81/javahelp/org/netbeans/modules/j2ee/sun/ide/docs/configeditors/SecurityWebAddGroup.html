sun-web.xml Visual Editor: Add Group Dialog Box

See Also

This dialog box enables you to specify the name of a group to add to a security role. You open this dialog box by clicking Add Group in the Security Role Mapping pane of sun-web.xml.

This dialog box enables you to add the following properties:

  • Group Name. This field specifies the name of the group to add to a security role. The name of the group must match a name specified on the Sun Java System Application Server.
