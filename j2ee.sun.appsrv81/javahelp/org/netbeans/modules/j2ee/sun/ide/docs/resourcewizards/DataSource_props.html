New JDBC Resource: Additional Properties

See Also

You see this page of the New File wizard when you are defining a JDBC resource.

This page of the wizard enables you to add additional configuration information and properties for a JDBC resource.

To enter new properties and values:

  1. Click Add to add a row for the new property to the table.
  2. In the Properties table, enter the Name and Value of the new property.
  3. Click Enter to save the changes to the property table.
  4. Click Finish to exit the wizard.
  • If the Finish button is dimmed, ensure that both the Name and Value properties are not empty and hit Enter on your keyboard.

To remove a property, select the property in the Properties table and click Remove.
