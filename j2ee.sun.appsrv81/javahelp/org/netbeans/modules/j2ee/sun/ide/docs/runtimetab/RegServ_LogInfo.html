Add Server Instance Wizard: Enter Domain Admin Login Info

See Also

You see this page of the Add Server Instance wizard when you specify domain details to register an instance of the Sun Java System Application Server. Use this page of the wizard to specify the username and password for the domain administrator.

When you register a server, you do not need to supply the domain admin username and password. If you do not supply the username and password, you are prompted to supply the username and password when needed.

  • If you enter a user name and password, the values are stored in your user directory. This represents a security risk.

In this page of the wizard you can supply the following information:

  • Admin Username. Enter the user name for the domain administrator.
  • Admin Password. Enter the password for the domain administrator.
