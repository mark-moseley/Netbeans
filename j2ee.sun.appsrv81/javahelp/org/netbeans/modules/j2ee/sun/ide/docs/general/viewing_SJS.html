Viewing the Sun Java System Application Server in the IDE

See Also

The Servers node in the Runtime window and its subnodes represent the Sun Java System Application Server, its instances, applications that have been deployed to the instance and other server resources.

The following table shows how to work with the Sun Java System Application Server.

Icon Description
Sun Java System Application Server
Expand this node to see the resources for the server instance.
JDBC resources
This node contains JDBC Resources and Connection Pools
JVMs
Right-click this node to display the properties of the JVM
Applications
Expand this node to see the applications and modules deployed to the server.
JavaMail Resources
Expand this node to see the JavaMail Resources registered with the server.
JMS Resources
Expand this node to see the Java Message Service (JMS) Resources registered with the server.
JNDI Resources
Expand this node to see the custom and external JNDI resources registered with the server.
Persistence Manager Resources
Expand this node to see any Persistance Manager resources deployed to the server.
