sun-web.xml Visual Editor: Edit Group Dialog Box

See Also

This dialog box enables you to edit the name of a group assigned to a security role. You open this dialog box by selecting a group name in the Security Role Mapping pane of sun-web.xml and clicking Edit Group.

This dialog box enables you to edit the following properties:

  • Group Name. Modify the name of the group assigned to a security role. The name of the group must match a name specified on the Sun Java System Application Server/GlasFish.
