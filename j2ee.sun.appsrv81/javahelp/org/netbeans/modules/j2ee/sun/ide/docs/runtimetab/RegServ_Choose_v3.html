Add Server Instance Wizard: GlassFish v3 Server Location

See Also

You see this page of the Add Server Instance wizard when you specify details to register an instance of GlassFish v3 with the IDE. To register a server instance, you specify the properties of a local or remote domain.

Use this page to specify the following information about the location of GlassFish v3.

  • Installation Location. Supply the path to a local installation of GlassFish v3 server. Click Browse to navigate to the location of the local installation of the server.

    If there is no installation of GlassFish v3 on your local system, you can download and install the GlassFish v3 server from the wizard by clicking the Download button. After you click Download, the IDE will install the current version of GlassFish v3.

    • You must accept the terms of the licensing agreement to enable the Download button.

After you specify the location of the GlassFish v3 installation, click Next to specify the details of the local or remote domain.
