Change Configuration Version Dialog Box

See Also

You open the Change Configuration Version Dialog Box when you use the visual editor to edit xml descriptor files such as sun-web.xml.

This dialog box enables you to change the Doctype of the xml file to correspond to the target Sun Java System Application Server. The Server Version drop-down list contains a list of the instances of the Sun Java System Application Server registered with the IDE.
