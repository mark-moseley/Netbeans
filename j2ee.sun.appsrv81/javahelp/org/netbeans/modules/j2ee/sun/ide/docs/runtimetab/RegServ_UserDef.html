Add Server Instance Wizard: Enter Domain Directory Path

See Also

You see this page of the Add Server Instance wizard when you are creating a new domain or specifying an existing local domain that is not the local default domain. Use this page of the wizard to specify the location of the domain you are registering.

If you are specifying the location of an existing domain, type the path to the domain directory in the Domain Directory text field, or click Browse to use the file chooser to navigate to the domain on your system.

If you are creating a new domain, in the Domain Directory text field, type the path to the directory you want to contain the domain, then type the name of the domain. The IDE creates a new directory with the name of the domain. Alternately, click Browse and use the file chooser to locate the containing directory and then type the name of the new domain.
