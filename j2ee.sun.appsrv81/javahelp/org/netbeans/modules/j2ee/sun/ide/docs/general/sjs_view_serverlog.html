Viewing the Sun Java System Application Server/GlassFish Log File

See Also

You can view the log files of a registered instance of the Sun Java System Application Server/GlassFish directly in the IDE. Log files describe the events that occur on the server and can be helpful when troubleshooting your application.

To view the log file:

  • Expand the Servers node in the Services window of the IDE.
  • Right-click the server instance node and choose View Server Log.

When you choose View Server Log, the server log is displayed in the Output window of the IDE.
