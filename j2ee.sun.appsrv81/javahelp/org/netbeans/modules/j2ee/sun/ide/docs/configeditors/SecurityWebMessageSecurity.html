Visual Editor: Message Security Dialog Box

See Also

This dialog box enables you to specify the message security entries for the message-security-binding property when using Message Security Binding for a web service endpoint or port. You can access this dialog from the Message Security Bindings dialog box when you create a new message security entry or when you edit the properties of message security entry. You access the Message Security Bindings dialog box by selecting the Message Security option for a web service endpoint in the deployment descriptor visual editor and clicking Edit Bindings.

You also use this dialog box when configuring message security policy for a port for web service clients.

This dialog box enables you to edit the following properties:

  • Java Method Name (required).
  • Request Authorization Source. Select the source from the drop-down list.
  • Request Authorization Recipient. Select the recipient from the drop-down list.
  • Response Authorization Source. Select the source from the drop-down list.
  • Response Authorization Recipient. Select the recipient from the drop-down list.

For more information on configuring message security, see "Using Message Security with Java WSDP" in the Java Web Services Tutorial.
