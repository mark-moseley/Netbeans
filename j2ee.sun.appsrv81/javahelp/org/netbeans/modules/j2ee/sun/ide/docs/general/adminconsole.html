Accessing the Java System Application Server Admin Console

See Also

The Sun Java System Application Server instance node in the IDE provides a subset of the functions found in the Admin Console of the Java System Application Server. You can manage server resources through the IDE (Sun Java System Application Server node), or by using the Java System Application Server Admin Console.

To access the Java System Application Server Admin Console:

  1. Make sure the Java System Application Server is running. (Denoted by the running badge (  ) next to the Sun Java System Application Server node in the Runtime window.)
  2. Right-click on the Sun Java System Application Server instance and choose View Admin Console.

The login screen for the Admin Console launches in your browser. After you log in, you can see that the options on the left are similar to those in the Runtime window in the IDE.

  • To access the Admin Console, you are required to log in as the server administrator. The default username is admin. Unless the password has been changed, the default password is adminadmin.

When you manage resources directly through the Admin Console, the resources are not directly associated with your project.
