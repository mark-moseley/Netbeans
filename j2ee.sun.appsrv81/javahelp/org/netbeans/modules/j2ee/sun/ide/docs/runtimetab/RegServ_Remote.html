Add Server Instance Wizard: Enter Domain Admin Port Info

You see this page of the Add Server Instance wizard when you are registering a remote domain instance of the Sun Java System Application Server. Use this page of the wizard to specify the host and communication port for the remote domain.

Enter the following information for the remote domain:

  • Host. Enter the host name or IP address for the remote Sun Java System Application Server.
  • Port. Enter the port number used to communicate with the remote server.
  • User Name: the admin user name
  • Password: the admin user password
