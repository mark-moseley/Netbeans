Add Server Instance Wizard: Domain Admin Port Info

See Also

You see this page of the Add Server Instance wizard when you are registering a remote domain instance of the Sun Java System Application Server/Glassfish. Use this page of the wizard to specify the host and communication port for the remote domain.

Enter the following information for the remote domain:

  • Host. Enter the host name or IP address for the remote Sun Java System Application Server/Glassfish.
  • Port. Enter the port number used to communicate with the remote server. The default port is 4848.

Click Next if you want to supply a username and password for the remote domain. Click Finish if you do not want to supply a username and password at this time. If you do not supply a username and password, you are prompted to supply a username and password when needed.
