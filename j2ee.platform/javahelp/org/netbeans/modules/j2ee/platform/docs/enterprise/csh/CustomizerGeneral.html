Standard Enterprise Application Project Properties Dialog Box: General

See Also 

You open the Standard Enterprise Application Project Properties dialog box by right-clicking any  and choosing Properties. The General page lets you view the location of the project folder and lets you set the target Java platform for the project.

The target Java platform is the JDK against which all source files are compiled and run. You can pick a Java platform from the combo box, or click Edit to register new Java platforms.
