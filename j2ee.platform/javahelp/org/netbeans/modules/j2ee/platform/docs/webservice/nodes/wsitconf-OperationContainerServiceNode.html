Web Service Attributes Editor: Operation

See Also 

In the Quality of Service tab's Operation section, on the service side, you can add Quality of Service security to a specific web service operation.

You can configure Quality of Service security on the following levels:

  • Operations. Use this section to specify a transaction setting and a security mechanism for a web service operation.
  • Input Message.
    Output Message. Use these sections to specify which message parts have sensitive information that needs to be encrypted and/or signed.
