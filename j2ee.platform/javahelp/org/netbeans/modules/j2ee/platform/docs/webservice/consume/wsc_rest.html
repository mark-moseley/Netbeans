To use the preregistered web service components:

  1. Go to the related homepage of the web service and obtain the required keys and codes, if necessary:
    • To get a Google Map key, go here:

    • To get a Strike Iron username and password, go here:

  2. Drag the item from the Services tab.
  3. Fill in the applicable details in the dialog box that appears.
  4. Drop the item in the resource class.
