Using the Preregistered RESTful Web Service Components

See Also 

When you generate RESTful web services, and open one in the editor, the component palette (Ctrl-Shift-8) displays a list of preregistered web service components. The list of preregistered web service components is as follows:

  • Google Map
  • Strike Iron
    • Address Verification
    • Email Verification
    • IP Address Lookup
    • Reverse Phone Lookup
    • Tax Complete
  • Yahoo News Search
  • For a tutorial that shows how the preregistered RESTful web service components are used, see:

To use the preregistered web service components:

  1. Go to the related homepage of the web service and obtain the required keys and codes, if necessary:
    • To get a Google Map key, go here:

    • To get a Strike Iron username and password, go here:

  2. Drag the item from the palette.
  3. Fill in the applicable details in the dialog box that appears.
  4. Drop the item in the resource class.
