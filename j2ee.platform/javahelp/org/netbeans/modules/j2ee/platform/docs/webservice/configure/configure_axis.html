Configuring Axis2 Web Services

  • For a full tutorial that shows how to work with Axis2 web services in the NetBeans IDE, see:

To configure an Axis web service, right-click the service node. Select Configure from the context menu and the Configuration dialog opens.

In the Configuration dialog, you can make the following changes:

  • Rename the service.
  • Change its implementation class.
  • Select whether or not to generate a WSDL file when you build the project, to bundle into the AAR. This is the same option you have when creating an Axis2 service from Java.

If you want to edit the WSDL file, Generate WSDL File must be selected in the configuration. When it is selected in the configuration, select Generate WSDL from the service's context menu, and the WSDL is created immediately rather than when the project is built. Select Edit WSDL, and that WSDL file opens in an editor. After editing and saving changes to the WSDL, propagate those changes back to the Java classes by selecting Refresh.
