Creating a Message Handler

See Also 

  1. Right-click the project node for a web application or an EJB module and choose New > File/Folder.
  2. Under Categories, select Web Services. Under File Types, select Message Handler. Click Next.
  3. Name the message handler and select or define the package that will contain it.
  4. In the Source Editor, modify the default message handler to suit your requirements.
