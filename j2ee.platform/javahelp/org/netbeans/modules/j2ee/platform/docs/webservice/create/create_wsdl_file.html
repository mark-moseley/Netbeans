Generating a WSDL File

See Also 

Web Services Description Language (WSDL) is an XML-based specification for describing a web service. WSDL defines a web service as a set of endpoints or ports operating on messages. WSDL is extensible to allow the description of endpoints and their associated messages regardless of what message formats or network protocols are used to communicate.

To create a WSDL file:

  1. In the Projects window or Files window, right-click the project node and choose New > Other. In the New File wizard, choose Web Services under Categories and choose WSDL Document under File Types. Click Next.

    2. The New WSDL File wizard opens.

  2. Type the WSDL file name name and specify a folder to house it.
  3. Specify the namespace of the elements (such as message, portType, binding) defined in the WSDL file.
  4. Specify the XML schemas needed by the WSDL file, if any.
  5. Click Finish.

    The IDE creates a WSDL file for you, with default code that you can modify according to your business needs.

  6. Create a web service from the WSDL file.
