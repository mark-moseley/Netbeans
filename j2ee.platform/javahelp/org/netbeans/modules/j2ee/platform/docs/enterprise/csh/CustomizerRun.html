Standard Enterprise Application Project Properties Dialog Box: Execution

See Also 

You open the Standard Enterprise Application Project Properties dialog box by right-clicking any  and choosing Properties. The Execution page lets you configure options for running your project.

In the Execution page, you can set the following properties:

  • Client Module URI. Specifies the name of the WAR file of the client module in the EAR file.
  • Server. Specifies the server to which the web application is deployed. To add servers, right-click the Server Registry node in the Runtime Window.
  • Display Browser on Run. Specifies whether the browser is displayed.
  • Relative URL. Specifies the URL relative to the context path to run.
  • Embedded Classpath Elements. Specifies the elements in the EAR file's classpath.
