Building an Enterprise Application

See Also

You can build the main project from the main menu or an individual project from its contextual menu.

To build the main enterprise application project and its modules:

  • Choose Build  Build Main Project (F11).

To build an individual enterprise application project and its modules:

  • Right-click the project's node in the Projects window and choose Build Project.

When you build a project, the IDE calls the corresponding target in the project's Ant build script. The IDE compiles the source files and builds a packaged build output, an EAR file in the dist folder.

The IDE displays the Ant output and any compilation errors in the Output window. You can double-click any error to go to the location in the source code where the error occurred.
