Accessing the JBoss Application Server Management Console

See Also

The JBoss Application Server instance node in the IDE provides access to the Management Console of the JBoss Application Server. You can manage server resources, such as deployed applications, through the Management Console.

To access the JBoss Application Server Management Console:

  1. In the Runtime window, expand the Servers node.
  2. Start the server by right-clicking the JBoss Application Server instance node and choosing Start.
  3. Once the server has started, right-click the node and choose View Admin Console.
