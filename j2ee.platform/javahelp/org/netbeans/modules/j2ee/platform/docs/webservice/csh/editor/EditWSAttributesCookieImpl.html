About the Web Service Attributes Editor

See Also 

For JAX-WS web services (Java EE 5), the Web Service Attributes Editor helps you to customize various aspects of your application, in relation to its web services and clients. Depending on the context, the editor consists of one or more of the following panels:

  • Quality Of Service.
  • WSDL Customization.
  • Web Service Provider Security Configuration.
  • Web Service Client Security Configuration.
