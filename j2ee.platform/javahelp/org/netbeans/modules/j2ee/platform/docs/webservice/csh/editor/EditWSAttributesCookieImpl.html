About the Web Service Attributes Editor

See Also 

For JAX-WS web services (Java EE 5), the Web Service Attributes Editor helps you to customize various aspects of your application, in relation to its web services and clients. Depending on the context, the editor consists of one or more panels. For example, for web services and clients created from WSDL files, the editor provides the WSDL Customization panel. You can use the WSDL Customization panel to modify the artifacts generated from a WSDL file.

The subsections of the Quality of Service panel are as follows, on the service side:

  • Service Port Binding
  • add Operation
  • Operation
  • Input Message
  • Output Message

On the client side, the subsections of the Quality of Service panel are as follows:

  • Transport
  • Security
  • Advanced Configuration

The subsections of the WSDL Customization panel are as follows:

  • Global Customization
  • Port Types
  • Port Type Faults
  • Bindings
  • Binding Operations
  • Services
  • Ports
  • External Binding Files
  • Other panels may be available in the Web Service Attributes Editor, depending on the modules installed in the IDE.
