Web Service Attributes Editor: WSDL Customization

See Also 

For web services and clients created from WSDL files, the editor provides the WSDL Customization panel. You can use the WSDL Customization panel to modify the artifacts generated from a WSDL file.

The subsections of the WSDL Customization panel are as follows:

  • Global Customization
  • Port Types
  • Port Type Faults
  • Bindings
  • Binding Operations
  • Services
  • Ports
  • External Binding Files
