Marshalling and Unmarshalling Java to XML

See Also 

The IDE provides two code templates that you can use to generate code snippets for marshalling or unmarshalling Java code.

To marshall or unmarshall Java code to XML:

  1. Make sure that you have installed the Full distribution of the IDE, not Basic, and not Standard, since the code templates are only provided with the Full distribution.
  2. If the JAXB JAR files are not on the application's classpath, right-click the Libraries node, choose Add Library, and find the JAXB 2.1 library in the Add Library dialog. Select the library and click Add Library.
  3. To generate a code snippet for marshalling Java code to XML, type jaxbm in the Source Editor for Java files and then press Tab. The snippet is created and you can now incorporate it in the rest of your code.
  4. Alternatively, to generate a code snippet for unmarshalling Java code to XML, type jaxbu in the Source Editor for Java files and then press Tab. The snippet is created and you can now incorporate it in the rest of your code.
