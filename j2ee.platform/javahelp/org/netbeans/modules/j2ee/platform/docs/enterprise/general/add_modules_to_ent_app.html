Adding a Module to an Enterprise Application

See Also

You can deploy EJB modules or web modules as stand-alone modules or as a part of a J2EE application.

When you add a module to a J2EE application, the IDE does the following:

  • Lists the module under the J2EE application's J2EE Modules node.
  • Adds the module project to the J2EE application project's list of required projects. This means that the module project is cleaned and built whenever you clean and build the J2EE project.
  • Adds a module reference to the J2EE application's deployment descriptor.

To add an existing module to an enterprise application:

  1. Open the module project and the J2EE application project.
  2. In the Projects window, right-click the J2EE Modules node for the J2EE application and choose Add J2EE Module.
  3. In the dialog box, select the module and click OK.
You can add an module to a J2EE application in the New Project wizard when you create the module.
