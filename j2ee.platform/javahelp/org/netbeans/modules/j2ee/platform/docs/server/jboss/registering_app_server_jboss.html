Registering a JBoss Application Server

See Also

Before you can start developing and deploying applications to an application server, you must register your application server with the IDE.

To register an instance of a JBoss application server:

  1. Choose Tools > Server Manager, and in the Server Manager, click Add Server.
    • Alternatively, right-click the Servers node in the Runtime window and choose Add Server from the contextual menu.
    The Add Server Instance wizard starts.
  2. In the Choose Server pane, do the following:
    1. From the Server drop-down list, select your application server.
    2. In the Name field, type a name for the server instance.
      This name is used to identify the server in the IDE.
    3. Click Next.
  3. In the Server Location pane, use the Browse button to navigate to and select the installation directory of your application server.
    • The Next button is only enabled when you have selected the correct installation directory.
  4. Click Next.
  5. In the Instance Properties pane, from the Domain drop-down list, select the JBoss application server domain you want to use for this server instance.
    The IDE automatically updates the Domain path field.
  6. Use the Port field to define the port for the server instance.
  7. Click Finish.
  8. If you started the Add Server Instance wizard from the Server Manager, click Close in the Server Manager dialog box.
    The IDE displays a server node for your application server in the Runtime window under the Servers node.
