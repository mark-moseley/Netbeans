Standard EJB Module Project Properties Dialog Box: Sources

See Also 

You open the Standard EJB Module Project Properties dialog box by right-clicking any  and choosing Properties. The Sources page lets you define the folders that contain the project's source packages and test packages.

Use the buttons on the right of the dialog box to add, remove, and change the order of the folders.
