enterprise information system

Enterprise information system (EIS) software includes enterprise infrastructure systems such as enterprise resource planning (ERP), mainframe transaction processing, database systems, and other legacy information systems that are not part of J2EE. The EIS tier, one of several tiers of the J2EE multitiered distributed application model, handles EIS software, which runs on an EIS server that is external to the J2EE application server. J2EE application components use the J2EE Connector architecture to access EIS tier resources.
