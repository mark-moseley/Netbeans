New Packaged Archive Wizard: Name and Location

See Also 

This wizard creates a new project containing a packaged archive of a Java EE application. You open the wizard by opening the New Project wizard and selecting the Packaged Archive template from the Enterprise category.

In the Name and Location page, you have to set the following properties:

  • Project Name. Specifies the display name of the project. This name is also used to create the project folder.
  • Project Location. Specifies the folder where you want to create the project folder.
  • Project Folder. Displays the location of the project folder, which contains the configuration file folders for your project. The project folder also contains your project's Ant build script and project metadata.
  • Archive File. Specifies the archive file on which you are basing the project. The file can be a WAR file, and EJB JAR file, or an EAR file.
  • Target Server. Specifies the server to which the newly packaged archive is deployed. To register servers, click Manage.

When you click Finish, the project is created on your computer and opened in the IDE. You can view the project's logical structure in the Projects window and its file structure in the Files window.
