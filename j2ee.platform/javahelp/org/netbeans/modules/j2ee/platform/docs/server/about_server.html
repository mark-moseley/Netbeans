About Servers

See Also

A server is a computer that serves up applications. Every server has an IP address and possibly a domain name. For example, if you enter the URL http://www.netbeans.org in your web browser, a request is sent to the computer that has netbeans.org as a domain name. By default, the computer then fetches the page named index.html and sends it to your web browser.

Any computer can be turned into a server if you install server software and connect the computer to the Internet. Server software for the Tomcat Web Server is bundled and registered with the IDE. To use an external Tomcat Web Server, you must register it with the IDE. In some distributions, the Sun Java System Application Server is bundled with the IDE. However, you need to register the Sun Java System Application Server with the IDE before you can use it.

When a server has been registered, you can see its node in the Runtime window under the the Server Registry node.
