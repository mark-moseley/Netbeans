Standard Enterprise Application Project Properties Dialog Box: Packaging Project

See Also 

You open the Standard Enterprise Application Project Properties dialog box by right-clicking any  and choosing Properties. The Packaging Project page lets you set up the the filter for which files are included in the WAR file.

Notes:

  • To change the EAR name and location, open the project.properties file. Enter the full path to the eAR file in the dist.dir property and the EAR file's name in the dist.jar property.
