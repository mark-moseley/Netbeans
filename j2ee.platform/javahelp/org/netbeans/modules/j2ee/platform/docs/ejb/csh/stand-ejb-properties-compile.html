Standard EJB Module Project Properties Dialog Box: Compiling

See Also 

You open the Standard EJB Module Project Properties dialog box by right-clicking any  and choosing Properties. The Compiling page lets you set general compilation options.

Enter any compiler options in a space-separated list in the Additional Compiler Options text field. For more infomation on compiler options, see:

Notes:

  • Do not use compiler options like -bootclasspath to cross-compile between Java platforms. Instead, register the target Java platform in the and set the target JDK for the proejct.
  • You can manage the project's classpath by clicking Libraries in the left panel of the dialog box. You can set up file filters for JAR file creation by clicking Packaging.
