Registering a Server

See Also

Before you can deploy a web application, JSP, servlet, or EJB module, the server to which you are going to deploy needs to be registered with the IDE. For versions of the IDE that are bundled only with the Tomcat Web Server, the Tomcat Web Server is automatically registered in the IDE. For versions of the IDE that are bundled with both the Tomcat Web Server and SJS Application Server, both servers are automatically registered in the IDE. You must register all other servers yourself.

To register an external server:

  1. Go to Tools > Server Manager.
  2. In the Server Manager, click Add Server.

    3. The Add Server wizard appears and displays the types of servers that are compatible with the IDE.

  3. In the dialog box, select the type of server you want to register and click Next.
  4. Specify server-specific information in the panels that follow and click Finish.
  • Alternatively, right-click the Servers node in the Runtime window. Then choose Add Server from the contextual menu and use the Add Server wizard to register the server.
