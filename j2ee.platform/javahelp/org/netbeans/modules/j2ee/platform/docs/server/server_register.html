Registering a Server

See Also

When you create a new enterprise application, the default server defined in the Runtime window is set as the new enterprise application's server.

To register a new server:
  1. In the Runtime window, right-click the Server Registry node and choose Add Server Instance from the contextual menu.

    The Add Server Instance wizard appears and displays the types of servers that are compatible with the IDE.

  2. In the dialog box, select the type of server you want to register and click Next.
  3. Specify server-specific information in the panels that follow and click Finish.
