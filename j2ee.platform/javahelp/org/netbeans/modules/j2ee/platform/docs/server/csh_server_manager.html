Before you can deploy an enterprise application, web application, JSP file, servlet, or EJB module, the server to which you are going to deploy needs to be registered with the IDE. For versions of the IDE that are bundled only with the Tomcat Web Server, the Tomcat Web Server is automatically registered in the IDE. For versions of the IDE that are bundled with both the Tomcat Web Server and Sun Java System Application Server, both servers are automatically registered in the IDE. You must register all other servers yourself.
