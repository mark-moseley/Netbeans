Starting and Stopping a Server

See Also

  1. In the Runtime window, expand the Server Registry node.
  2. Right-click the server's node and choose Start/Stop Server from the contextual menu. A Server Status dialog box appears showing whether the server is running or stopped.
  3. Click Start Server to start a stopped server, click Start Server (Debug) to start a stopped server in debug mode, or click Stop Server to stop a running server.
