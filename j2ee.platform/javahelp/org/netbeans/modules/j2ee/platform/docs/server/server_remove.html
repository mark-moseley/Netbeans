Removing an Application from a Server

See Also

When you no longer need to run an application that was added to the server using the IDE, you can use the Runtime window to remove the application's context from the server. The IDE recreates the context if the application is subsequently executed from the IDE.

To remove an application from a server:

  1. In the Runtime window, expand the Servers node.
  2. Expand the server's instance node.
  3. If the server is not running, start the server by right-clicking the server, choosing Start/Stop Server from the contextual menu, and clicking Start Server.
  4. Expand the server node and expand the Web Applications node. Note that the Web Applications node does not appear if the server is not running.
  5. Right-click the web application's context node and choose Undeploy from the contextual menu.
