Testing a Web Service on the Service Side

See Also 

You can test a web service from the IDE. Do this right before you expose a web service to make sure that it is working as expected and providing the functionality that you want the client to consume.

To test a web service on the service side:
  1. Deploy the web service.
  2. Prepare the web service for testing.
  3. Expand the Web Services node in the Runtime window, then expand the web service instance node and the SEIPort node. Right-click the web service's operation node and choose Test Operation.

    4. The Test Web Service Operation dialog box appears. You can test the operation and view information such as the operation's input and return data types.

  4. Type values for the parameters, click the Submit button, and view the results.
