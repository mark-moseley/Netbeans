Creating an Enterprise Bean

See Also

You can only create an enterprise bean inside of an EJB module project. You can create an empty enterprise bean or generate a CMP entity bean from a database connection or database schema.

When you create an enterprise bean, the IDE adds a node for the enterprise bean under the project's Enterprise Beans node. This node contains a node for the bean class and nodes for the bean's business methods and CMP fields. The IDE also updates the deployment descriptors for the EJB module and the J2EE module that contains the EJB module.

To create a new enterprise bean:

  1. In the Projects window, select any existing EJB module project.
  2. Choose File  New (Ctrl-N) from the main window.
  3. From the Enterprise category, select one of the EJB templates.
  4. Follow the steps in the rest of the wizard. For more information, see the following topics:
    • New Session Bean Wizard
    • New Message-Driven Bean Wizard
    • New Entity Bean Wizard
    • New CMP Entity Bean Wizard
