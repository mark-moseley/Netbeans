New CMP Bean from Database Wizard: Select Database Tables

See Also 

This wizard generates a set of container-managed persistence (CMP) entity beans from an existing database connection. The IDE generates a CMP bean for each related table with CMP fields for each of the collumns in the database tables. You can also automatically generate finder methods for each of the CMP fields.

You open the wizard by right-clicking the project node or Enterprise Beans node for any EJB module project and choosing New > CMP Bean from Database.

The Select Database Tables page lets you select the tables for which you want to generate beans. Use this page as follows:

  1. Select any tables in the left panel and click the Add button. If the table is linked to other tables through foreign keys, you do not have to add those tables to the list in the right panel. The IDE automatically generates beans for each of the linked tables.
  2. Click Finish.
