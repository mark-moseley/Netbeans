Web Service Attributes Editor: Quality of Service

See Also 

For JAX-WS web services (Java EE 5), the Web Service Attributes Editor can help you to customize various aspects of your application, in relation to its web services and clients. The Quality of Service tab is for configuring components that enable interoperability between Java web services and .Net web services. For details, click here.

The subsections of the Quality of Service tab on the service side are as follows:

  • Service Port Binding
  • Operation
    • Operation
    • Input Message
    • Output Message

The subsections of the Quality of Service tab on the client side are as follows:

  • Transport
  • Security
  • Advanced Configuration
