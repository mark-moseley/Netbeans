For JAX-WS web services (Java EE 5), the Web Service Attributes Editor can help you to customize various aspects of your application, in relation to its web services and clients.

For example, if you have the WSIT modules installed, you can use the WSIT tab to specify advanced web service interoperability settings.

The subsections of the WSIT tab on the service side are as follows:

  • Service Port Binding
  • add Operation
  • Operation
  • Input Message
  • Output Message

The subsections of the WSIT tab on the client side are as follows:

  • Transport
  • Certificates
  • Username Authentication
  • Secure Token Service
