Creating an Enterprise Application Project

See Also 

  1. Choose File  New Project (Ctrl-Shift-N).
      Before you can create an enterprise application project, you must register the SJS Application Server.
  2. From the Web category, select one of the following project templates:
    • Enterprise Application. Creates an empty enterprise application in a .
    • Enterprise Application with Existing Sources. Imports an existing web application into a standard project.
  3. Follow the steps in the rest of the wizard.

    4. The projects are structured according the Project Conventions for Enterprise Applications, which are Java BluePrints Guidelines, available at
