Creating an Enterprise Application Project

See Also 

  1. Choose File  New Project (Ctrl-Shift-N).
      Before you can create an enterprise application project, you must register the Sun Java System Application Server 8.
  2. Select the right template for your project:
    • Populated Enterprise Application. Template that creates a project populated with an empty web module and an empty EJB module.
    • Empty Enterprise Application. Template that creates a project without adding any modules.
  3. Follow the steps in the rest of the wizard.

    The projects are structured according the Project Conventions for Enterprise Applications, which are Java BluePrints Guidelines, available at
