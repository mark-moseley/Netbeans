Creating an Enterprise Application Project

See Also 

  1. Choose File  New Project (Ctrl-Shift-N).
  2. Select the right template for your project:
    • Populated Enterprise Application. Template that creates a project in the system with an additional web module and an additional EJB module. The projects are laid out according the Project Conventions for Enterprise Applications.
    • Enterprise Application. Template that creates a project in the system but does not add any files to the project. This template is best if you would like to start from scratch.
  3. Follow the steps in the rest of the wizard.
