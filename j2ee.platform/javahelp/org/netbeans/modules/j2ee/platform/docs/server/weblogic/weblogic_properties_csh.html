Add Server Instance Wizard: BEA WebLogic Application Server Instance Properties

See Also

You see this page of the Add Server Instance wizard when you are registering a Weblogic Application Server instance.

Use this page to specify the properties of the instance you are registering.

  • Local instance. Select a server instance.
  • Username. Type the username.
  • Password. Type the password.
