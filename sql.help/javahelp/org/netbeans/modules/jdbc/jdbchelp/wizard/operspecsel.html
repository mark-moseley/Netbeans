JDBC Database Wizard: Select Operation Specific Attributes Page

See Also

The JDBC Database wizard enables you to create a WSDL file for accessing tables in a database. The wizard generates default WSDL elements for the following database operations:

  • Insert
  • Update
  • Select
  • Poll

The Select Operation Specific Attributes page allows you to select the columns to include in the generated WSDL file. For each column selected, you can also specify which database operations to include in the generated file.

  • Select. Select the tables to include in the generated WSDL file.
  • Advanced. Opens a window that allows you to select specific columns and database operations to include in the generated WSDL file. By default, all columns and database operations are selected.
