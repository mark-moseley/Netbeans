JDBC Database Wizard: Select Operation Specific Attributes Page

See Also

The JDBC Database wizard enables you to create a WSDL file for accessing tables in a database.

The Operation Specific Attributes page contains the following fields:

  • Select. To select a table, select this checkbox.
  • Table Name. Specifies a table name.
  • Properties. To set the properties, click Advanced.
