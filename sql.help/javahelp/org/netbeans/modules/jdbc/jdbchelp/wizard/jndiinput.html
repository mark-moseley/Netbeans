JDBC Database Wizard: Enter JNDI Name Page

See Also

The JDBC Database wizard enables you to create a WSDL file for accessing tables in a database.

The Enter JNDI Name page contains the following field:

  • JNDI Name. Enter the JNDI name of the data source.

When you click Finish, the wizard generates the WSDL file and the XML schema file in the location that you specified earlier.
