JDBC Database Wizard: Select DataSource and Table Objects Page

See Also

The JDBC Database wizard enables you to create a WSDL file for accessing tables in a database.

The Select DataSource and Table Objects page contains the following fields:

  • Data Source. A drop-down list of all currently available data sources. Select a data source from the list to display the available tables from the data source.
    • Even if there is only one data source listed, you must select it to display the available tables.
  • Available Tables and Selected Tables. Select one or more tables from the list of displayed tables and move them to the Selected Tables list.
