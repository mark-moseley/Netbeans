JDBC Database Wizard: Select DataSource and Table Objects Page

See Also

The JDBC Database wizard enables you to create a WSDL file for accessing tables in a database.

The Select DataSource and Table Objects page contains the following fields:

  • Data Source. Select the data source.
  • Available Tables and Selected Tables. Select the tables by moving them from the Available Tables list to the Selected Tables list.
