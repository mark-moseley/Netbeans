JDBC Database Wizard: Enter WSDL Name Page

See Also

The JDBC Database wizard enables you to create a WSDL file for accessing tables in a database. The WSDL file defines the WSDL port type and binding elements for the supported database operations (insert, update, delete, find, pollrecords).

The wizard also generates an XML schema file corresponding to the tables in the database. This schema file is referenced in the generated WSDL file.

The Enter WSDL Name page contains the following fields:

  • File Name. Enter a name for the WSDL file.
  • Project. This field is read only.
  • Folder. If you want to change the default folder, then click Browse and select the new folder.
  • Created File. This field is read only.
