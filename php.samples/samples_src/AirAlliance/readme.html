What is AirAlliance?

AirAlliance is an imaginary airlines company that lets you book tickets through their web site. This PHP application is created to highlight PHP features supported by NetBeans IDE.

This sample demonstrates the following features:

  • Session Management
  • DB Interaction
  • OOP Concepts

Requirements

Follow these before you run the sample:
  • Install NetBeans 6.1 IDE with PHP support
  • Install and configure PHP Runtime
  • Install and configure MySQL database

What is included?

  • NetBeans PHP Application Project
  • Database SQL Script - Use this script to populate tables and sample records
  • Additional Stylesheet

Getting Started

  1. Firstly, we have to create a new MySQL database and populate it with data:
    1. Select Window > Services from the main menu.
    2. Expand the Databases node, right-click on MySQL Server and select Create Database.
    3. Name the database AirAlliance and confirm the window.
    4. In New Database Connection window specify the User Name and Password to MySQL Database Server and confirm. Default database login username is root and without password.
    5. Open sql/aadb_dump.sql in the AirAlliance project, select the connection to AirAlliance database and run the script. It should execute SQL script successfully.
  2. You can either (both options strongly varies with your PHP runtime configuration):
    • Right-click on the AirAlliance project node and select Upload Files and then select from the same context menu Run Project. Project must have pre-configured upload path to PHP documents directory.
    • Manually copy the web directory to your PHP documents directory and run your web browser with specified path of your PHP documents URL.
  3. Click View Itinerary link in the right pane. You should see some records shown from the database.
  4. Play futher with the sample:)

Project folder's structure

  • classes - Contains some helper classes highlighting Object Oriented Programming.
  • conf - Contains the PHP file (Conf.php) from where you can edit the DB Connection settings.
  • css -Contains the stylesheet used for the website.
  • include - Conains header/footer/navigation PHP files for the web site.
