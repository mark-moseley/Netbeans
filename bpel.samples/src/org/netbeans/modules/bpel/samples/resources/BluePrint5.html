BPEL BluePrint 5: Concurrent Asynchronous Coordination of Events Using BPEL

Creates a sample BPEL Module project and a Composite Application project that illustrate concurrent asynchronous coordination of events. Using this pattern will enable a BPEL developer to support business processes that involve multiple asynchronous web service invocations. Refer to the BPEL BluePrint5 documentation for more information on using this sample.
