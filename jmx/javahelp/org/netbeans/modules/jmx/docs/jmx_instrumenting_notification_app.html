Sending MBean notifications

An MBean can send notifications in any part of its implementation (Attributes, Operations, non exposed method, ...)
This chapter highlights the  things to do in order to send a notification when an attribute is set.
The code extract is based on an MBean that has been updated with JMX Notification support ( delegation to a notification broadcaster and sequence number handling generation). This can be done thanks to JMX MBean Wizard or thanks to the action to update an existing MBean (See Updating existing MBeans chapter).

Adding code to send Attribute Change Notifications

In the editor, select the set<MyAttribute> method name. After your attribute is set, add the following code:

//Send a notification when the attribute is set
Notification n = new Notification(AttributeChangeNotification.ATTRIBUTE_CHANGE, this, getNextSeqNumber());
broadcaster.sendNotification(n);
