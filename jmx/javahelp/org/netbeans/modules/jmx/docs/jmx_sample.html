Creating the JMX Sample Project


This chapter demonstrates how to create and run the JMX Sample project "Anagram Game Managed with JMX".
A tutorial is accessible from http://management.netbeans.org/jmx
It details the required steps to write this sample application.

Creating the sample project

  1. Select File/New Project...
  2. Select the Samples | JMX category
  3. Select the "Anagram Game Managed with JMX"
  4. Click Next>
  5. Type in a project name
  6. Click Finish>
  7. The project is added to the project tree.

Compiling and Running the project

A single action:
  • Compiles the project
  • Runs the Anagram Game
  • Connects JConsole Management console to the Anagram game

  1. Select Run | Run Main Project with Monitoring and Management... (or click on the associated shortcut in the Management toolbar)
  2. JConsole is displayed. You can visualize JVM and the Anagram game management info.
  3. Kill JConsole to terminate the session.
