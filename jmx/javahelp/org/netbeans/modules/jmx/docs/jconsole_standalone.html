JConsole Management Console

JConsole is a JMX-compliant GUI tool that connects to a running JVM and displays some JVM metrics as well as the set of living MBeans.
JConsole is tightly integrated with a Java Application type of Project. See Running and Debugging a Java Application Project chapter for more informations on this feature.

For other types of project, JConsole can also be useful to visualize your management.

To launch the JConsole management console from any kind of project :

  1. Click on Run/Start JConsole... action menu item.
  2. JConsole is displayed.
To configure the JConsole management console :
  1. Goto Options/External tools Settings/JConsole
  2. Update JConsole properties.

Using JConsole to visualize your MBeans

Once JConsole is displayed :

  1. Click on MBeans Tab to visualize your MBean (located under DefaultDomain Tree node).
  2. Select an MBean. Its Attributes, operations and a notification buffer are displayed on the right frame.
  3. To set Attributes, type in directly in the Attributes value column.
  4. To call Operations , select Operations tab. Type your inputs in the dedicated TextField. Click on the operation name button.
  5. To register for notifications, select Notifications Tab then Subscribe button.
  6. You can visualize other infos by selecting Info tab.

For more informations on JConsole, check http://java.sun.com/developer/technicalArticles/J2SE/jconsole.html
