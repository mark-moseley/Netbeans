Adding Attributes / Operations to an MBean


Contextual (enabled only if a Standard MBean is selected) actions are located under the management menu. You can access the management menu from :
  • Top level menu
  • Project tree node contextual menu (right click on selected node)

Actions applicability


These actions are enabled for Standard MBean only (MBean that follow the Standard MBean JMX design pattern).
Current tooling doesn't allow for Dynamic MBean attributes/operations update.
These actions apply to any Standard MBean. You don't need to have them generated from the MBean wizard.
MBeans imported from other context can be updated.

Using the Add MBean Attributes... action


  1. From the management menu select Add MBean Attributes...
  2. A windows is displayed with a Table containing existing attributes. You can't update nor remove existing attributes.
  3. Add attributes using Add Attribute button.
  4. Enter valid name, type and description for each added attribute.
  5. You can remove added attribute using Remove Attribute button.
  6. Click OK when you are finished
  7. New Attributes are added at the end of the MBean interface and class.
  8. The update MBean class file is automaticaly opened in the editor.

Using the Add MBean Attributes... action


  1. From the management menu select Add MBean Operations...
  2. A windows is displayed with a Table containing existing operations. You can't update nor remove existing op[erations.
  3. Add operations using Add Operation button.
  4. Enter valid name, parameters type, exceptions and description for each added operation.
  5. You can remove added operation using Remove Operation button.
  6. Click OK when you are finished
  7. New Operations are added at the end of the MBean interface and class.
  8. The update MBean class file is automaticaly opened in the editor.
