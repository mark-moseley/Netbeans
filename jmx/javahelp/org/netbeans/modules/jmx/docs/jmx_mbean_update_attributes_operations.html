Adding Attributes / Operations to an MBean


Contextual actions (enabled only if a Standard MBean is selected) are located under the management menu. You can access the management menu from :
  • The contextual menu for the project tree node (right click on selected node)

Applicability of Actions


These actions are enabled for Standard MBeans only (MBeans that follow the Standard MBean JMX design pattern).
The current tooling doesn't allow you to update Dynamic MBean attributes or operations.
These actions apply to any Standard MBean. You don't need to have generated them using the MBean wizard.
MBeans imported from other contexts can be updated.

Using the Add MBean Attributes Action


  1. From the JMX menu select Add MBean Attributes...
  2. A window is displayed with a Table containing the existing attributes. You cannot update or remove existing attributes.
  3. Add attributes using the Add Attribute button.
  4. Enter a valid name, type and description for each added attribute.
  5. You can remove an added attribute using the Remove Attribute button.
  6. Click OK when you have finished
  7. New Attributes are added at the end of the MBean interface and class.
  8. The updated MBean class file is automatically opened in the editor.

Using the Add MBean Operations Action


  1. From the JMX menu select Add MBean Operations
  2. A window is displayed with a Table containing the existing operations. You cannot update or remove existing operations.
  3. Add operations using the Add Operation button.
  4. Enter a valid name, parameter type, exceptions and description for each added operation.
  5. You can remove added operation using the Remove Operation button.
  6. Click OK when you have finished
  7. New Operations are added at the end of the MBean interface and class.
  8. The updated MBean class file is automatically opened in the editor.
