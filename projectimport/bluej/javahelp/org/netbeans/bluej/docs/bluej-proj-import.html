Welcome to NetBeans IDE BlueJ Edition. If you have been using BlueJ to learn Java technology, NetBeans IDE BlueJ Edition is a logical next tool for learning Java programming and creating your first real world applications.

This BlueJ Edition of NetBeans IDE is a scaled down version of the NetBeans IDE that is customized for people who are used to using BlueJ. You can create new projects and files much as you would in BlueJ and then take advantage of NetBeans IDE coding productivity and project management features to extend the range of your applications.

NetBeans IDE provides a wealth of features that make it easier to code applications of all sizes. These features include:

  • Code completion, which helps you find and quickly fill in classes, methods, and fields.

These help files are optimized for the NetBeans IDE standard distribution, not this special BlueJ Edition of NetBeans IDE.

When you work with BlueJ projects in NetBeans IDE, you have access to many features that are useful for reducing coding time and creating larger programs. The following are just a few of the advantages that NetBeans IDE provides: *make a note about code formatting *abbreviations *easier tracking of coding errors (with error stripe and error reporting) The BlueJ Edition of NetBeans IDE is designed so that you can work in either of or a combination of the following two ways: *Keep working with the BlueJ project structure but take advantage of the NetBeans IDE coding productivity features. *Convert to the NetBeans IDE project structure in order to create larger, more scalable projects
