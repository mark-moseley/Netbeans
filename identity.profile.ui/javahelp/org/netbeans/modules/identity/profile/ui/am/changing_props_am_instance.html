Changing the Properties of Sun Java System Access Manager Instances

See Also

You can modify the properties of a Sun Java System Access Manager instance after you have added it in the Runtime window.

  • You cannot modify the properties of the Default Instance, which is the Sun Java System Access Manager instance that was installed and configured during the installation of the NetBeans Enterprise Pack.

To change the properties of an instance of the Sun Java System Access Manager:

  1. In the Runtime window, right-click the Sun Java System Access Managers node and choose Properties.
    The Edit server-instance dialog box opens.
  2. Make your changes and click OK.
