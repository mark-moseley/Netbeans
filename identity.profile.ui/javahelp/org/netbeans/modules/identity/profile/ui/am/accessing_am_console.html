Accessing the Sun Java System Access Manager Instance Admin Console

See Also

To access the Admin Console for a Sun Java System Access Manager instance:

  1. In the Services window, expand the Sun Java System Access Managers node.
  2. Right-click the instance for which you want to access the Admin Console and choose View Admin Console.
    The login screen for the Admin Console launches in a web browser.
