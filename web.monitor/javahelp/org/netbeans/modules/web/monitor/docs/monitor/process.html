Working with the HTTP Monitor

The following table shows how to work with the HTTP Monitor.

Set up the HTTP Monitor
  1. Ensure that the HTTP Monitor is configured correctly.
  2. Ensure that the HTTP Monitor is enabled for the web server.

Analyze Session Data
  1. Run the web application.
  2. Use the HTTP Monitor to analyze session data.


Optional

  • Save session data.
  • Edit and replay session data.
