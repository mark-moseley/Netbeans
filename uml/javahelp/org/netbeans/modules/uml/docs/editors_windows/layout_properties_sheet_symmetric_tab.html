Layout Properties Sheet: Symmetric Tab

See Also

The Symmetric tab of the Layout property sheet lets you specify several options for any diagram that uses the Symmetric layout. The table below describes these options.

Section Option Description
Spacing Options Node Spacing The distance between elements. You can set a node spacing value to adjust the spacing in the final layout. The symmetric algorithm uses this parameter as a guide, so that it is possible that the spacing for any particular pair of nodes might be other than you specified. The larger the node spacing you specify, the farther
apart the nodes are in the final layout.
Layout Quality Draft Specifies the speed of the algorithm used to produce the layout. Draft represents the lowest quality, but fastest layout. Proof represents the highest quality, but slowest layout.
Default
Proof
