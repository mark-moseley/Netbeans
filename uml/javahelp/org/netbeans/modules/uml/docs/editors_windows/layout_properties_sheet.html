Layout Properties Sheet

See Also

The Layout properties sheet lets you specify how you want your UML system diagrams to be laid out. For most types of diagrams, the Layout property sheet offers a selection of options to help you fine-tune each layout style for the specific needs of your diagrams and your UML Project.

Tabs

  • Disconnected Tab. Lets you specify connectedness between diagram elements and between nodes, and the aspect ratio.
  • Hierarchical Tab. Lets you specify orientation, level alignment, layout quality, spacing, routing, and other parameters for diagrams that use the hierarchical layout.
  • Orthogonal Tab. Lets you specify node and edge spacing, layout quality, and aspect ratio for diagrams that use the orthogonal, or squared-line, layout.
  • Symmetric Tab. Lets you specify node spacing options and layout quality for diagrams that use the symmetric layout.
  • Routing Tab. Lets you specify horizontal and vertical spacing, and whether to fix node sizes and positions.
  • General Tab. Not Supported.

Command Buttons

  • OK. Accepts your specifications for all applicable diagrams, and then closes the dialog box.
  • Cancel. Closes the Layout property sheet without applying any of your settings.
  • Apply. Applies your specifications to all applicable diagrams without closing the dialog box.
  • Layout. Applies your specifications to all applicable diagrams without closing the dialog box, and immediately displays any specification changes in the active Diagram editor.
  • Reset. Resets all values in the dialog box to the values that were displayed when you opened the dialog box.
  • Defaults. Resets all values in the dialog box to the values that were originally specified when you installed the IDE.
