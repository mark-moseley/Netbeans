Navigating Between Instances

The IDE lets you navigate a diagram by centering the Diagram window on the target object, its incoming object, or its outgoing object. You can use this navigation method by right-clicking an object on the diagram's gray header.

This feature is useful for quickly reaching a common reference point for components in a large, complex diagram.

Navigating Between Classifiers

The IDE lets you navigate the diagram by centering the Diagram window on the target classifier in its corresponding Class diagram. You can use this navigation method by right-clicking an object on the diagram's gray header.
