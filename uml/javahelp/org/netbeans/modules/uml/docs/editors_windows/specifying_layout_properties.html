Specifying Layout Properties

See Also

The IDE's UML Modeling feature includes the Layout properties sheet, which lets you specify how to lay out your system diagrams. By arranging your system diagrams with your preferred layouts, you can facilitate your workflow.

To specify layout properties:

  1. Right-click the background of the target UML Diagrams window.
  2. Under Layout, choose Properties.
    The IDE opens the Layout properties sheet.
  3. Specify the layout properties and apply your changes by clicking OK, Apply, or Layout.
    See Layout properties sheet for details about each tab and the command buttons.
