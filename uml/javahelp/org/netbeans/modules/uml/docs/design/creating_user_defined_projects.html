Creating User-Defined Projects

See Also

The UML Design Center has three standard pattern projects. In addition, you can create your own user-defined projects to contain the patterns you create. You create a user-defined project in the open UML Design Center window.

To create a user-defined project in the UML Design Center:

  1. Expand the Design Center node.
  2. Right-click the Design Pattern Catalog node and choose New > Project. The New Wizard is displayed.
  3. Type a name for your project in the Project Name field.
  4. Use the browse button to choose the location where your project files should be stored.
  5. Click OK.
    Your project appears in the UML Design Center.
  6. To save the design pattern catalog with your newly created project, right-click the Design Pattern Catalog node and choose Save.

Projects in the UML Design Center have virtually the same functionality as projects appearing in the Projects window. You can create packages and diagrams just as you would using the Projects window and the Diagram editor.
