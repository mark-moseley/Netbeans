Sharing Patterns

After creating your own pattern to solve a specific software development problem, you might want to share the pattern with other users.

First, if you have not already done so, promote your user-defined pattern to the UML Design Center. After that step, sharing the pattern is as easy as copying a folder to another filesystem or computer.

To share your pattern with other users:

  1. In the UML Design Center window, select a project to share.
  2. In the Properties window, locate the folder containing the project by selecting the File Name property.
  3. Copy the entire folder to the target location.
  4. At the target location, insert the project into the UML Design Center window.
  5. Be sure to save the design pattern catalog. To do so, right-click the DesignPatternCatalog node and select Save.

You are ready to apply the pattern. See Applying Patterns for more details.
