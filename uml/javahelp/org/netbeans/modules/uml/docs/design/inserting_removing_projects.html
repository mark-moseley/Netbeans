Inserting and Removing Projects

See Also

To manage space in the UML Design Center, you can remove projects that you are no longer using and insert new ones. This means that you can share your patterns with other users, ensuring consistency and standardization in your design pattern solutions.

You insert projects into and remove projects from the open UML Design Center window.

To remove a project from the UML Design Center:

  1. Expand the Design Center node and the DesignPatternCatalog node.
  2. Select the project you want to remove.
  3. Right-click the project and choose Remove > Remove Project From Workspace.
    The project is removed from the design pattern catalog.
  4. Save the design pattern catalog by right-clicking the DesignPatternCatalog node and choosing Save.

To insert a project into the UML Design Center:

  1. Expand the Design Center node and the DesignPatternCatalog node.
  2. Right-click the DesignPatternCatalog node and choose Insert > Insert Project Into Workspace.
    The Open dialog appears.
  3. Navigate to the folder containing the UML Modeling Project File (*.etd) and click Open.
    The project is inserted into the UML Design Center.
  4. Save the design pattern catalog.
