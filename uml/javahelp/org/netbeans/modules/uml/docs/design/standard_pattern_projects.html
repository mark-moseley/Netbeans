Standard Pattern Projects

The design pattern catalog in the UML Design Center contains the following standard pattern projects:

Project Contents
EJB1.1 Patterns that can be used in applications based on the Enterprise JavaBeans Specification version 1.1.
EJB2.0 Patterns that can be used in applications based on the Enterprise JavaBeans Specification version 2.0.
GoF Patterns developed by the Gang of Four (GoF) to solve common problems in object-oriented software development.
