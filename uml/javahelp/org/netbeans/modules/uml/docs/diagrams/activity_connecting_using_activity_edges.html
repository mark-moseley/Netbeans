Connecting Nodes, Invocations, Decisions, and Final Flow Nodes Using Activity Edges

See Also

You can quickly connect Nodes, Invocations, Decisions, and Final Flow Nodes using Activity Edges.

To connect two elements using an Activity Edge:

  1. Right-click a Node in the Diagram Editor.
  2. Click the Activity Edges icon in the context palette.
  3. In the Diagram editor, click a Node, Invocation, Decision, or Final Flow Node and drag the Activity Edge to another Node, Invocation, Decision, or Final Flow Node.
    The two elements appear connected.
