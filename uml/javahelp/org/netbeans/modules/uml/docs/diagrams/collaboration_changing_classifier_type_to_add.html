Changing the Type of Classifier to Add

You can change the types of Classifiers to add to diagrams.

To change the type of classifier to add to a diagram:

  1. On the Modeling menu, choose Preferences.
    The Preferences Editor opens.
  2. In the Preferences Editor, expand the New Project node.
    The New Project information is listed.
  3. Select Unknown Classifier.
  4. In the Details box, choose the Type to Create box, and then choose one of the following types of Classifiers:
    • DataType
    • Class
    • Interface
  5. Click OK.
