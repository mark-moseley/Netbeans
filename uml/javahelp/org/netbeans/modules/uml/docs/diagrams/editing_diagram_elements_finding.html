Finding Diagram Elements in the Projects Window

You can locate any diagram element in the Projects window from its native diagram.

To find a diagram element:

  • In a diagram that is open in the Diagram editor, right-click the element and choose Select in Model .
    The IDE highlights the name of the element in the Projects window.
  • The IDE also offers several methods for using links or message lines to locate elements in diagrams. For more information on these methods, see Navigating Within Diagrams.
