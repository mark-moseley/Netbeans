Working With Messages

You can work with messages in Collaboration and Sequence diagrams as described in the following table:

Item Description/Action
Message A message represents a call from one instance to another on a Collaboration or Sequence Diagram.
Message Numbers In all Collaboration and Sequence Diagram messages and operations, you can toggle message numbers off and on. To toggle numbers, right-click the Diagram editor and choose Show Message Numbers.
Interaction Message Numbers In all Collaboration and Sequence Diagram messages and operations, you can toggle interaction message numbers off and on. To toggle numbers, right-click the Diagram editor and choose Show Message Numbers.
Numbering Messages In all Collaboration and Sequence Diagram messages and operations, you can toggle numbering message off and on. To toggle interaction message numbers, right-click the Diagram editor and choose Show Message Numbers.
