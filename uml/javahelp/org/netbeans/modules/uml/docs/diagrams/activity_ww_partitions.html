Working With Partitions

See Also

The IDE's UML Modeling feature supports partition functionality so that you can group Activity Nodes and edges. For example, you can divide the nodes and edges to constrain and show a view of these containments, which correspond to organizational units in business-process modeling. The IDE lets you:

  • Create Partitions
  • Add and modifying Partition attributes
  • Add Subpartitions
  • Name Subpartitions
  • Delete Subpartitions
