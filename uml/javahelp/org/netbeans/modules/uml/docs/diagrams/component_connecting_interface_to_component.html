Connecting an Interface to a Component Element

See Also

You can connect an Interface to a Component element.

To connect an Interface to a Component element:

  1. In the Diagram editor, click the Select button to clear all selections.
  2. In the Modeling Palette, click the Interface icon .
  3. In the Diagram editor, click the Component element.
    The IDE creates a Port on the Component element.
  4. Click the Interface element.
    The IDE connects the Component and the Interface elements.
