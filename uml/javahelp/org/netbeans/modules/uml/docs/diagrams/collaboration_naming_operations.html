Naming Operations

You can name operations on a Collaboration diagram or in the Properties window.

To name an operation on a Collaboration diagram:

  • In the target Diagram window, double-click the target Operation, and then type the name.

To name an operation in the Properties window:

  1. In the target Diagram window, select the target operation.
    The IDE displays the Operation information in the Properties window.
  2. In the Properties window, expand the nodes to drill down to the details of the target Operation.
  3. Click the target boxes, and then modify the Operation and/or add Parameters.
