Adding a Classifier

You can populate diagrams with Classifiers.

To add a classifier:

  1. In the Modeling Palette for Collaboration diagrams, select the Lifeline icon .
  2. Click anywhere in the Diagram editor.
    A Lifeline is added to the diagram.
  3. Double-click the right side of the lifeline to enter a new or existing classifier.

  • If the Classifier you specify for this element is not found, the IDE prompts you to add Classifier.
