Associate Dialog Box

See Also

The Associate dialog box lets you associate any element with any other element. You can also associate an element with a diagram, or a diagram with another diagram. This dialog box has the following components:

  • Find what. Type the search string you want the IDE to find. Use the drop-down list to select a recent search string.
  • Match Case. Specify whether you want case to be considered during the search.
  • This is an XPath Expression. Enable searches for XPath expressions. This option can be very useful for power users who require very specific UML code searches. For more information on XPath expressions, visit: .
  • Also search alias. (Optional) Specify whether you want to search for the alias.
  • Search In. (Optional) Specify whether you want to search in the names or descriptions of the project elements.
  • Bottom grid. Any results of the latest search are displayed in a grid format. Double-click any row in this grid to center its corresponding element in a Diagram window.
  • Find button. Click to execute the string search.
