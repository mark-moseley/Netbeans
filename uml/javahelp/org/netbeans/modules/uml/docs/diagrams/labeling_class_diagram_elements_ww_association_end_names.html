Working with Association End Names

The IDE lets you assign end names to the ends of an association. When you assign end names, they appear in the Properties window under the Ends node, and in the association's Class diagram. You can show and hide these end names in their Diagram window.

To show or hide association end names:

  • In the target association Diagram window, right-click the target end of the target association, select Labels and then choose Both End Names.
    The IDE displays or hides the end name, as applicable.

To assign a new end name:

  1. In the target association Diagram window, double-click the target association end name.
    The end name becomes editable.
  2. Type the new end name and click the diagram background.
    The IDE accepts the change, displaying the end name at the specified end of the association, as well as under the Association Ends node of the Properties window.
  • The IDE also lets you change an association's end names from the Properties window. Select an association to display its information in the editor, and then double-click an end name to edit it.
