Renaming Attributes

You can rename the attributes of a modeling element from either of two places in the IDE.

To rename an attribute from the Projects window:

  1. Right-click the attribute node and choose Rename.
    The Rename dialog box appears with the whole attribute shown in the New Name field.
  2. In the field, select the part of the attribute that you want to change, such as the attribute's name.
  3. Type the new name, and then click OK.
    The new name is displayed in the Projects window. It is also displayed in the corresponding element of the diagram, if the diagram is open in the Diagram editor.

To rename an attribute from the Diagram editor:

  1. In the diagram, in the class element, double-click the attribute.
    The attribute name becomes editable.
  2. In the editable field, type the attribute's new name, and then do one of the following:
    • Click outside the class element.
    • Press Enter.
    The name change is displayed in the diagram and on the corresponding node in the Projects window.
