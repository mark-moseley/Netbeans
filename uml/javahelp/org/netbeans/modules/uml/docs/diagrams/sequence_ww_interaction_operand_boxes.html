Working With Interaction Operand Boxes

Each section of a combined fragment box is separated by a dotted line and represents a different interaction operand for the interaction operator displayed in the top box tab. After you create a combined fragment box in a Diagram window, you can work with each interaction operand box in the following ways:

  • Insert a new interaction operand box above any combined fragment's selected interaction operand box
  • Add a new interaction operand box at the bottom of all boxes in the combined fragment
  • Delete the currently selected interaction operand box
  • Resize the currently selected interaction operand box

To insert an interaction operand box:

  • In the target Diagram window, right-click the target interaction operand box, and, next to Interaction Operand, choose Insert New Above.
    The IDE inserts a new box above the target box.

To add an interaction operand box:

  • In the target Diagram window, right-click any interaction operand box, and, under Interaction Operand, choose Add New to Bottom.
    The IDE adds a new box at the bottom of the combined fragment.

To delete an interaction operand box:

  • In the target Diagram window, right-click the target interaction operand box, and, under Interaction Operand, choose Delete.
    The IDE deletes a new box at the bottom of the combined fragment.

To resize an interaction operand box:

  1. To resize the entire target combined fragment, in the target Diagram window, drag any handle bordering the combined fragment box.
    The IDE resizes the entire combined fragment box, including all interaction operand boxes, accordingly.
  2. To vertically resize a single interaction operand box, click the target box's dotted line and drag up or down.
    The IDE vertically resizes the target interaction operand boxes.
