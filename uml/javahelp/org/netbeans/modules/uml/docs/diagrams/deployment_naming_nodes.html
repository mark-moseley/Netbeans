Naming Nodes

You can name UML Nodes on a deployment diagram and in the Properties window.

To name a node on a deployment diagram:

  • In the target Diagram window, double-click the target Node, and then type the name.

To name a node in the Properties window:

  1. In the target Diagram window, select the target Node.
    The IDE displays the target Component in the Properties window.
  2. Choose the Name list, and then type the new name.
