Assigning a Preset Multiplicity Value to an Association

The IDE lets you assign preset multiplicity values to an association from its Diagram window. You can assign these values in the Diagram window only when the association multiplicity value is shown.

To assign a new preset multiplicity value:

  • In an association Diagram window, right-click the target end of the target association, and then choose a value.
    The IDE accepts the value and displays it at the specified end of the association, as well as under the Ends node of the Properties window.
