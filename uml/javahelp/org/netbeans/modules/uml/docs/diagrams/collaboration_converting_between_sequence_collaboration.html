Converting Between Sequence and Collaboration Diagrams

You can convert a Sequence diagram into a Collaboration diagram and vice versa.

To convert between sequence and collaboration diagrams:

  1. In the Projects window, expand the Model node for the diagram you want to convert.
  2. Right-click the interaction node for the diagram.
    The interaction node has the Interaction icon .
  3. Select Create Diagram from Selected Elements from the pop-up menu.
    The Create New Diagram dialog box opens.
  4. In the Create Diagram dialog box, select the type of diagram you want to create and type the name for the new diagram.
  5. Click OK.
    The IDE converts the Sequence diagram to a Collaboration diagram or converts the Collaboration diagram to a Sequence diagram with the new name.
