Creating a Scoped Diagram

To create a scoped diagram:

  1. In the Projects window, expand a project node, right-click the Model node, and choose Add > Package.
    The New - Create Package wizard opens.
  2. In the Name field, type a name for the package.
  3. In the Namespace field, select the name of the project that you want to house the diagram.
  4. Select the Create Scoped Diagram checkbox.
  5. In the Diagram Name field, type a name for the new diagram.
  6. In the Diagram type box, choose a diagram type.
  7. Click OK.
