Setting General Code Engineering Preferences

See Also

Code engineering preferences can be set for general behavior during model element creation, deletion or manipulation.

To set general preferences:

  1. Choose Tools > Options.
  2. In the Options window, click the Advanced Options button.
  3. Expand the UML node and select the Code Engineering node.
    The Code Engineering preferences appear in the Properties pane of the Options window.
  4. In the Properties pane, define your preferences.
  5. Click Close.
