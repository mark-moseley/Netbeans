Setting Java Code Engineering Preferences

See Also

Code Engineering settings let you select preferences for behavior when Java source code is being generated.

To set Java preferences:

  1. Choose Tools > Options.
  2. In the Options window, click the UML category.
  3. Click the Java Code Generation tab and define your preferences in the pane.
  4. Click OK to save your settings.
