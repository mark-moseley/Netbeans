Deleting Projects

See Also

  1. Right-click the UML project's node in the Projects window and choose Delete Project.
  2. Click Yes to delete the UML project and all the source files under the UML project directory.
    The node for the UML project is removed from the Projects window.
  • The Java project and Java source files associated with the deleted UML project are not deleted.
