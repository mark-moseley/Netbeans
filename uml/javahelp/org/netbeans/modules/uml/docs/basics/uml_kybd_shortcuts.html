Keyboard Shortcuts: UML Modeling

See Also

The UML Modeling tool provides keyboard shortcuts that enable you to use the tool without the use of a mouse. Keyboard shortcuts are available for the following UML Modeling areas and functionalities:

  • Diagram Editor. Keyboard shortcuts for commands available in the Diagram editor and the Diagram toolbar.
  • Working with Diagram Elements. Create and handle diagram elements using keyboard shortcuts.
  • Modeling Palette. Navigate the UML Modeling Palette window using keyboard shortcuts only.
  • Documentation Window. Use the functionalities available in the Documentation window with keyboard shortcuts.
  • Design Center Window. Lists the keyboard shortcuts available in the Design Center window.
