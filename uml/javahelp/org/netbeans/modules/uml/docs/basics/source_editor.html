Source Editor

See Also

The Source Editor displays source code for any element you select in the Diagram editor. You can hide and display the Source Editor as needed. When you display the Source Editor, it updates the source code for any element changes you make in the active diagram. Contextual menu options let you configure the display of code in the Source Editor, navigate within a source file, and manipulate code.

To display source code in the Source Editor:

  • Right-click an element node in the Projects window and choose Navigate to Source.
