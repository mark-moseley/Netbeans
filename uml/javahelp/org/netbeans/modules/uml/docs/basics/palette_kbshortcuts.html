Keyboard Shortcuts: UML Modeling Palette

See Also

Use the following keyboard shortcuts to access the Modeling Palette and its contents:

Keys Action
Ctrl-Shift-8 Places focus on the Modeling Palette window.
TAB Selects the first element (Class) in the palette .
Up Arrow Traverses up through the elements on the palette.
Down Arrow Traverses down through the elements on the palette.
Ctrl-Enter Places the currently selected element from the palette to the active diagram in the Diagram editor.
