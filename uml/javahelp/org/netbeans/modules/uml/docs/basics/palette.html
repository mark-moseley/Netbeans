Modeling Palette

See Also

When a diagram is open in the Diagram editor, the Modeling Palette window is also available. This palette appears populated with icons for elements that are appropriate to the type of diagram. For example, when a Class diagram is active, you see the Class Modeling Palette.

Like the Properties and Documentation windows, the Modeling Palette can be docked and undocked.

To create a new element in a diagram:

  • Select an icon in the Modeling Palette, drag it to the Diagram editor, and click once.

To create a second new element like the first:

  • In the Diagram editor, move to another place and click again.

To stop replicating the same element:

  • In the Diagram editor, right-click.

To name an element:

  • Select the element (or leave it selected) and type a name.

The following topics describe the elements available in the Modeling Palette for each type of diagram:

  • Activity Diagram Elements
  • Class Diagram Elements
  • Sequence Diagram Elements
  • State Diagram Elements
  • Use Case Diagram Elements
