Documentation Window

See Also

The Documentation window displays the description of any UML model element, saving that description to the corresponding project's *.etd file. When you select an element in the Diagram editor or a node in the Projects window, that element's description is displayed in the Documentation window.

You can document any project, diagram, or element that appears in the Diagram editor or Projects window by selecting the element and then typing text into the Documentation window.

To display the UML Documentation window:

  • Choose Window > Other > UML Documentation.
    The UML Documentation window appears as a new window in the IDE.

The table below describes the functionality available on the Documentation windows' toolbar:

Functionality Icon Description
Bold Bolds the highlighted text. Highlight and click again to remove bold.
Italicize Italicizes the highlighted text. Highlight and click again to remove italics.
Underline Underlines the highlighted text. Highlight and click again to remove underline.
Font Color Opens a color grid dialog box where you can select a new color for your fonts in the Documentation window.
Align Left Lets you align all selected text to the left in the Documentation window.
Align Center Lets you center all selected text in the Documentation window.
Align Right Lets you align all selected text to the right in the Documentation window.
