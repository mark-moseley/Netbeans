Diagram Elements

See Also

Within any UML Modeling project, you can create and name elements, assign them attributes and operations, and diagram their relationships to other elements in diagrams. UML Modeling diagrams display each element as an icon in the Diagram editor and in the Projects window. An icon symbolizing each type of element that is appropriate to a diagram type appears in the Modeling Palette window when a diagram of that type is active in the Diagram editor.

You can place an element in a diagram in three ways:

  • Drag an icon from the Modeling Palette
  • Drag an instance of the element from the Projects window into a diagram in the Diagram editor
  • Select an element in one diagram, choose Edit > Copy, right-click in another diagram, choose Paste, and click to place the copy.

For information about specific diagram elements, see the following topics:

  • Activity Diagram Elements
  • Class Diagram Elements
  • Sequence Diagram Elements
  • State Diagram Elements
  • Use Case Diagram Elements
