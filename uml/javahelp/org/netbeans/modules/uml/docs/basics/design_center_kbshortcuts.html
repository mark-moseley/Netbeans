Keyboard Shortcuts: UML Design Center

See Also

Use the following keyboard shortcuts to use the UML Design Center:

Keys Action
Ctrl-Enter Associates the requirement with the selected element on the diagram.
Alt-Shift-W Places focus on the UML Design Center window.
Shift-F10 Displays the pop-up menu of UML Design Center commands available for the selected node.
Up Arrow Traverses up through the elements on the palette.
Down Arrow Traverses down through the elements on the palette.
DEL Deletes the currently selected element.
