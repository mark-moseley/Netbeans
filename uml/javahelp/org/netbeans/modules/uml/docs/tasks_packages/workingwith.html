Working With Packages

The IDE lets you do the following with any package:

  • Create packages
  • Rename packages
  • Delete packages
  • Change package element display

You can work with packages in these ways directly from the Projects window and from any diagram open in the Diagram editor. In the Projects window, you can also drag packages onto the following:

  • Any Actor in the Projects window
  • Any package in the Projects window
  • Any Class in the Projects window
  • Any diagram open in the Diagram editor

When you drag a package onto an Actor or Class element, or onto another package listed in the Projects window, that package becomes a sub-node of the element in the Projects window.
