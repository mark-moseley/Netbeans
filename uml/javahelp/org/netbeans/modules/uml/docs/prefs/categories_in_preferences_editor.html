UML Categories in the Options Window

See Also

UML Modeling offers several categories of configurable preferences. The table below describes the nodes in the UML tree of the IDE's Options window.

Node > Subnode Description
UML Includes overall preferences that control the behavior of UML Modeling:
  • Automatically Hide Modeling Windows
  • Delete File when Deleting Artifacts
  • Show Aliases
  • Prompt to Save Diagram
  • Prompt to Save Project
Code Generator Includes the following property that controls the behavior of the Generate Code action and dialog:
  • Show Process Output
Design Center Preferences for the Design Center.
Design Center > Design Pattern Catalog Includes the following property for the Design Pattern Catalog:
  • Overwrite Existing Participants
Diagrams Includes the following properties for all diagram types:
  • Ask Before Layout
  • Automatically Size Elements
  • Display Compartment Titles
  • Display Edit Control Tooltips
  • Display Empty Lists
  • Reconnect to Presentation Boundary
  • Resize with Show Aliases Mode
  • Show Stereotype Icons
Diagrams > Activity Includes the following property for Activity diagrams:
  • Indicate Interruptible Edges
Diagrams > Collaboration Includes the following properties for Collaboration diagrams:
  • Delete Connector Messages
  • Show Message Numbers
Diagrams > Sequence Includes the following properties for Sequence diagrams:
  • Automatically Create Classifiers
  • Classifier Type to Create
  • Delete Combined Fragment Messages
  • Group Operations by Classifier
  • Move Invoked Operation
  • New Message Action
  • Show Interaction Boundary as Created
  • Show Message Numbers
Display Includes the following property for controlling the display of various items:
  • Tagged Values
Expansion Variables Includes the following variable that is specific to an individual machine:
  • Configuration Location
Find Dialog Includes the following property that controls the behavior of the Find dialog box:
  • Allow Lengthy Searches
Find Dialog > Displayed Columns Includes the following properties to control which columns to display in results:
  • Alias
  • Fully Scoped Name
  • Icon
  • ID
  • Name
  • Project
  • Type
Information Logging Includes the following properties that control the type of information that is written to a log file while running within an IDE:
  • Log Messages
  • Log Errors
  • Log Exceptions
  • Log Informational Messages
  • Log Method Entry
  • Log Method Exit
  • Log File Name and Location
New Project Includes the following properties for all new projects:
  • Create New Diagram
  • Default Diagram Name
  • Default Element Name
New Project > Unknown Classifier Includes the following property for handling references to classifiers that do not exist in the project or any of the imported packages:
  • Create Classifier
New Project > Unknown Stereotype Includes the following property for handling stereotypes that are not part of a known profile:
  • Automatically Create
Notify When Namespace Is Deleted Includes the following properties for determining whether or not elements are notified when their enclosing namespace is deleted from a project:
  • Actor Elements
  • Association Class Elements
  • Class Elements
  • Design Pattern Role Elements
  • Interface Elements
  • Package Elements
  • Use Case Elements
Presentation Includes the following properties for presentation:
  • Documentation Font
  • Grid Font
  • Set Global Colors and Fonts
Reverse Engineering Includes the following properties for behavior during reverse engineering:
  • Show Process Output
Reverse Engineering > Operations Element Includes the following properties for behavior during reverse engineering action on source projects and files:
  • Create New Operation
  • Prompt for Source Folders
  • Source Folders Config File
Code Engineering Includes the following properties for behavior during model element creation, deletion, and manipulation:
  • Transform When Code May Be Lost
  • Warn When Impacted Count Reaches
Code Engineering > Elements Includes the following properties for determining which diagram elements are affected during model element creation, deletion, and manipulation:
  • Aggregation
  • Association
  • Association End
  • Attribute
  • Class
  • Generalization
  • Implementation
  • Interface
  • Multiplicity
  • Multiplicity Range
  • Navigable End
  • Operation
  • Package
  • Parameter
  • Project
  • Template Parameter
  • Enumeration
  • Enumeration Literal
Code Engineering > Java Includes the following properties for behavior during source code creation using the Java programming language:
  • Capitalize Attribute Name in Accessors
  • Collection Override
  • Create Accessor Methods
  • Create Constructor Methods
  • Create Finalize Methods
  • Display Duplicate Operation Dialog
  • Modify Redefined Operations
  • Name Navigable Ends
  • Prefix for Member Attribute
  • Prefix for Read Accessors
  • Prefix for Write Accessors
  • Remove Prefix From Accessor Names
UML Properties Pane Includes the following properties for the UML Properties window:
  • Default Filter
  • Display Type Fully Scoped Name
  • Maximum Display
Web Report Includes the following properties that control the generation of model reports:
  • Include Hidden Tagged Values
  • Maintain Directory Structure
