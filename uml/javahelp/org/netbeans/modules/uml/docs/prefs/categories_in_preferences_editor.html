UML Categories in the Options Window

See Also

UML Modeling offers several categories of configurable preferences. The table below describes the tabs in the UML category of the IDE's Options window.

Tab Name Description
General Includes overall preferences that control global settings for handling UML Modeling projects and diagram types:
  • Project
    • Open Diagrams
    • Show Aliases by Default
  • Diagrams
    • Automatically Resize Elements
    • Display Compartment Titles
    • Display Empty Lists
    • Reconnect to Presentation Boundary
    • Resize with Show Aliases Mode
    • Show Stereotype Icons
    • Gradient Background
    • Ask Before Layout
    • Set Global Colors and Fonts
Templates Includes the properties you set for the templates to use during your UML Modeling session. You can add new templates or edit or remove existing templates. See Configuring Domain Templates Options for more information.
Java Code Generation Includes the following settings that controls the behavior of the Generate Code action and dialog:
  • Create by Default
    • Accessor Methods
    • Constructor Methods
    • Finalize Methods
  • Accessors
    • Remove Prefix from Name
    • Capitalize Attribute Name
    • Prefix for Write Accessors
    • Prefix for Read Accessors
    • Prefix for Member Attributes
  • Collections
    • Collection Override Default
    • Collections Use Generics Default
  • Miscellaneous
    • Display Duplicate Operation Dialog
    • Name Navigable Ends
Dialogs

Includes the following properties to customize whether a dialog appears for certain actions you take during your UML Modeling session:

  • Delete File When Deleting Artifact
  • Filter Warning
  • Transform When Elements May Be Lost
  • Overwrite Existing Participants
  • Delete Connector Messages
  • Automatically Create Classifiers
  • Delete Combined Fragment Messages
  • Move Invoked Operation
  • Allow Lengthy Searches

When the property is set to Ask, the corresponding dialog will appear. Otherwise, the specified action for the property will occur automatically.
