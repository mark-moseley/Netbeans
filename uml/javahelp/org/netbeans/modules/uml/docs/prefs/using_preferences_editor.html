Setting UML Modeling Preferences

See Also

Use the IDE Options window to make changes to the default preferences used throughout UML Modeling.

The Options window displays in tree form all preferences that can be added, edited, or deleted. Expand the nodes of this tree to display each node's defaults in the Properties pane. Related Help Text appears at the bottom of the Properties pane.

To change UML Modeling preferences:

  1. Choose Tools > Options.
  2. In the Options window, click the Advanced Options button.
  3. To change general UML preferences, select the UML node and make your changes in the Properties pane.
  4. To change other UML preferences, expand the UML node until you find the preferences you want to change, and then make your changes in the Properties pane.
  5. Click Close.
    Your changes are saved and the Options window closes.
