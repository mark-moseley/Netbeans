Locating Diagram Elements Using the Projects Window

You can easily locate in the Diagram editor any element that is listed in the Projects window.

In the Projects window, double-click the diagram element or right-click the element and choose Open.

  • If the element exists in only one diagram, that diagram opens in the Diagram editor. If the element exists in more than one diagram, the Element Navigation dialog box appears, offering a selection of diagrams.
