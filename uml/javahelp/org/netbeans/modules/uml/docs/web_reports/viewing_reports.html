Viewing Model Reports

See Also

When you use UML Modeling to create a Model Report for a project, the IDE produces a project_dir/report folder, where project_dir is the directory which contains your UML project files.

To view a Model Report:

  1. Navigate to the folder for your UML Modeling project and expand the report folder.
  2. Open index.html.
    A web browser opens and displays the Overview page of the model report. The page also displays the top and bottom navigation panes on the left-hand side.
  3. In the upper left pane, select one of the hyperlinks to All Elements, All Diagrams, or Packages.
    The lower left pane updates with a list corresponding to the selected hyperlink.
  4. In the lower left pane, select one of the hyperlinks.
    The right pane updates with the package, element, or diagram information, depending on what hyperlink was selected.
