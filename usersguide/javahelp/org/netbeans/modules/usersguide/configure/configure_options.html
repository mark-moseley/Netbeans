Using the Options Window

See Also

The Options window enables you to change any of the IDE's configurable settings. The Options window offers two views for editing IDE settings.

  • Modern view. In Modern view, you select a category in the left pane to display its settings in the right pane. Depending on the category, click on the nodes or tabs in the right pane to access additional settings. When you change settings in Modern view, the changes are not applied until you click OK.
  • Classic view. In Classic view, select a node in the tree view in the left pane to display the property sheet in the right pane. The property sheet contains the names and values of the settings for the item selected in the left pane. When you change settings in the property sheet, the changes are applied immediately. Classic view enables you to access some settings not available in Modern view. In Classic view you can apply specific settings globally or at the user level.

When you open the Options window, the default view is Modern view. To switch between views, click the Classic View or Modern View buttons in the Options window.

To edit IDE settings in the Options window:

  1. Open the Options window by choosing Tools > Options.
  2. If you want to edit settings in Classic view, click Classic View in the Options window.
  3. Depending on which view you are in, select the category or node in the left pane for the settings you want to configure.
  4. Modify the settings in the right pane.
  5. Close the Options window.

Editing Settings in Classic view

Editing property sheets
When you are editing settings in Classic view, you can edit the property sheet in the following ways:

  • If the setting is a string, you edit the value inline and press Enter to save the change.
  • If a combo box appears when you select the option's node, click the arrow to select the value.
  • If an ellipsis (...) button appears when you select the option node, click it to open the property's editor.

In Classic view, changes are applied immediately.

Levels of settings
When you are in Classic view, you can store settings at two levels:

  • Default. Initial settings provided with the IDE installation. The IDE's default settings cannot be altered, however, they can be overriden by defining them at the User level.
  • User. Settings defined at the User level that override settings at the Default level.

The following table defines the icons used in Classic view to indicate the levels at which settings are applied.

Icon Description
The settings for the corresponding options are defined at the level indicated in the column's title. Any changes made to those settings apply at that level.
The settings for the corresponding options are inherited from a higher level.
The settings for the corresponding options are ignored at the indicated level.

To define an option's value at the User level:

  1. Select the node for the option you want to configure.
  2. Click the cell in the User column for the options node you selected.
  3. Choose Define Here.

    The  icon appears in the User column.

  4. In the property sheet, configure the desired settings.

    The new values for the selected options override the default values.

To revert to the default settings for an option:

  1. Select the node for the option you want to configure.
  2. Click the cell in the Default column for the options node you selected.
  3. Choose Revert Definition.

    A dialogue appears prompting you to check your work.

  4. If you are sure you want to remove the selected option's User-level configuration, click Yes.

    The default values for the selected options are restored. The  icon appears in the Default column and the  icon appears in the User column.

  • When reverting to default settings, the IDE removes any values stored at the User level for the selected option.
