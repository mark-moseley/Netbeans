Using the Options Window

See Also

You can use the Options window to change any of the IDE's configurable settings. You can apply specific settings globally or at the user level.

The left pane of the window lists all of the IDE's configurable items in tree view and a table view displaying the levels at which settings are stored. The right pane displays a property sheet, which contains the names and values of the settings for the item selected in the left pane.

Editing Settings in the Options Window

To edit IDE options:

  1. Open the Options window by choosing Tools  Options.
  2. Navigate to and select the node for the option you want to configure.
  3. Enter the setting's value in the property sheet.
    • If the setting is a string, you edit the value inline and press Enter to save the change.
    • If a combo box appears when you select the option's node, click the arrow to select the value.
    • If an ellipsis (...) button appears when you select the option node, click it to open the property's editor.

    The specified value is applied at the User level for the selected option.

Levels of Settings

The IDE can store settings at two levels:

  • Default. Initial settings provided with the IDE installation. The IDE's default settings cannot be altered, however, they can be overriden by defining them at the User level.
  • User. Settings defined at the User level that override settings at the Default level.

The following table defines the icons used to indicate the levels at which settings are applied.

Icon Description
The settings for the corresponding options are defined at the level indicated in the column's title. Any changes made to those settings apply at that level.
The settings for the corresponding options are inherited from a higher level.
The settings for the corresponding options are ignored at the indicated level.

To define an option's value at the User level:

  1. Select the node for the option you want to configure.
  2. Click the cell in the User column for the options node you selected.
  3. Choose Define Here.

    The  icon appears in the User column.

  4. In the property sheet, configure the desired settings.

    The default values for the selected options are overidden with the new values.

To revert to the default settings for an option:

  1. Select the node for the option you want to configure.
  2. Click the cell in the Default column for the options node you selected.
  3. Choose Revert Definition.

    A dialogue appears prompting you to check your work.

  4. If you are sure you want to remove the selected option's User-level configuration, click Yes.

    The default values for the selected options are restored. The  icon appears in the Default column and the  icon appears in the User column.

  • When reverting to default settings, the IDE removes any values stored at the User level for the selected option.
