Importing Settings from a Previous Version of the IDE

See Also

When you launch a new version of the IDE for the first time, you are prompted with the option of importing settings from a previous version of the IDE. This prompt occurs only if you have a user directory on your system from a previous version of the IDE and the user directory is in the default location.

If you do not import the settings at the first launch, you can manually import the settings.

To copy settings from a previous version of the IDE:

  1. Within your system's file manager, navigate to the user directory of the old IDE installation.

    To find your user directory choose Help > About and click the Detail tab in the About dialog box. User Dir is listed last in the dialog box.

  2. From the old IDE's user directory, navigate to config.
  3. Find the settings file you are interested in.

    In particular, you probably would want to copy the settings in the Editor folder.

  4. Copy the desired settings file to the corresponding location in the user directory of the new version of the IDE.

Notes:

  • When copying properties between XML files, pay attention to the XML syntax for the properties, especially when copying properties from previous versions of the IDE.
  • There might not be an XML file for certain settings if you are using the default settings. If you delete an XML settings file, the default settings are restored.
