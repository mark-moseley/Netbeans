Importing Settings from a Previous Version of the IDE

See Also

When you launch a new version of the IDE for the first time, you are prompted with the option of importing settings from a previous version of the IDE. This prompt occurs only if you have a user directory on your system from a previous version of the IDE and the user directory is in the default location.

The settings that are imported include the following items:

  • Keyboard shortcuts
  • Most Source Editor font and color settings
  • Java Platform Manager contents
  • Library Manager contents
  • Database drivers
  • Servers

Importing Editor Settings After the First Launch

If you do not import the settings at the first launch, you can manually import selected settings later. In particular, you might want to import Source Editor settings if you have carefully customized your previous installation.

Source Editor settings that you have customized are stored in XML files in your user directory.

If you created named editor profiles in the Options window of the IDE, the default settings for that profile are located within the Editors folder. For example, if you created a profile called MyNetbeansProfile, the default settings for that profile are located in config/Editors/MyNetbeansProfile, the Java editor specific settings for that profile are located in config/Editors/text/x-java/MyNetbeansProfile, the HTML editor specific settings are located in config/Editors/text/html/MyNetbeansProfile and so on.

To copy settings from a previous version of the IDE:

  1. Within your system's file manager, navigate to the user directory of the old IDE installation.

    To find your user directory choose Help > About and click the Detail tab in the About dialog box. User Dir is listed last in the dialog box.

  2. From the old IDE's user directory, navigate to config/Editors.
  3. Copy the folders within the config/Editors folder.
  4. Navigate to the user directory in your new installation and open the config/Editors folder.
  5. In the config/Editors folder, create a new folder and give it a name such as NetBeans41.

    The name of the folder becomes the name of a profile in the IDE that you can switch to in order to adopt a group of settings.

  6. Paste the copied folders inside of the new folder.

    The pasted files become part of the new profile.

  7. Restart the new version of the IDE.
  8. Choose Tools > Options and click Fonts & Colors.
  9. In the Profile drop-down list, select the name .
