Options Window: Miscellaneous: Versioning

See Also

The Versioning tab in the Options > Miscellaneous window enables you to configure VCS-specific parameters that affect the behavior of Subversion, CVS, and local history in the IDE.

You access this panel by choosing Tools > Options from the main menu, then selecting Miscellaneous and the Versioning tab within the Options window.

To make changes to settings for Subversion, CVS, or local history, select the appropriate versioning tool from the Versioning Systems pane on the left side of the window. Available settings immediately display within the window.

View the following documents for help related to specific versioning tools:

  • CVS
  • Local History
  • Subversion
