The IDE allows you to configure options for it's many areas of functionality as well as customize its workspace.

The IDE's Configuration Tools

The primary places where you can configure the IDE:

  • Setup Wizard, allows you to set general IDE settings, enable/disable installed modules and edit Update Center settings.
  • Options Window, allows you to edit any of the IDE's configurable settings. For more information, see Using the Options Window.
  • Keyboard Shortcuts Window, allows you to customize the IDE's configurable keyboard equivalents. For more information, see Setting Menu Shortcuts and Setting Keyboard Shortcuts for the Source Editor.
  • Update Center, allows you to add new or updated modules to the IDE. For more information, see Updating the IDE.
You can also pass startup parameters to the IDE launcher in the IDE-HOME/etc/netbeans.conf file by using startup switches. For more information, see IDE Startup Parameters.

Configuring Other IDE Modules

If you have additional modules installed that are not part of the standard IDE distribution, you can configure them using special tools. Common areas requiring configuration include:
