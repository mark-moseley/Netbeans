Customizing the Display of Commands in a Toolbar

See Also

You can add, remove, and change the order of buttons in a toolbar to make those you frequently use more accessible. You can also group related commands with a separator.

You can customize toolbars using the Toolbar customizer or in the Options window.

To add a toolbar button using the Toolbar Customizer

To add a toolbar button to a toolbar:

  1. Choose Tools > Options, and then click Classic view in the Options window and expand IDE Configuration > Look and Feel > Actions.
  2. Within the Actions node, expand the action categories, and navigate to the command you want to add to the toolbar. Then right-click the command and choose Copy.
  3. Expand IDE Configuration > Look and Feel > Toolbars. Then right-click the node for the toolbar where you want to place the button and choose Paste.

    The IDE adds the button for that command to the toolbar.

To remove a toolbar button from a toolbar:

  1. Choose Tools > Options and expand IDE Configuration > Look and Feel > Toolbars.
  2. Within the Toolbars node, navigate to the node for the command that you want to remove from the toolbar. Then right-click the command and choose Delete.

To reorder a button in a toolbar:

  1. Choose Tools > Options and expand IDE Configuration > Look and Feel > Toolbars.
  2. Right-click the node for the button that you want to reorder and choose Move Up or Move Down to move the button to the desired position.

To group related commands with a separator:

  1. Choose Tools > Options and expand IDE Configuration > Look and Feel > Toolbars.
  2. Within the Toolbars node, right-click the node for the toolbar where you want to add the separator and choose Add Separator. The separator is added to the bottom of the toolbar list.
  3. Right-click the node for the separator and choose Move Up to place the separator in the desired position.
  • To find the name of the toolbar you want to change, rest the pointer on the textured drag area of the toolbar to display a tool tip with the toolbar name.
