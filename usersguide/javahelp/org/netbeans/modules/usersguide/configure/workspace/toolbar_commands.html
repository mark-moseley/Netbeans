Customizing the Display of Commands in a Toolbar

See Also

You can add, remove, and change the order of buttons in a toolbar to make those you frequently use more accessible. You can also group related commands with a separator.

You can customize toolbars using the Toolbar customizer or in the Options window.

  • To find the name of the toolbar you want to change, rest the pointer on the textured drag area of the toolbar to display a tool tip with the toolbar name.

To add a toolbar button to a toolbar:

  1. Choose View > Toolbars > Customize to open the Customize Toolbars dialog box.
  2. Select the button of the action you want to add to the IDE toolbar and drag and drop the button onto the toolbar in the main window.

    The IDE adds the button for that command to the toolbar.

To remove a toolbar button from a toolbar:

  1. Choose View > Toolbars > Customize to open the Customize Toolbars dialog box.
  2. In the main window, click and drag the toolbar button above the toolbar and release your mouse button when you see a small trash can icon appear.

    The Customize Toolbars dialog box must be open when you are removing toolbar buttons.

To reorder a button in a toolbar:

  1. Choose View > Toolbars > Customize to open the Customize Toolbars dialog box.
  2. In the main window, drag and drop the button to the desired position.

To change the size of toolbar buttons:

  1. Choose View > Toolbars > Small Toolbar Icons.

    Alternately, right-click the empty space in the toolbar area and choose Small Toolbar Icons from the pop-up menu.
