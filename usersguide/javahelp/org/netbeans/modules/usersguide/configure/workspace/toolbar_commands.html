Customizing the Display of Commands in a Toolbar

See Also

You can add, remove, and change the order of buttons in a toolbar to make those you frequently use more accessible. You can also group related commands with a separator.

To add a toolbar button to a toolbar:

  1. Right-click the empty space in the toolbar area and choose Customize Toolbars from the contextual menu to display the Toolbars dialog box.
  2. Expand the nodes of the different toolbars until you find the command you want to add to the toolbar.
  3. Right-click the command and choose Copy.
  4. Right-click the node of the toolbar where you want to place the command, and choose Paste  Copy.

    The IDE adds the button for that command to the toolbar.

To remove a toolbar button from a toolbar:

  1. Right-click the empty space in the toolbar area and choose Customize Toolbars from the contextual menu to display the Toolbars dialog box.

  2. Expand the node of the toolbar you want to change.
  3. Right-click the command you want to remove and choose Delete.
  4. Confirm that you want to delete the command from the toolbar.

    5. The IDE removes the button for that command from the toolbar.

To reorder the commands within a toolbar:

  1. Right-click the empty space in the toolbar area and choose Customize Toolbars from the contextual menu to display the Toolbars dialog box.

  2. Expand the node of the toolbar you want to change.
  3. Right-click the command you want to move and choose Move Up or Move Down.

    4. The IDE updates the toolbar with the changes you have made.

To group related commands with a separator:

  1. Right-click the empty space in the toolbar area and choose Customize Toolbars from the contextual menu to display the Toolbars dialog box.

  2. Right-click the node of the toolbar to which you want to add the separator and choose Add Separator. The separator is added to the bottom of the toolbar list.

  3. Right-click the separator and choose Move Up and Move Down to move the separator to the desired position in the toolbar.

    The IDE updates the toolbar with the changes you have made.

  • To find the name of the toolbar you want to change, rest the pointer on the textured drag area of the toolbar to display a tool tip with the toolbar name.
