Customizing Toolbars in the IDE

See Also

The IDE provides comprehensive control over how toolbars are presented. You can add, position and control the visibility of toolbars. You can also create customized toolbar configurations.

Choose View > Toolbars from the main menu to see a list of the available toolbars. The submenu displays all the toolbars in the IDE.

To add a toolbar to the main window:

  1. From the main window, choose View > Toolbars > Customize.
  2. In the Customize Toolbars dialog box, click New Toolbar.
  3. In the New Toolbar dialog box, type a name for the toolbar and click OK.

    The IDE adds an empty toolbar to the main window.

To show or hide a toolbar:

Toolbars that are currently visible are indicated by a check mark in the Toolbars pop-up menu.

  1. Choose View > Toolbars from the main menu and choose a toolbar name in the submenu.

    Alternately, right-click the empty space in the toolbar area and choose the toolbar name from the pop-up menu.

To move a toolbar:

  • Click in the textured drag area on the left side of the toolbar and drag the toolbar to the desired location.
