Customizing Toolbars in the IDE

See Also

The IDE provides comprehensive control over how toolbars are presented. You can add, remove, position and control the visibility of toolbars. You can also create customized toolbar configurations.

To add a toolbar to the main window:

  1. Right-click the empty space in the toolbar area and choose Customize Toolbars from the contextual menu to display the Toolbars dialog box.
  2. Right-click the Toolbars node and choose Add  Toolbar to display the New Toolbar dialog box.

  3. Type a name for the toolbar in the Name text field.
  4. Click OK.

    The IDE adds an empty toolbar to the main window.

To remove a toolbar from the main window:

  1. Right-click the empty space in the toolbar area and choose Customize Toolbars from the contextual menu to display the Toolbars dialog box.
  2. Right-click the toolbar you want to remove and choose Delete.
  3. Click Yes in the confirmation dialog box.

    The IDE removes the toolbar from the main window.

To position a toolbar in a new location:

  • Click in the textured drag area on the left side of the toolbar and drag the toolbar to the desired location in the main window.

To show or hide a toolbar:

  1. Right-click the empty space in the toolbar area.
  2. Select a toolbar name from the contextual menu.

To save the current toolbar configuration:

  1. Right-click an empty space in the toolbar area and choose Save Configuration from the contextual menu. The Save Toolbar Configuration dialog box is displayed.
  2. Type a name for the configuration.
  3. Click OK.

    4. The name is added to the toolbar contextual menu.
