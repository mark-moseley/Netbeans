Customizing Toolbars in the IDE

See Also

The IDE provides comprehensive control over how toolbars are presented. You can add, remove, position and control the visibility of toolbars. You can also create customized toolbar configurations.

To add a toolbar to the main window:

  1. Choose Tools > Options and expand IDE Configuration > Look and Feel.
  2. Right-click the Toolbars node and choose Add Toolbar.
  3. In the Name field, type a name for the toolbar and click OK.

    The IDE adds an empty toolbar to the main window.

To remove a toolbar from the main window:

  1. Choose Tools > Options and expand IDE Configuration > Look and Feel  > Toolbars.
  2. Right-click the node for the toolbar that you want to remove and choose Delete.

To move a toolbar:

  • Click in the textured drag area on the left side of the toolbar and drag the toolbar to the desired location.

To show or hide a toolbar:

  • Right-click the empty space in the toolbar area and choose the toolbar name from the contextual menu.
