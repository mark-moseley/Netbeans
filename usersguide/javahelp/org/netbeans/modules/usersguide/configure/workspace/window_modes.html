Changing Window Modes

See Also

The IDE provides two window modes that you can use to manage your windows:

  • Compact Windows Mode. Also called multiple document interface (MDI), this window mode confines all of the IDE's internal window panes within a single primary window. This is the default and recommended window mode.
  • Separate Windows Mode. This window mode displays a single primary window with a set of supplemental secondary windows. This window mode is also called single document interface (SDI).

To change the IDE's display mode:

  1. From the main window, choose Tools > Options and click Classic View.
  2. In the left pane, expand IDE Configuration > System and select the System Settings node.
  3. For the UI Mode property in the right pane, select the desired mode from the drop-down list.
  4. Click Close.

The IDE displays its various windows in the selected window mode.
