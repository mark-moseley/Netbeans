Configuring System Settings

  1. From the main window, choose Tools Options.
  2. In the Options window, select the System Settings node.
  3. Modify the properties to your preferred settings.
    Confirm Delete If True, opens a Confirm Deletion dialog box when you attempt to delete an item.
    Home Page Sets the web page that the web browser displays on startup.
    Proxy Host Sets the host name of the proxy server.
    Proxy Port Sets the port number of the proxy server.
    Show File Extensions If True, displays file extensions in the Explorer and Object Browser.
    Show Tips on Startup If True, opens a Welcome window when you start the IDE. This window helps you start working with objects and provides over thirty tips on how to use the IDE.
    Use Proxy If True, uses a proxy for HTTP/FTP connections in the web browser and Update Center.
    User Interface Sets the IDE as a single document interface (SDI) or a multiple document interface (MDI). An SDI uses a single primary window with a set of supplemental secondary windows. An MDI confines all of an application's internal frames inside a primary window.
    Web Browser Sets the web browser to use when you are viewing Javadoc documentation and browsing the web. If you select <unspecified>, the IDE uses the first accessible browser.
