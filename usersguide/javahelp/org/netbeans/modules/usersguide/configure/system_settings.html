Options Window: System Settings

See Also

The System Settings node in the Options window contains general IDE settings.

You can access the System Settings node by choosing Tools > Options, clicking Advanced Options in the Options window and then expanding IDE Configuration > System and selecting the System Settings node.

The System Settings node contains the following options:

  • Ignored Files. Specifies which files the IDE ignores. Enter a regular expression for the files you want to ignore. These files are not displayed in any of the IDE windows and are not processed by IDE commands.
  • Show Tool Tips in IDE. If selected, shows a short text string that describes the interface element beneath the pointer.
