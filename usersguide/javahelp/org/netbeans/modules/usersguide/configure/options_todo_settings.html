Options Window: Miscellaneous: ToDo Tasks

See Also

You use this panel to specifying the ToDo patterns used for identifying To Do tasks.

You can access this panel by choosing Tools > Options from the main menu, clicking the Miscellaneous category and then clicking the ToDo Tasks tab.

You can set the following To Do options:

  • ToDo Patterns. The text files displays the tag strings used in the source code to identify tasks. Use the buttons to the right of the text field to add, delete or modify task tags.
  • Show ToDos from comments only. If selected, ToDo patterns that are outside of comment blocks are ignored.
