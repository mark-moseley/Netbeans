Options: To Do Settings

See Also

Use this node to configure general settings for specifying and displaying To Do tasks.

You can access this node by choosing Tools > Options > Advanced radio button, and then choosing Editing > To Do Settings.

Properties

  • Skip Tasks Outside Comments. If selected, matched tokens that are outside of comment blocks are ignored.
  • Task Tags. Specifies the tag strings used in the source code to identify tasks. Click the ellipsis button (...) to open the Task Tags dialog box to add, delete or modify task tags.

Expert

  • List Limit. Specifies the number of tasks that should be listed before the IDE stops searching for additional tasks.
