Plugins Manager: Available Plugins

See Also

You use the Plugins manager to dynamically update the IDE from the registered update centers and to manage IDE plugins. You use the Available Plugins tab of the Plugins manager to view and install new plugins available from the registered update centers.

You can access this tab by choosing Tools > Plugins from the main menu and clicking the Available Plugins tab in the Plugins manager.

To check the update center and refresh the list of available plugins, click Reload Catalog.
