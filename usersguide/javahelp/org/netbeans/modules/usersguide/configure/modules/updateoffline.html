Installing Modules Offline

See Also

If you have already downloaded a module's .nbm file, you can manually install it without having to connect to an Update Center server.

To install a module offline:

  1. Choose Tools  Update Center from the main menu.
  2. Select the Install Manually Downloaded Modules button and click Next.
  3. On the Select Modules to Install page, click Add.
  4. In the file chooser, navigate to the module's .nbm file. Select the file and click OK.
  5. Repeat the previous step for every module that you wish to add. Then click Next. The modules appear in the Include in Install panel.
  6. Complete the remaining pages of the wizard to install the module.
If the module that you have selected in the file chooser does not appear in the Available Updates and New Modules panel, you probably have the same or a newer version of that module installed already. You can use the Setup Wizard to verify that the module enabled.
