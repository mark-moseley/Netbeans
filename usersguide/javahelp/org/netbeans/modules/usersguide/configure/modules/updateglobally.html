Updating a Multi-User IDE Installation

See Also

If you are using a multi-user installation of the IDE, you can install a module globally so that the module is available to all users. When you install or update a module using the Update Center wizard, the IDE places the module .jar file and docs in your user directory. The IDE places .jar files and docs for globally installed modules in your installation directory instead of an individual user directory.

To globally install a module:

  1. Download the module using the Update Center wizard.
  2. In the View Certificates and Install Modules page of the wizard, select the Global checkbox for the desired modules.
  3. Click Finish to install the module.
  4. Restart the IDE to activate the new module.
  • In order to install a module globally, you must have write access for the IDE installation directory.
