Plugins Manager: Installed

See Also

You use the Plugins manager to dynamically update the IDE from the registered update centers and to manage IDE plugins. You use the Installed tab of the Plugins manager to manage installed IDE plugins.

You can access this tab by choosing Tools > Plugins from the main menu and clicking the Installed tab in the Plugins manager.

The left pane displays a list of the installed plugins. The active column displays the state of the plugin. The following table displays the icons used to indicate the state of the plugin.

Icon Description
The plugin is installed and activated.
The IDE needs to be restarted to fully deactivate the plugin.
The plugin is installed but deactivated.

When you select a plugin in the left pane, the description of the plugin is displayed in the right pane. You can activate and deactivate installed plugins in the right pane. A deactivated plugin is not loaded on IDE startup. If a plugin is deactivated, it still exists on your local system and you can re-activate the plugin without downloading it again.

  • An installed plugin can be active or inactive. If a listed plugin is inactive, you may need to install additional plugins to use the plugin.

If you want to completely remove a plugin from your local system, select the checkbox for the plugin in the left pane and then click Uninstall.
