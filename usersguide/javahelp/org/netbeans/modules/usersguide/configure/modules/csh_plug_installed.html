Plugins Manager: Installed

See Also

You use the Plugins manager to dynamically update the IDE from the registered update centers and to manage IDE plugins and modules. You use the Installed tab of the Plugins manager to manage installed IDE plugins.

You can access this tab by choosing Tools > Plugins from the main menu and clicking the Installed tab in the Plugins manager.

The left pane displays a list of the installed plugins. The active column displays the state of the plugin. The following table displays the icons used to indicate the state of the plugin.

When you select a plugin in the left pane, the description of the plugin is displayed in the right pane. You can activate and deactivate installed plugins in the right pane.

  • An installed plugin can be active or inactive. If a listed plugin is inactive, you may need to install additional plugins to use the plugin.

The following table defines the icons used in the Installed tab of the Plugins manager to indicate the status of plugins.

Icon Description
The plugin is installed and activated.
The plugin is installed but is not yet activated.
The plugin is installed but deactivated.
