Enabling or Disabling a Module

See Also

The IDE allows you to disable the modules you do not need to save startup time and memory. Disabled modules are not deleted from your installation directory, but are simply ignored by the IDE. You can re-enable disabled modules at any time.

To enable or disable a module:

  1. From the Tools menu, choose Setup Wizard.
  2. Click Next to advance to the Module Installation page.
  3. On the Module Installation page, select the module you wish to enable or disable and click the check box in the Enabled column.
  4. Click Finish.
