Options Window: Keymap

See Also

You use this panel in the Options window to set the keyboard shortcuts for the IDE. You can change between pre-configured shortcut sets, create a new shortcut set, or change the keyboard shortcut for an individual command.

Profile settings

You can create profiles to save different custom keymap settings. You can switch between profiles to quickly change between saved settings. If you modify either of the profiles included in the IDE, you can click Restore to restore the selected profile to the default settings.

To create a profile:

  1. Click Keymap in the Options window.
  2. Click Duplicate next to the Profile drop-down list to open the Create New Profile dialog box.
  3. Type a name in the Create New Profile dialog box.
  4. Modify the settings in the Options window.
  5. Click OK to close the Options window.

You use this window as follows:

  • Expand a node in the Actions list and select a command to view the keyboard shortcut for that command.
  • Create a new shortcut by clicking Add and entering the new shortcut in the Add Shortcut dialog box. Click OK.
  • Remove a shortcut by selecting the shortcut from the Shortcuts list and clicking Remove.

    As you press the key sequence, the correct syntax for that sequence automatically appears in the text field. If you simultaneously hold down the Alt key, the Control key, and the J key, "Alt+Ctrl+J" appears. To use a Backspace or Tab key in the shortcut, use the buttons at the bottom of the dialog box. You can only specify a keyboard shortcut that is not being used by another command.

Notes:

  • To set keyboard shortcuts for recorded macros, click the Editor category in the Options window and then click the Macros tab.
