Options Window: Files

See Also

The IDE categorizes all files by type. The Files tab displays the file types recognized by the IDE, the mime resolver for that type and the location of the file types listed as belonging to that type.

You can view all your existing file types by choosing Tools > Options from the main menu and clicking Miscellaneous in the Options window. Click Files and then set your file associations.

File Associations

  • File Extension. Specifies name of mime resolver for files of this type.
  • Associated File Type (MIME). Specifies the related MIME type for the selected file type.

Files Ignored by the IDE

  • Ignored Files Pattern. Type POSIX-style regular expressions to specify the files you want the IDE to ignore. The IDE includes some default patterns for files that are ignored by default. Click the Default button to restore the default patterns.
