Options Window: File Types

See Also

The IDE categorizes all files by type. The File Types node displays the file types recognized by the IDE, the mime resolver for that type and the location of the file types listed as belonging to that type.

You can view all your existing file types by choosing Tools > Options from the main menu and clicking Advanced Options in the Options window. In the left pane, expand IDE Configuration > System > File Types > file-type node.

Properties

  • Name. Specifies name of mime resolver for files of this type.
  • All Files. (Read only) Location of list of files represented by this object.
  • File Size. (Read only) Length in bytes of first file listed in All Files location.
  • Modification Time. (Read only) Modifaction date of first file listed in All Files location.
