Installing Sample Applications

See Also

You can make use of sample projects included with the NetBeans IDE to learn about programming and supported APIs.

To install a project included with the IDE:

  1. Choose File > New Project.
  2. Under Categories, expand the Samples folder, then select the folder of the technology you want to work with. When you select a folder, related sample projects display under Projects in the right pane.
  3. Select a sample project from the Projects pane and click Next.
  4. Follow the steps in the New Project wizard to open the project.
  • If the sample project you selected requires a web or application server, you have to register the server before opening the project. For more information, see the following documents:
    • Registering a Sun Java System Application Server Instance
    • Registering an External Tomcat Web Server
  • You can get more samples in the Plugins Manager (Tools > Plugins).
