Options Window: Global XML Settings

See Also 

In the Options window, you can configure the way the IDE handles and displays XML files.

To access global XML settings:

  1. From the main menu, choose Tools > Options and click Miscellaneous.
  2. Click Files and select "xml" from the File Extension drop-down.
  3. Select the file type to be associated with XML files.
