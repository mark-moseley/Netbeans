Registering an XML Catalog

See Also 

The Runtime window contains all of your currently registered under the DTD and XML Schema Catalogs node ().

To register an XML catalog:

  1. Expand the DTD and XML Schema Catalogs node, right-click and the User Catalog node, and choose Add Local DTD or XML Schema.
  2. Select whether which ID to map to the local file:
    • Public ID. Generally you use this option to redirect references to a DTD.
    • System ID. Generally you use this option to redirect references to an XML schema.
  3. Browse to the location of the local DTD or XML schema in the URI field.

    The appropriate XML catalog entry node () appears under the DTD and XML Schema Catalogs node. You can double-click the node to open the DTD or XML schema in the Source Editor.

To remove an XML catalog:

  1. Expand DTD and XML Schema Catalogs > User Catalog in the Runtime window.
  2. Right-click the XML catalog node you want to remove and choose Delete.
