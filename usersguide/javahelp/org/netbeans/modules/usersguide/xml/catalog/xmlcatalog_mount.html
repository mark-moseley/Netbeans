Mounting and Unmounting an XML Catalog

See Also 

The Runtime window contains all of your currently mounted XML catalogs under the XML Entity Catalogs node ().

To mount an XML catalog:

  1. In the Runtime window, right-click the XML Entity Catalogs node and choose Add Catalog.
  2. Select the type of catalog you want to add in the Catalog Type combo box. You can choose from the following types:
    • OASIS Catalog Resolver. Select this option to add an OASIS XML catalog.

      Select the Prefer Public ID checkbox if you want to use the public identifier specified in the catalog rather than the system identifier located in the XML document. In most cases, catalogs map the public identifier to a local copy of the entity they represent. In this case, you need to select the Prefer Public ID checkbox for entity resolution to function properly.

    • XML Catalog. Select this option to add X Catalog v0.2 or XML Catalog v0.4 deprecated catalog formats.
    • NetBeans Catalog. Select this option to add the Netbeans IDE catalog in read-only form.
  3. Type the location of the catalog in the Catalog URL field, or click Browse to specify the file's location. Then click OK.

    The appropriate XML catalog node () appears under the XML Entity Catalogs node.

To remove an XML catalog:

  1. Expand the XML Entity Catalogs node in the Runtime window.
  2. Right-click the XML catalog node you want to remove and choose Remove.

    The designated XML catalog node () is removed from the XML Entity Catalogs node.
