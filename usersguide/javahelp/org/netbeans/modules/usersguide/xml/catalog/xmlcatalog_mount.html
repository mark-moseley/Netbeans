Registering a Local DTD or XML Schema Resource

See Also 

The Runtime window contains all of your currently registered DTD or XML schema resources under the User Catalog node ().

To register a local DTD or XML schema:

  1. Expand the DTD and XML Schema Catalogs node, right-click the User Catalog node, and choose Add Local DTD or XML Schema.
  2. Select which ID to map to the local file:
    • Public ID. Generally you use this option to redirect references to a DTD.
    • System ID. Generally you use this option to redirect references to an XML schema.
  3. Browse to the location of the local DTD or XML schema in the URI field.

    The appropriate User Catalog entry node () appears under the User Catalog node. This entry represents the public ID or system ID mapping to the local DTD or XML schema resource.

To remove a DTD or XML schema entry from the User Catalog:

  1. Expand DTD and XML Schema Catalogs > User Catalog in the Runtime window.
  2. Right-click the XML catalog node you want to remove and choose Delete.
