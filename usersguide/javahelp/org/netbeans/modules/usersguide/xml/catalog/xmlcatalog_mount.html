Mounting and Unmounting an XML Catalog

See Also 

The Runtime window contains all of your currently mounted XML catalogs under the XML Entity Catalogs node ().

To mount an XML catalog:

  1. In the Runtime window, right-click XML Entity Catalogs and choose Mount Catalog.
  2. Select the type of catalog from the installed format recognizers:
    • NetBeans Catalog. Select this option to mount the Netbeans IDE catalog in read-only form.
    • OASIS Catalog Resolver. Select this option to mount an OASIS XML catalog.

      The OASIS Catalog Resolver contains a Prefer public checkbox. Select this checkbox if you want to use the public identifier specified in the catalog rather than the system identifier located in the XML document. In most cases, your catalog maps the public identifier to a local copy of the entity it represents. In this case, you need to select the Prefer public checkbox for entity resolution to function properly.

    • XML Catalog. Select this option to mount XML Catalog v0.4 catalogs.
  3. Type the location of the file, or click File to browse to the file's location. Then click OK.

    The appropriate XML catalog node () appears under the XML Entity Catalogs node.

To unmount an XML catalog:

  1. Expand the XML Entity Catalogs node in the Runtime window.
  2. Right-click the XML catalog node and choose Unmount Catalog.

    The designated XML catalog node () is removed from the XML Entity Catalogs node.
