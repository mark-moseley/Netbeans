Creating and Editing an XML Catalog

See Also 

The IDE provides a template for creating new OASIS XML Catalog files. You can edit catalog files as you would any other XML document.

To create a new OASIS XML Catalog:

  1. From the main window, choose File  New File.
  2. Select the XML node, choose OASIS XML Catalog, and click Next to proceed.
  3. Specify a file name for the new catalog.
  4. Enter the location for the file in the Folder field or click Browse to navigate to the desired directory. Then click Finish.

    The IDE creates the new file in the specified location on your system.

To edit an XML catalog:

  1. Mount the catalog file in the Files window if it is not already mounted.
  2. Double-click the catalog file node.

    The file is displayed in the Source Editor.

To view a mounted catalog entry in read-only form:

  1. In the Runtime window, expand the XML Entity Catalogs node.
  2. Expand the node for the desired catalog.
  3. Right-click the catalog entry node () and choose View.

    If the mapping is valid, the Source Editor displays the catalog entry in read-only mode.
