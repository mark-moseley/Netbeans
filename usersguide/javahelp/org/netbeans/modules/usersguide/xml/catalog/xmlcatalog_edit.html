Creating and Editing an XML Catalog

See Also 

The IDE provides a template for creating new OASIS XML Catalog files.

To create a new OASIS XML Catalog:

  1. From the main window, choose File  New File.
  2. Expand the XML node and select OASIS XML Catalog. Click Next to proceed.
  3. Specify a name and location for the file and click Finish. You can edit catalog files as you would any other XML document.

To edit an XML catalog:

  1. Mount the catalog file in the Files window if it is not already mounted.
  2. Double-click the catalog file node.

    The file is displayed in the Source Editor.

To view a mounted catalog entry in read-only form:

  1. Click the Runtime window.
  2. Expand the XML Entity Catalogs node.
  3. Expand the node for the desired catalog.
  4. Right-click the catalog entry node () and choose View.

    If the mapping is valid, the Source Editor displays the catalog entry in read-only mode.
