Creating and Editing an XML Catalog

See Also 

The IDE provides a template for creating new OASIS files. You can edit catalog files as you would any other XML document.

To create a new OASIS XML Catalog:

  1. From the main menu, choose File > New File.
  2. Select the XML node, choose OASIS XML Catalog, and click Next to proceed.
  3. Specify a file name for the new catalog.
  4. Enter the location for the file in the Folder field or click Browse to navigate to the desired directory. Then click Finish.

    The IDE creates the new file in the specified location on your system.

To edit an XML catalog:

  • Double-click the catalog node to open it in the Source Editor.

    The file is displayed in the Source Editor. Here is a simple of an OASIS XML Catalog 1.0 catalog. You can view the spec for this format at:

To view a mounted catalog entry:

  1. Choose Tools > DTDs and XML Schemas from the main menu.
  2. Expand the node for the desired catalog. Catalogs are marked as being either read-only or read-write.
  3. Select the catalog entry node and click Open in Editor.

    If the mapping is valid, the Source Editor displays the catalog entry in read-only mode.
