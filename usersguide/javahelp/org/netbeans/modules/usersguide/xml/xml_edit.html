Editing an XML Document

See Also 

The IDE's Source Editor provides such features as syntax coloring, indentation, and basic code completion to assist you when editing XML documents. If your XML file has an associated DTD, you can use code completion to automatically enter the names of any supported element, attribute, or entity. As you type, a combo box displays all matching elements in the DTD.

To open an XML file in the Source Editor:

  1. Right-click the XML file's node in the Files window.
  2. Choose Edit from the contextual menu.

    The IDE opens the file in a new Source Editor tab and shifts focus to it. Alternately, you can also simply double-click the file's node in the Files window.

  • The Source Editor toolbar contains shortcuts to the most commonly used XML commands. To disable the Source Editor toolbar, choose Tools  > Options and expand the Editing node. Then select the Editor Settings node and set the Show Toolbar property to False.
