Project Group: Create New Group Dialog Box

See Also

The Create New Group dialog box enables you to create groupings of projects so that you can open and close several projects at once. In addition, each group can have a different main project.

You can open the Create New Group dialog box by choosing File > Project Group > New Group.

You can create the following types of groups:

  • Free Group. When you create a free group, you can choose either of the following options:
    • Use Currently Open Projects. Adds the currently open projects to the group.
    • Automatically Save Project List. Changes the contents of the project group according to the projects that you open and close when the project group is open.
  • Project and All Required Projects. Creates a group that consists of a master project and all of the projects that it depends on.
  • Folder of Projects. Creates a group out of all of the projects that are within a folder you specify.

Once you have created a project group, that project group is available in the Project Group menu. When you select a project group from that menu, all other open projects are closed. If you want to close all open projects, choose File > Project Group > (none).

  • If you want to create a group that contains a specific set of projects, make sure that those projects are the only open projects. Then choose File > Project Group > Create New Group. In the Create New Group dialog box, select the Free Group radio button and select the Use Currently Open Projects checkbox.
  • If you do not want the contents of a free group to change as you open and close projects, clear the Automatically Save Project List.
  • If you want to modify the contents of a free group that you set up without the Automatically Save Project List option selected, open any projects that you want to add and close any projects that you want to remove from the group. Then choose File > Project Group > Properties of GroupName and click Save Current Project List Now.
