Files Window

See Also

The Files window shows a directory-based view of your projects, including any files and folders that are not displayed in the Projects Window. You can open the Files window by choosing Window > Files (Ctrl-2).

To view documentation specific to the type of project you are working in, see the following pages:

  • Java Projects
  • C/C++ Projects

Context-Sensitive Menu

Each node in the Files window has a context-sensitive pop-up menu of commands that you can apply to that node. You open a node's context-sensitive menu by right-clicking the node.
