Recovering Deleted Files

See Also

The IDE's local history repository maintains previous versions of your project sources. This includes files or folders which you have recently deleted. Sometimes when developing you may find it useful to be able to reexamine and possibly reinstate previously deleted files into your project. You can take advantage of the IDE's local history tools to recover previously deleted files.

If you have previously deleted files and they are still maintained in the IDE's local history repository, you can invoke the Revert Deleted command to retrieve them:

  • From the Projects or Files window, select the project or directory that previously contained the files which you deleted and choose either:
    • Local History > Revert Deleted from the right-click menu.
    • Versioning > Local History > Revert Deleted from the IDE's main menu.
    The previously deleted files reappear beneath the project or folder you selected.
  • In order to recover deleted files using the Revert Deleted command, files must not be created or deleted externally.
