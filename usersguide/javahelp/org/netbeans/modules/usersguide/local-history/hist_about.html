About Local File History

See Also

The IDE's built-in local file history tools provide basic versioning support for files in your local working directory. Similar to conventional versioning systems, the IDE maintains an internal history of recent changes to project resources. Whenever you save a file, this is registered as a 'commit' of a new version of the file in the local history. You can inspect all revisions in the local history and diff them against the current files.

The IDE's Local History Tools

The IDE provides the following versioning tools for local files:

  • Graphical Diff Viewer. Enables you to compare differences in file revisions graphically.
  • Revert Dialog. Provides an interface allowing you to select a revision to revert changes to.
