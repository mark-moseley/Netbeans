The Navigator window is a tool that displays information about the java source files in the currently opened project. You can view source elements either as a simple list or as an inheritance tree. By default, the Navigator window is located in the lower lefthand corner of the IDE's workspace below the Projects, Files, and Runtime windows.

You can open the Navigator window by choosing Window  Navigator (Ctrl-7).

Using the Navigator Window

The IDE's Navigator Window enables you to:

  • Quickly navigate to specific methods and fields in your java source code.
  • Perform basic actions on source elements using the contextual menu.
  • View the structure of a selected java source file.
The Navigator window's functionality replaces the navigation combo box that was previously located in the IDE's Source Editor toolbar.

Navigator Window Views

The Navigator window enables you to display supported documents' source code in two views.

  • Member view. Alphabetical listing of the selected source file's members.
  • Inheritance view. Hierarchical tree view of the selected source file's members.
