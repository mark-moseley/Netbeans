Code Template Abbreviations: Quick Reference

See Also

You can reduce the number of keystrokes when you are typing code by typing abbreviations to generate code snippets. The abbreviation is expanded into a longer code snippet after you type a space.

The IDE comes with a set of code templates. You can also create your own code templates.

Below is the list of code templates that the IDE provides by default.
