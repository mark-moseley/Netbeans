Keyboard Shortcuts

See Also

Keyboard shortcuts are keystroke combinations consisting of a modifier key and a character key such as Control-Z, for example. Keyboard shortcuts enable you to activate a command and shift the input focus (designate the active UI component) by using the keyboard instead of the mouse.

Types of Shortcuts in the IDE

The IDE organizes keyboard shortcuts into the following groups:

  • IDE Menu Shortcuts. Shortcuts to the commands in the main IDE menus. See Menu Shortcuts for a full list of these shortcuts and Setting Menu Shortcuts for instructions on changing these shortcuts.
  • Source Editor Shortcuts. Shortcuts to commands and actions available in the Source Editor. Some Source Editor shortcuts are common to all file types, while others are only available when editing certain types of files. See Editor Shortcuts for a full list of these shortcuts and Setting Keyboard Shortcuts for the Source Editor for instructions on changing these shortcuts.
  • IDE Window Shortcuts. Non-editable shortcuts for navigate, activate, and select operations in windows in the IDE. See Window Navigation Shortcuts for a full list of these shortcuts.
  • Help Viewer Shortcuts. Non-configurable shortcuts for navigating the Online Help System. For a full list of these shortcuts, see Help Viewer Shortcuts.
