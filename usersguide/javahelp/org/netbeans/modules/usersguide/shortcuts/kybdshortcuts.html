Keyboard Shortcuts

Keyboard shortcuts are keystroke combinations consisting of a modifier key and a character key, for example, Control-Z. Keyboard shortcuts enable you to activate a command and move the input focus (designate the active UI component) by using the keyboard instead of the mouse.

The IDE provides the following keyboard shortcuts:

  • Menu Shortcuts
  • Editor Shortcuts
  • Help Viewer Shortcuts

To set or change keyboard shortcuts:

  • Setting Menu Shortcuts
  • Setting Keyboard Shortcuts for the Source Editor
  • Key Bindings Property Editor
