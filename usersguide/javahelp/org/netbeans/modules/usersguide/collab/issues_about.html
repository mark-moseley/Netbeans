About Issue Trackers

See Also

An issue tracker can help you to record the status of your project's issues. If your project is associated with an issue tracking system, you can submit issues and assign responsibility for resolving issues to members of the team collaborating on the project.

Support for the following issue tracking systems is integrated in the IDE.

  • Bugzilla ()
  • JIRA ()

You can perform the following tasks from within the IDE.

  • Find issues
  • Report new issues
  • Update and resolve issues
  • Create and save issue queries
  • JIRA support requires the JIRA plugin available from the NetBeans Beta Update Center.

    For more on the JIRA plugin, see the following FAQ.

    For more information on how to use the Plugins manager to add plugins, see Updating the IDE and Managing IDE Update Centers.
