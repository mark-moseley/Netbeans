About Issue Trackers

See Also

Anb issue tracker helps you to track and resolve issues with your project. An issue tracker enables you to create issues and assign responsibility for resolving issues to members of the team collaborating on the project, and view the status and progress of issues.

Usually you designate one issue tracker for a project.

You can perform the following tasks from within the IDE.

  • Find issues
  • Report an issue
  • Resolve issues
  • Create an issue query
