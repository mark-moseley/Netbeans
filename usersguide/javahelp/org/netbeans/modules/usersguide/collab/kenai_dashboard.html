Working With Kenai Projects

See Also

The Kenai dashboard enables you to quickly view information about Kenai projects that you have joined or opened. If you are logged in to Kenai.com, the projects that you have joined are automatically listed in the Kenai dashboard.

To open the Kenai dashboard:

  • Choose Window > Kenai from the main menu.

Each Kenai project that you have opened of joined is represented by a node in the Kenai dashboard. The Kenai dashboard enables you to easily monitor the project's issues and access the project page on kenai.com.

You can expand the project node to access the following features.

  • Issues. The Issues node enables you to quickly view and submit project issues. You can work with issues directly in the IDE. For example, you can click New issue to open the New Issue tab in the main window and submit an issue.
  • Sources. The Sources node enables you to see the NetBeans projects associated with that Kenai project. Click Get to retrieve the source code to a local repository.

    (Kenai.com supports Subversion, Mercurial and Git repositories, but you can also specify your own external source code repository.)

  • Wiki. Click the wiki to open the project wiki in your browser.
  • Downloads. Click downloads to open the project downloads page in your browser. Project administrators can put files (for example, the latest build of a project) in the downloads space for project members to download.
