Working With Kenai Projects

See Also

If you are a member of a Kenai project or open a Kenai project, a node for the project is displayed in the Kenai dashboard.

Expand the project node to access the following features.

  • Issues. The Issues node enables you to quickly view and submit project issues. You can work with issues directly in the IDE. For example, you can click New issue to open the New Issue tab in the main window and submit an issue.
  • Sources. The Sources node enables you to see the project's source code repositories. You can click Get to retrieve the source code to a local repository.

    (Kenai.com supports Subversion, Mercurial and Git repositories, but you can also specify your own external source code repository.)

  • Wiki. Click the wiki to open the project wiki in your browser.
  • Downloads. Click the downloads to open the project downloads page in your browser. Project administrators can put files (for example, the latest build of a project) in the downloads space for project members to download.
