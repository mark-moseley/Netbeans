Opening a Kenai Project

See Also

If you are interested in a Kenai project or a member of a project, you can open the project directly from the IDE. You do not need to login to Kenai to open a Kenai project, and you can open a project even if you are not a member of that project.

To open a Kenai project:

  • Choose Team > Kenai > Open Kenai Project from the main menu.

    Alternatively, you can click Open project in the Kenai dashboard.

  • Type the search terms in the Open Kenai Project dialog box and click Search.
  • Select a project from the list. Click OK.

When you open a Kenai project, a node representing the project is displayed in the Kenai dashboard.
