Opening a Kenai Project

See Also

If you are interested in or a member of a project on Kenai, you can open the project directly from the IDE. When you open a project, a node representing the project is displayed in the Kenai Dashboard. From the project node you can view project issues and download the project sources.

  • You do not need to open a project if you are only interested in the project sources.

To open a Kenai project:

  • Choose Team > Kenai > Open Kenai Project from the main menu.

    Alternatively, you can click Open project in the Kenai dashboard.

  • Type the search terms in the Open Kenai Project dialog box and click Search.
  • Select a project from the list. Click OK.

To download project sources:

  • In the Kenai Dashboard, expand the Sources node of the project and click get to open the Get Sources from Kenai wizard.

    Alternatively, choose Team > Kenai > Get Sources from Kenai.

  • Select the Kenai Repository from the dropdown list.
  • Specify the folder in the remote repository that you want to retrieve.
  • Specify the location for the local repository. Click Get From Kenai.

After checkout is complete, you can choose File > Open Project to open the project (if it is a NetBeans project) or work with the files in the Favorites window.

Projects on Kenai are usually open source. In most cases, you do not need to login to Kenai or be a member of a project to open the project or get the sources. In some cases, however, the project administrator might modify the default project permissions to limit access.
