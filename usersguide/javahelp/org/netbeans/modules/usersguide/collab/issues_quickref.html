Issue Tracking Tasks: Quick Reference

The following table describes common issue tracking tasks.

To perform this task Follow these steps
Find open issues
  1. From the main menu, choose Team > Find Issues.
  2. Select the issue tracker for the project.
  3. Specify the search criteria.
  4. Click Search.
Report an issue
  1. From the main menu, choose Team > Report an Issue.
  2. Select the issue tracker for the project.
  3. Specify the details of the issue.
  4. Click Submit Issue.
Update issue
  1. Find the issue that you want to resolve.
  2. Open the issue.
  3. Modify the fields.
  4. Click Submit Changes
Update issue
  1. Find the issue that you want to resolve.
  2. Open the issue.
  3. Modify the fields.
  4. Click Submit Changes.
Apply attached diff path
  1. Open the issue with the attached patch.
  2. Click "apply" next to the attachment.
  3. Locate the file or directory where you want to apply the patch. Click Apply.
Add an issue tracker
  • Choose Window > Services from the main menu.
  • Right-click the Issue Trackers node and choose Create Issue Tracker.
  • Specify the connection details. Click OK.
