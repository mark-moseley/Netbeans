Adding an Issue Tracker

See Also

To use issue tracker support in the IDE, you need to register the issue tracker with the IDE. After you add the issue tracker, you can choose to find, report and resolve issues using that issue tracker in the IDE.

  1. Open the Services window.
  2. Right-click the Issue Trackers node and choose Create Issue Tracker.

    Alternatively, open the Find Issue or Report a New Issue window and click New next to the Issue Tracker dropdown list.

  3. Specify the tracker connection details. Click OK.

When you click OK, a node for the issue tracker is added below the Issue Trackers node in the Services window. You can select the tracker from the Issue Trackers dropdown list in the Find Issues window (Team > Find Issues) or the Report a New Issue window (Team > Report an Issue).

  • The Issue Trackers node in the Services window only displays issue trackers that are not associated with a project on Kenai.
