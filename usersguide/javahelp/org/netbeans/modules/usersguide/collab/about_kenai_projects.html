About Kenai Projects

See Also

A Kenai project represents a collection of services provided by Kenai.com that are designed to help increase the productivity of developers by providing an infrastructure for developers collaborating on a project.

A Kenai project can be used to contain any type of project (for example, Java, Ruby, PHP) and can contain multiple sub-projects. Each Kenai project can have a dedicated source repository for the project, an issue tracking system, wiki, a forum and a space for downloads associated with that project.

For more details about the features available, see the following document.

Each Kenai project has a corresponding project page on Kenai.com. Project administrators can use the project page to enable and disable features and add members who can collaborate on the project.

Kenai Project Integration

Support for Kenai services is tightly integrated with the IDE. From within the IDE you can perform the following tasks:

  • log in to Kenai.com
  • create new Kenai projects
  • search for and open Kenai projects
  • retrieve and commit project sources
  • use the issue tracking system for the project

Use the Kenai dashboard (Window > Kenai Dashboard) to work with projects on Kenai. When you create or open a Kenai project in the IDE, the project is added to the Kenai dashboard.
