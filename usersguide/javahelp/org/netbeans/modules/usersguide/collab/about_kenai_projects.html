A Kenai project represents a collection of services designed to help increase the productivity of developers by providing an infrastructure for developers collaborating on a project.

After you create or join a Kenai project, you have access to a dedicated source repository, issue tracking system, wiki, forum and space for downloads. Each Kenai project has a corresponding project page on kenai.com. Project administrators can use the project page to enable and disable features and add members who can collaborate on the project.

For more information about the features available, see the following document.

Support for Kenai services is tightly integrated with the IDE. From within the IDE you can perform the following tasks:

  • log in to Kenai.com
  • create new Kenai projects
  • search for and open Kenai projects
  • work with Kenai projects that you have joined
A Kenai project can be of any type (for example, Java, Ruby, PHP) and can contain multiple sub-projects.

In the IDE, information about Kenai projects are displayed in the Kenai dashboard. After you login to kenai.com, you can open the Kenai dashboard to view information about all Kenai projects for which you are a member. You can search for and open Kenai projects even if you are not a member of that project.
