New Kenai Project Wizard: Name and Licenses

See Also

The New Kenai Project wizard enables you to create a new project on Kenai.com. You open the wizard by choosing Kenai > New Kenai Project from the Team menu.

This wizard enables you to create a project on Kenai from within the IDE. When you create the project, the IDE sends the project information to Kenai to create the project and Kenai services are assigned to the project.

  • You must be logged in to Kenai to create a Kenai project.

In the Name and Licenses page you need to set the following properties.

  • Logged in to Kenai as. If you are logged in, displays the user name. Click Login to Kenai if you are not logged in to open the log in dialog box.
  • Kenai Project Name. The name that is used for the project URL on Kenai.com.
  • Project Title. The display name of the location. This name is used to identify the project in the IDE and the project page on Kenai.com.
  • Project Description. (optional) A description of the project. You can add or modify the description on Kenai.com.
  • Project License. Select a license from the drop-down menu. You can modify the license on Kenai.com.
