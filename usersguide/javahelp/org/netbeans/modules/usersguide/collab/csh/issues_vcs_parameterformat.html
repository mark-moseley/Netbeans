Use this dialog to specify the format for the information that is added to issues and commit messages.

The Specify Format dialog is invoked from the Commit dialog when you click Change Format.

  • Add Revision Information to the Issue.
  • Add Issue Information to the Commit Message.
  • Text. The Text window displays the format that is used for the message. You can modify the text in the window to change the
  • Position. You can choose if the information is displayed below or above the message.
