Options: Miscellaneous: Issue Tracking

See Also

You use this panel to configure properties for the issue trackers. You can access this panel by choosing Tools > Options from the main menu, clicking the Miscellaneous category and then clicking the Issue Tracking tab.

  • Issue Tracking Systems. Choose the an issue tracking system from the list to modify the properties of that tracker.

    Bugzilla

    • Refresh opened issues. Specify how often the IDE checks open issues for changes to the issue.
    • Refresh opened queries. Specify how often the IDE checks open queries for udates.
