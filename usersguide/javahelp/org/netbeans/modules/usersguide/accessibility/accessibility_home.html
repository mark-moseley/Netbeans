Accessibility

The IDE provides many accessibility features that allow you read about and use the product in the manner most comfortable and convenient to you.

Keyboard Shortcuts

The IDE provide keyboard shortcuts for most actions. To see the available keyboard shortcuts, and edit them if you wish, choose Tools Keyboard Shortcuts. For information on changing or adding keyboard shortcuts, see Editor Shortcuts and Menu Shortcuts.

Style Sheet

You can use your own style sheet for formatting the README, Javadoc documentation, and the online help. For more information see Specifying a Style Sheet.
