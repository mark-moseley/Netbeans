About Accessibility

The IDE provides many accessibility features that enable you to read about and use the product in the manner that is most comfortable and convenient to you.

Keyboard Shortcuts

The IDE provides keyboard shortcuts for most actions. To see the available keyboard shortcuts, and edit them if you want, choose Tools > Keyboard Shortcuts. For information on changing or adding keyboard shortcuts, see Editor Shortcuts and Menu Shortcuts. For information on the keystrokes needed to navigate through the help viewer, see Help Viewer Shortcuts.

Mnemonics

The IDE provides mnemonics throughout the application. A mnemonic is an underlined letter that shows you which key to press in conjunction with the Alt key to activate a command or navigate to a component.

Fonts and Colors

You can set the font size of the IDE user interface in the IDE-HOME/etc/netbeans.conf file. For instructions, see IDE Startup Parameters.

You can also set individual font and color properties for the Source Editor, Diff viewer, and Help viewer.

Code Completion

The IDE has a dynamic code completion feature for the Java editor, HTML editor, and JSP editor. This feature enables you to type a few characters and then display a list of possible matches that can be used to automatically complete the expression.

Menus and Toolbars

You can customize the menus and toolbars in the IDE by adding and removing commands. For more information on these features, see Customizing the Display of Commands in a Menu and Customizing the Display of Commands in a Toolbar.

Web Browsers

You can use a web browser to view the online help. For information, see Displaying Help in a Web Browser.
