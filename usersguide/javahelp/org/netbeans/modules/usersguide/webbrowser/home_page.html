Setting a Home Page for the Web Browser

See Also

You can use the Options window to set the home page for the default web browser. The browser opens to this home page when open the browser by choosing View > Web Browser from the main menu.

  • This does not change the home page specified in the preferences for the web browser.
  1. From the main menu, choose Tools > Options and click Advanced Options in the Options window.
  2. In the left pane of the Options window, expand IDE Configuration > System and select the System Settings node.
  3. In the property sheet in the right pane, find the Home Page property, click its value or the ellipsis button (...), and type the URL of the new home page.
