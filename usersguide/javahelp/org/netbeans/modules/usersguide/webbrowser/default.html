Specifying the Default Web Browser

See Also

You can configure the IDE to use either an external web browser or the internal Swing HTML web. Both types of browsers enable you to view external and local web pages, including online help and Javadoc documentation. The Swing HTML browser provides the highest level of integration with the IDE because you can embed an internal browser into another IDE window or an MDI frame.

The preferred browser for use with the IDE is an external browser, which provides more functionality than an internal browser and does not close when you shut down the IDE. A disadvantage of using an external browser is that you cannot embed it in an IDE window. Three implementations of the external browser are available for use with the IDE:

  • External browser (Command Line). A simple browser that executes a specified command when asked to display a URL. Users will typically customize this browser to run Netscape NavigatorTM.
  • External browser (Windows). The HTML browser, such as Internet Explorer or Netscape Navigator, set as the default browser on your Microsoft Windows platforms.
  • External browser (UNIX). The HTML browser set as the default browser in the UNIX environment.

At startup, the default browser is the Microsoft Windows or UNIX external browser, depending on your platform.

To change the default web browser:

  1. From the main menu, choose Tools  Setup Wizard.
  2. In the Setup wizard, select a browser type from the Web Browser combo box.
