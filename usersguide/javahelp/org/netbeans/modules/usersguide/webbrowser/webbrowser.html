Opening a Web Browser from the IDE

See Also

By default, the IDE opens your system's default web browser. If necessary, you can specify a different browser in the Options window.

To launch the web browser and open the specified home page.

  • Choose View > Web Browser from the main menu.
    The default browser opens to the home page specified in the IDE Options window. For more, see Setting a Home Page for the Web Browser.

Troubleshooting

If the IDE cannot find the operating system's default web browser, try one of the following:

  • Choose a different web browser.
    1. Choose Tools > Options from the main menu and click General in the left pane of the Options window.
    2. Select a different browser from the Web Browser drop-down list.
  • Specify the location of the browser executable.
    1. Choose Tools > Options from the main menu and click Advanced Options in the Options window.
    2. Expand IDE Configuration > Server and External Tools Settings > Web Browsers and select the browser type you are using.
    3. Open the property editor for Browser Executable property and set the full path to the executable in the Process field. You can click the ellipsis button (...) to browse to the executable.
