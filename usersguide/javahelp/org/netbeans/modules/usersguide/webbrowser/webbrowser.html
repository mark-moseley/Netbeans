Using the Internal Web Browser

The IDE has an internal web browser, which you can use to access web sites that are bookmarked in the Help menu or other sites on the World Wide Web.

To open the internal web browser:

  • Choose View Web Browser from the main window

The toolbar contains buttons to cycle back and forth through previously-seen pages, to stop the current page from being loaded, to reload the current page, to switch to the home page, and to view the history of previously loaded pages. You can load a new page by typing its URL in the Location field and pressing Enter on your keyboard.

You can search for text strings in the web browser by choosing Edit Find from the main window or by using the Ctrl-F keyboard shortcut.

To set a different home page:

  1. Choose Tools Options from the main menu.
  2. In the Options window, select the System Settings node and type the new home page in the Home Page property.

To adjust the fonts, the background color, or encoding in the web browser:

  1. Choose Tools Options from the main menu.
  2. In the Options window, select the HTML Browser node.
  3. Make the desired changes for the Default Background, Encoding, Fixed Font, and Proportional Font properties.

To add a bookmark to the list of bookmarks in the Help menu:

  1. Choose Tools Options from the main menu.
  2. In the Options window, right-click the Bookmarks node and choose New Bookmark.
  3. Make the desired changes for the Default Background, Encoding, Fixed Font, and Proportional Font properties.
