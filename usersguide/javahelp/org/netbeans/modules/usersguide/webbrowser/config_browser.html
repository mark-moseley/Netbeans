Configuring a Web Browser

  1. Choose Tools  Options from the main menu.
  2. Expand IDE Configuration  Server and External Tools Settings  Web Browsers and select the browser type you are using.

To specify the path to a browser's executable:

  1. Select the browser in the Options window.
  2. Open the property editor for the Browser Executable property.
  3. Enter the full path to the browser executable in the Process field. You can also use the ellipsis button (...) to browse to the executable.
  4. Enter any arguments in the Arguments field.
  5. Click Close.

To register a new browser with the IDE:

  1. Right-click the Web Browsers node and choose New  External Browser.
