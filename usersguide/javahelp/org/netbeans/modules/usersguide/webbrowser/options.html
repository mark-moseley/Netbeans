Options Window: Web Browser

See Also

You can use the Options window to specify options for the web browsers registered with the IDE.

You can access the web browser options by choosing Tools > Options from the main menu, and then clicking General.

The Web Browsers drop-down in the Options window contains subnodes for each of the IDE's registered web browsers, such as Internet Explorer and Mozilla Firefox. The <Default System Browser> node represents your operating system's default web browser.

Properties:

  • Browser Executable. This property enables you to specify the path to the browser executable and any arguments that are used when the browser is activated. To specify the path to the browser executable, click the ellipsis button (..) to open the Browser Executable dialog box and enter the path to the browser executable in the Process field. Alternately, click the ellipsis button (...) and navigate to the location of the web browser executable.
