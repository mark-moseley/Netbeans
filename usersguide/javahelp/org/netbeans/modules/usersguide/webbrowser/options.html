Web Browser Options

See Also

The Web Browsers node in the Options window contains subnodes for each of the IDE's registered web browsers, such as Internet Explorer and Netscape. The <Default System Browser> node represents your operating system's default web browser.

You can view these settings by opening the Options window and expanding IDE Configuration  Server and External Tool Settings  Web Browsers.

Except for the Swing HTML Browser, each web browser has the following properties:

  • Browser Description. Provides a description of this browser type.
  • Browser Executable. Specifies the path to the browser executable and any arguments that are used when the browser is activated. If the IDE cannot find your browser, use the property editor for this property to check that the executable location is correct.
