Creating a Template

See Also

You can save a file as a template and add it to the types of templates available in the template chooser.

To create a new template:

  1. In the Projects window, right-click the file you would like to turn into a template and choose Save As Template.
  2. In the Save As Template dialog box, select a category for the template and click OK.

To modify a template:

  1. Choose Tools > Options and click the Advanced Options button at the bottom of the Options widnow.
  2. Expand Source Creation and Management > Templates.
  3. Expand the appropriate category node and navigate to the template.
  4. Right-click the template's node and choose Open.
  5. In the Source Editor, make the desired changes to the template.
  6. Choose File > Save from the main menu.
