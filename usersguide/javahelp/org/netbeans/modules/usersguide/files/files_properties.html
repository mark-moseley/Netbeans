Most objects and object elements that are represented by nodes in the Projects window have properties that can be viewed in a property sheet. The Properties window displays the property sheet for the currently selected node in the Projects window. To open the Properties window, choose Window > Properties.

When the Properties window is open, it displays the names and values of the properties of the currently selected node in the Projects window. Depending on the type of object selected, you might see additional tabs for distinct categories of properties.

Editing Properties in the Properties window

You can edit the values of most properties in the Properties window. To change a property value, click the property's value field. The appearance of the field changes depending on the type of value required:

  • If a string value is required, the field becomes a text field. You can type the new value directly in this field and press Enter.
  • If a specific list of values is allowed, a drop-down arrow appears. You can click this arrow and select the value from the combo box that appears. You can also double-click the property name to rotate through the available values.
  • If a property editor is available for that property, an ellipsis (...) button appears. You can click the ellipsis button to open the property editor.

Node Properties Sheets in the IDE

You can display property sheets for selected nodes in the IDE's various windows, such as:

  • The Options window
  • The Debugger window
  • The Form Editor (if your IDE version includes the Form Editor module)
  • You can also open Properties dialog boxes that are dedicated to a particular nodes by right-clicking the node and choosing Properties from the pop-up menu. Even when you select another node, this Properties dialog box continues to display properties for the node from which it was called.
