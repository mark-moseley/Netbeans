Runtime Window

See Also

The Runtime window is the main entry point to your runtime resources. It shows a logical view of important runtime resources such as your servers, databases, and registered web services. You can open the Runtime window by choosing Window  Runtime (Ctrl-5).

Runtime Resources

The following table lists the runtime resources as they appear in the Runtime window, with their icons and their descriptions.
Icon Description
Servers
Lists all servers currently registered with the IDE. Only the bundled Tomcat Web Server is registered by default. If your version of the IDE is bundled with the Sun Java System Application Server (SJS Application Server), this server is registered by default as well. Use the IDE to register all other servers.
Processes
Lists all processes currently being executed within the IDE. You can terminate a running application by right-clicking its node and choosing Terminate Process. This information also appears in the Execution view, which you can open by choosing Window  Execution.
Databases
Lists all registered database drivers.
HTTP Server
When you right-click the node, you can choose commands to start or stop the IDE's internal HTTP server. You can also configure HTTP server options.
VCS Commands
Lists a history of VCS commands called in the current IDE session.
Web Services
Lists all registered web services. A web service that is registered can be tested from the IDE. Before you can register a web service, you must publish it.
If the SJS Application Server is not registered with the IDE, this node is disabled.
XML Entity Catalogs
Lists all your currently mounted XML Catalogs. When you right-click the node, you can mount and unmount XML catalogs.

Other nodes may appear, depending on the modules that are installed in the IDE.
