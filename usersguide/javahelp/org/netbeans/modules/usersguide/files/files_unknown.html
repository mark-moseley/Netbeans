Working with Unknown File Extensions

See Also

If you have a file of a type that the IDE does not recognize, you can register it as a known file type.

To register a file extension as belonging to a file type:

  1. Choose Tools  Options.
  2. In the Options window, expand IDE Configuration  System  Object Types.
  3. Select the node of the object type you want to add the file extension to.
  4. Select the Extensions and MIME Types property and add your extension to the list.
When you register a file extension as belonging to a file type, you do not necessarily get all of the Source Editor features that come with that file type. For example, code completion does not work on files that you designate as belong to the Java file type.
