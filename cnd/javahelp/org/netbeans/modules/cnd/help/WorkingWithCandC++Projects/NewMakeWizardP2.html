New C/C++ Project From Existing Code Wizard: Makefile and Build Actions

See Also 

The New C/C++ Project From Existing Code wizard enables you to create a project using an existing makefile.

In the Makefile and Build Actions page, you have to set the following properties:

  • Makefile. Specifies the path to the existing makefile for the project.
  • Build Result. Specifies the path to the output (executable or library) generated by the existing makefile.

After you specify the path to the makefile, the wizard automatically fills in the following fields for you:

  • Working Directory. Specifies the directory where you want the IDE to execute the make and clean commands..
  • Build Command. Specifies the make command for the project.
  • Clean Command. Specifies the make clean command for the project.
