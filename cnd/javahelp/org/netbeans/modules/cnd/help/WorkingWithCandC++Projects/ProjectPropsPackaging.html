C/C++ Project Properties Dialog Box: Packaging

See Also 

The Packaging Panel lets you specify the properties for packaging your completed project as a tar file, zip file, or Solaris SVR4 package. You can specify the package type, the output path, the files to include in the package, and whether to build the package in verbose mode.
