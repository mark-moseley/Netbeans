New C/C++ Project From Existing Code Wizard: Name and Location

See Also 

The New C/C++ Project From Existing Code wizard enables you to create a project based on an existing makefile.

In the Name and Location page, you can set the following properties:

  • Project Name. Specifies the display name of the project. This name is also used to create the project folder.
  • Project Location. Specifies the folder where you want to store the project metadata.
  • Set as Main Project. Sets the newly created project as the main project.
