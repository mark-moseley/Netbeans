Configure Code Assistance Wizard: Simple Configuration

See Also 

For a C/C++ Project From Existing Code, the Configure Code Assistance wizard lets you specify how your code will be parsed by the built-in parser used for the code assistance features of the IDE.

The Simple Configuration page is displayed if you selected Simple Mode on the Select Mode page and your project has been built with debug information. The Simple Configuration page lets you specify additional libraries that shold be searched by the built-in parser, and the configuration level at which the options are set.

  • Additional Libraries. Additional libraries to be searched.
  • Configuration Level. You can select from the following levels:
    • Configure File Properties. Options are set for each file. This level creates the most precise configuration but makes project maintenance more difficult.
    • Configure Folder Properties. Options are set for each folder from a union of file configurations.
    • Configure Project Properties. Options are set for the entire project from a union of file configurations. This option makes project maintenance simple.
