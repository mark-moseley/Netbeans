Configuring C and C++ Semantic Highlighting

See Also 

You can use the Options window to configure semantic highlighting for your C and C++ projects.

To configure C and C++ semantic highlighting:

  1. Choose Tools > Options.
  2. Click C/C++ in the top pane of the window.
  3. Click the Semantic Highlighting tab.
  4. Modify the properties as desired.

You can set the following properties:

  • Enable Mark Occurrences. If this option is selected, then when you double-click on a class, function, or variable, all occurrences of that class, function, or variable in the current file are highlighted
  • Keep Marks. If this option is selected, highlighted classes, functions, and variables remain highlighted.
  • Highlight Macros. If this option selected, macros are highlighted.
  • Differ System Macros from User Macros. If this option is selected, system macros are highlighted such that you can differentiate between them.
