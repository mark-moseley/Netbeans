Configuring C and C++ File Extensions

See Also 

You can use the Options window to configure file extensions for your C and C++ projects.

To configure C and C++ file extensions:

  1. Choose Tools > Options.
  2. Click C/C++ in the top pane of the window.
  3. Click the Other tab.
  4. Modify the properties as desired.

You can set the following properties:

  • C File Extensions. The accepted extensions for C source files are listed with the default extension in bold type. Click Edit to edit the list. In the Extensions List dialog box, you can add an extension, remove an extension, or change the default extension.
  • C++ File Extensions. The accepted extensions for C++ source files are listed with the default extension in bold type. Click Edit to edit the list. In the Extensions List dialog box, you can add an extension, remove an extension, or change the default extension.
  • C/C++ Header File Extensions. The accepted extensions for C and C++ header files are listed with the default extension in bold type.Click Edit to edit the list. In the Extensions List dialog box, you can add an extension, remove an extension, or change the default extension.
