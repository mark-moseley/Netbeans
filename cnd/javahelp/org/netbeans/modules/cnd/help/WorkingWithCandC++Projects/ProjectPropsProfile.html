C/C++ Project Properties Dialog Box: Profile

See Also 

The Profile panel enables you to specify the properties of the profiling tools that are displayed in the Run Monitor tab when C/C++ projects are run.

  • Profile on Run: Select to run profiling tools when the project runs.

  • Data Provider: Select the tool to collect data on your running program. Choices shown depend on your operating system and toolchain. On Windows platforms only Simple can be used, and detailed data cannot be shown. Sun Studio data provider can be used on Solaris and Linux systems where Sun Studio tools are installed. DTrace can be used only on Solaris platforms.

  • Indicators: Select which indicator graphs to display when the project runs.
