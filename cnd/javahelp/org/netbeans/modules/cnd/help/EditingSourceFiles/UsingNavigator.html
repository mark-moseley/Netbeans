Using the Navigator

See Also

The Navigator window provides a compact view of the file that is currently selected, and simplifies navigation between different parts of the file.

By default, the Navigator window is open as a tab in the same area of the IDE as the Class View. To open the Navigator window, choose Window > Navigator.

To display source file in the Navigator window:

  • Select the file in the Editor window, the Projects window, or the Files window.

To navigate to an element of the file displayed in the Navigator window:

  • Double-click the element in the Navigator window and the cursor in the Editor window moves to that element.
