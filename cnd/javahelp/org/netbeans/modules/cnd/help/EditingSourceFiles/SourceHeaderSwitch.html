Switching Between Source and Header Files

See Also

The Source/Header switch lets you easily visit corresponding source and header files. The switch toggles between files with same name and corresponding extension, for example, code.cc and code.h.

To switch between source and header files, do one of the following:

  • With the cursor in a source or header file in the Source Editor window, click the Go to Source/Header button in the tool bar.
  • With the cursor in a source file in the Source Editor window, right-click and choose Go to Header.
  • With the cursor in a header file in the Source Editor window, right-click and choose Go to Source.
If a file with the same name and a corresponding extension is found in the project, the file is opened in the editor.
