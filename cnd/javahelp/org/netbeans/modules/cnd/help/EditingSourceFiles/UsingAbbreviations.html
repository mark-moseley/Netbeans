Using Code Templates

See Also

The Source Editor has a set of customizable code templates for common snippets of C and C++ code.

You can generate snippets from these code templates by doing the following:

  • Type the abbreviation for the code template and then press the spacebar to generate the full code snippet. For example, you can type iff, followed by the spacebar, to automatically generate if () {}. See C and C++ Code Template Abbreviations: Quick Reference for a list of abbreviations

To keep a code template abbreviation from expanding into the full code snippet, type Shift-Spacebar after typing the abbreviation.
