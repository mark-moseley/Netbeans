Creating a Makefile

See Also

You can create a makefile using the Makefile wizard. To open the wizard, right-click the project node of the project in which you want to create the makefile and choose New > File/Folder.

In the Choose File Type page of the wizard, do the following:

  1. Choose the file type:
    • For a simple makefile, expand the Makefiles category and the Simple Makefiles subcategory. Select the file type that is appropriate for your project. The Description pane displays a description of the selected file type.
    • For a makefile to build multiple targets or an executable linked with libraries, select the Makefiles category and the Makefile file type.
  2. Click Next when you have selected the file type you want.
