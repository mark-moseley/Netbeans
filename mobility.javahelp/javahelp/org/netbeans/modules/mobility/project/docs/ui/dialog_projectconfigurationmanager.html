Project Configuration Manager

See Also

Use the Project Configuration Manager to add a new to a project or remove project configurations from a project.

You open the Project Configuration Manager by opening the Project Properties and clicking the Manage Configurations button.

The buttons provide the following actions:

  • Add. Opens the Add Project Configuration dialog, where you can enter a name for a new project configuration. The new configuration name is added to the list of available configurations. The new configuration has the same properties as DefaultConfiguration.
  • Duplicate. Opens the Duplicate Project Configuration dialog, where you can enter a name for a new project configuration. The new configuration name is added to the list of available configurations. The new configuration has the same properties as the configuration you are duplicating.
  • Remove. Deletes the selected configuration.
  • Save As Template. Saves the selected project configuration as a template. You can then use the template to create other project configurations in the Add Project Configuration dialog.
