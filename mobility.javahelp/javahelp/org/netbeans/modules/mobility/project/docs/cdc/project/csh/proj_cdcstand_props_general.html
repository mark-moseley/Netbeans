CDC Project Properties Dialog Box: General

The General panel enables you to set general options for your project. Here you change attributes such as application display name, UID, application icon and so on.

In the General options page, you can set the following properties:

  • Application UID. Specifies or generates unique digital inprint for your CDC application. For more information please see
  • Application Name. Specifies the display name of the application on the target device.
  • Application Icon. Specifies the icon that will be displayed together with the Application Name on the target device.
  • Application Vendor. Specifies the vendor name of the application.
  • Application Version. Specifies the version number of the application.
  • Autoincrement Version. If checked, the version number will be automatically incremented after each SIS file build.
