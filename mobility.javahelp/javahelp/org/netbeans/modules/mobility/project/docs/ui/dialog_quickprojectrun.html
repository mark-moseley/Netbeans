Choosing the Run With command from the MIDP Project node contextual menu enables you to test how a your application performs on different mobile devices.

Use this dialog to select an emulator platform and device to quickly run the most current build of your project. The IDE runs your application on the selected platform, but does not build the project or create a distribution JAR for that platform.
