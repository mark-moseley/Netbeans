Enter Password

See Also

Use the Enter Password dialog to enter a password when:

  • You are deploying an application and did not specify a password in the Deploying Properties page.
  • You are unlocking a locked keystore file or locked key pair.
    Some keystore files and individual key pairs within a keystore file are locked with passwords as an additional security method. You have to unlock these keystores and key pairs before you can use them to sign your MIDlet suite.

The Enter Password dialog opens when:

  • You are running and deploying an application and no password is set in the Deploying Properties page.
  • You select any locked keystore or key pair in the Signing Properties page or in the Add Keystore dialog and click Unlock.
    The Unlock button is only activated when you select a locked keystore or key pair.
