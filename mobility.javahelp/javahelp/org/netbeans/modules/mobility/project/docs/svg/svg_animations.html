Working with SVG Animations

See Also

The svgPlayer component in the Visual Mobile Designer lets you insert an animated SVG file anywhere in your MIDP application flow or within a screen. The player is supported by the SVGAnimator class in the JSR-226 API.

To add an animated SVG file into your application:

  1. Drag the svgPlayer component from the Component Palette and drop it in the Flow Designer.
  2. Drag an SVG file from the Navigator window and drop it on the component.

To control how the animation behaves in the screen or application flow, you can set the set the following properties:

  • Start Animation Immediately. Whether the component should start animation immediately when it appears on the display.
  • Animation Time Increment. The minimal time, in seconds, that should elapse between frame rendering. This value must be greater than zero.
  • Reset Animation When Stopped - If true, the component automatically resets the animation to the its beginning when the component is dismissed from the screen. If false, the animation is not reset and next time the component is displayed, the animation automatically continues where it was stopped.
  • Full Screen - Determines whether the component is shown in a full screen.
