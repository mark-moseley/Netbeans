New Mobile Project Wizard:
Default Platform Selection Page

See Also

You open the New Mobile Project wizard by choosing File > New Project and selecting a project from the Mobile category.

In the Default Platform Selection page, you choose the initial for the project. The emulator platform determines the execution environment for the project.

  • After you create the project, you can create additional project configurations with alternate platform settings.

You set the following properties on this page:

Emulator Platform. The execution environment for the project. The drop-down menu provides a list of available emulator platforms.

Device. A specific mobile device emulator supported by the selected emulator platform. When executed, the application's performance is simulated on this device.

Device Configuration. The version of supported by the emulator platform. Choices that are not supported by the selected emulator platform are grayed out.

Device Profile. The version of supported by the emulator platform. Choices that are not supported by the selected emulator platform are grayed out.

Clicking the Next button takes you to the More Configurations page. In this page, you can select additional project configurations to add to your project.

Clicking the Finish button completes the wizard and creates a project you can view in the Projects window of the IDE.
