Export Key

See Also

Use the Export Key dialog to export a key pair and its corresponding certificate to an .

You open the Export Key dialog from the Signing page of the project properties and clicking the Export Key into Java ME SDK/Platform Emulator button, or from the Security Manager dialog by selecting a keystore and clicking the Export button.

The Export dialog has the following components:

  • Keystore File. Shows the name of the keystore file to which the key pair belongs. This field cannot be edited.
  • Key Pair Alias. The name given to the key pair within the keystore. This field cannot be edited.
  • Certificate Details. Shows the details of the certificate of the key to be exported.
  • Platform. Enables you to select the the key is to be exported to emulator platform. The drop-down menu lists all the emulator platforms available.
  • Security Domain. Enables you to select a security domain for the key pair. The menu lists all domains supported by the selected emulator platform.
  • Keys Registered in the Platform. Lists all keys that have been registered in the selected platform.
  • Delete Key. Enables you to delete a key you select from the Keys Registered in Platform window.
  • Export. Exports a key to the selected emulator.
