About the Game Builder

See Also

The Game Builder enables you to design a new game using javax.microedition.lcdui.game package included in MIDP 2.0 that is based on the simple game included in the MIDP 2.0 samples list of the IDE. You can add scenes, sprites and layers to your game using the included elements, or make completely new games based on content you create.

The following elements are typically used in the design of MIDP games:

  • Sprites. components, and associating commands with the components. It enables you to view forms in both Source and Design views.
  • Tiled Layers. Enables you to design and modify screens within your game application.
  • Scenes Scenes are withing levels of the game. Each scene has a scenario the player must successfully play thorugh..
