The MIDP Project node in the Projects window () provides a view of a MIDP application in the Projects window. Each MIDP Project node expands to reveal the contents of the MIDP application's (MIDlet suite's) JAR file.

You can create a project in the New Project wizard. For more information, see Creating a MIDP Project.

Contextual Menu

Each node in the Projects window has a contextual menu of commands that you can run on that node. You open a node's contextual menu by right-clicking the node. The contextual menu for project nodes contains all of the major project commands, like Build, Run, and Debug. The contextual menu also contains a Properties command, which you can use to configure your project.

Commands specific to MIDP projects include:

  • Quick Run With. Enables you to select a device emulator and platform to test how a your current build works with different mobile devices.
  • Set Active Project Configuration. Enables you to change the current or create a new project configuration.
  • Build All Configurations. Builds a distribution JAR file for each project configuration.
  • Clean & Build All Project Configurations. Cleans and builds a distribution JAR file for each project configuration.
  • Clean All Project Configurations. Removes distribution JAR files for all the existing project configurations.
  • Deploy All Project Configurations. Builds and deploys a MIDlet suite for each project configuration.
