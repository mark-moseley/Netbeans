Use the Add Ability dialog to associate an ability with a project configuration or to create new abilities.

You open the Add Ability dialog by clicking the Add button in the Abilities Property page.

To associate an ability with the current project configuration:

  • Choose an ability from Ability Name drop-down list and click OK.

    • Note: The Ability Name drop-down list displays all of the abilities that have been created previously in the IDE.

To create a new ability:

  1. Choose any Ability Name from the drop-down menu.
  2. Change the selected ability name to match the characteristic you are adding to the Ability Name list.
  3. Add a value such as screen size or resolution, if needed, to the Value field and click OK.
    The new ability is added to the list of Abilities drop-down list.
