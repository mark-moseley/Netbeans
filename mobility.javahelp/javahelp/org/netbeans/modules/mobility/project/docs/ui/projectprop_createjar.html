MIDP Project Properties: Creating JAR

See Also

Use the Creating JAR properties page to modify JAR and creation settings in a MIDP project.

You open the Creating JAR properties page by choosing File > <active project name> Properties. In the Project Properties dialog, select Build > Creating JAR.

  • If you are setting the properties for a project configuration other than the default configuration (DefaultConfiguration), you must uncheck the Use Values from "DefaultConfiguration" checkbox at the top of the page to customize these options.

The Creating JAR page lets you set the following options:

  • JAD File Name. Name of the JAD file created by the project sources.
  • JAR File Name. Name of the JAR file created by the project sources.
  • Compress JAR. If checked, the JAR file is compressed.
