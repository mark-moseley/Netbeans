MIDP Project Properties: Application Descriptor

See Also

You can view and modify the attributes of the Java Application Descriptor (JAD) file of a project configuration by choosing File > <active project name> Properties. In the Project Properties dialog, select any of the following:

  • Attributes. Lists the required, optional, and user-customized attributes of the selected project configuration.
  • MIDlets. Lists the MIDlets contained in the MIDlet suite for the selected project configuration.
  • Push Registry. Lists Push Registry Entries for classes in the selected projected configuration.
  • API Permissions. Lists the API Permissions assigned for the selected project configuration.

The active project configuration is listed in the dropdown menu at the top of the Project Properties dialog.
