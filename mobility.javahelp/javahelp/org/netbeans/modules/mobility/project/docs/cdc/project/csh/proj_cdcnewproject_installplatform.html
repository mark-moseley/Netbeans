New CDC Project Wizard: Install Platform Page

You open the New CDC Project wizard by choosing File > New Project and selecting a project from the CDC category.

The Install Platform page is called if there is no Java ME CDC platform-compatible SDK installed in the IDE.

To add a CDC Platform SDK:

  1. Click the Install SDK/Platform/Emulator button.
  2. Use the Add Platform wizard to install the CDC platform into the IDE.

After a CDC platform is installed, you will be returned to this page to continue with creation of a new project. Click Next to continue.
