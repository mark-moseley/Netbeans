Visual Mobile Designer Action Editor

See Also

The Visual Mobile Designer (VMD) Action Editor enables you to edit the properties associated with commands.

You open the Action Editor by double-clicking on a transition in the Flow Designer, or by clicking on the Ellipsis (...) button of the Action properties of a command.

In the Action Editor, you can edit three areas of action properties:

Action

What should be done when this action is performed. Three options are available:

  • Switch to screen. When the action is performed, the flow switches to a new screen:
    Target Screen. The new screen that appears.
    • Alert Forward Screen. If the target screen is an alert, use the dropdown menu to choose the screen that appears after the alert is closed.
    • Exit application. When the action is performed, the application is closed.
    • Do Nothing. No switching occurs.

In the Generated Code box, the IDE shows the actual snippet of code to be generated.
