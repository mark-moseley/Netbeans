Screen Designer

See Also

The Screen Designer enables you to customize screens that have been added to the flow of your application in the Flow Designer window.

The Screen Designer window opens when you double-click on a Screen icon in the Flow Designer window. The Screen Designer has two main sections:

  • Device Screen
  • Assigned Commands

Device Screen

The Device Screen simulates how the component appears on a real device. All components appear in the order they are listed in the Inspector view.

New components can be dragged and dropped directly to the desired location of the Device Screen. Only components that are valid for the screen type can be added.

To change the size of a Device Screen, right-click on the Device Screen and choose Set Screen Size. This opens a dialog where you can set the height and width of the device screen in pixels.

Assigned Commands

This portion of the Screen Designer shows all the Transition Sources attached to the screen. You can select the box to display and edit the properties of the Transaction Source in the Properties view.
