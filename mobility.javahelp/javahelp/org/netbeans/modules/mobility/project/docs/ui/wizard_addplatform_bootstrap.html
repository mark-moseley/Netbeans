Bootstrap Libraries page

The Add Java Platform wizard enables you to register Java platform and Java ME, MIDP platform-compatible SDKs or , such as the Sun Java Wireless Toolkit, in the NetBeans IDE. Once you register a platform emulator, you can use to set that platform for compilation, execution and debugging.

Use the Bootstrap Libraries page of this wizard to specify the classpaths of libraries used by a custom (non-UEI compliant) emulator platform.

After you have entered the information required by the General Information page, the Bootstrap Libraries page detects the libraries and APIs used by the platform.

You can add additional libraries by clicking the Browse button and navigating to the library file you want to add.

You can click Next to specify additional source files and Javadoc, or click Finish to register the emulator platform in the IDE.
