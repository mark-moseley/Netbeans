Save Mobility Project Configuration as Template

See Also

Saving a project configuration as a template enables you to use the same configuration settings for multiple projects. For example, a configuration that supports a mobile device with a specific screen size can be saved as a template and used for other devices with the same screen size.

You open this dialog by clicking the Manage Configurations button in the Project Properties dialog, selecting a project configuration in the Project Configuration Manager and clicking Save as Template.

In the project configuration name field, enter a name for the configuration template. The name can be any valid Java identifier.
