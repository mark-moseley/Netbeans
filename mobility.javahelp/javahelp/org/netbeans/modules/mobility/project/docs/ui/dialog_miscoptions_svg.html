Options Window: Miscellaneous: SVG

See Also

The SVG tab in the Options > Miscellaneous window enables you to configure the IDE to launch an external program to edit Scalable Vector Graphics (SVG) files.

To configure the IDE to use an external SVG editor:

  1. Choose Tools > Options from the main menu.
  2. Click the Miscellaneous icon, then select the SVG tab.
  3. Click the Browse button in the External Editor Path field.
  4. Navigate to the location on your system where the SVG editor you want to use is installed.
  5. Choose the program_name.exe of the SVG editor then click Select. Click OK to register the selected program in the IDE.
