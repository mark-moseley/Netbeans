Import Wireless Toolkit Project Wizard:
Specify Wireless Toolkit Project Page

See Also

You open the Import Wireless Toolkit Project wizard by choosing File New Project and selecting the Import Wireless Toolkit Project template from the Mobility category. This wizard enables you to import projects you have created using the Sun Java Wireless Toolkit emulator platform.

In the Specify Wireless Toolkit Project page, choose the project you want to import.

Use the Browse button to choose the directory where the project is stored. The IDE displays all the available applications contained in the directory.

Select a project and choose Next to continue.
