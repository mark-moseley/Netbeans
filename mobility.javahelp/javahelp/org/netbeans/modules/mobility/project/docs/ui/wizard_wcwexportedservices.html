J2ME Wireless Connection Wizard: Exported Services

See Also

The Wireless Connection Wizard simplifies the process of creating distributed, mobile client/server applications. The wizard generates the client-side classes and a servlet that delegates the client requests to the entrprise services. You open the Wireless Connection Wizard by selecting a Mobile Project from the Projects view and choosing File New File, then choosing MIDP Mobile Enterprise Wireless Connection Wizard.

In the Exported Services page, you select the enterprise methods that you want exported to the MIPD client. .

You can use the J2ME Wireless Connection Wizard to export methods from the server that meet the following criteria:

  • The class containing the methods must have public access
  • The class containing the methods must contain a public constructor that takes no parameters
  • An exported method must have public access
  • An exported methods's parameter types and return types must be supported.

The following data types are supported as parameter types and return types.

  • boolean, boolean[]
  • byte, byte[]
  • short, short[]
  • char, char[]
  • int, int[]
  • long, long[]
  • java.lang.String
  • java.lang.String[]

To export services, expand the menu tree and check the check boxes next to the methods you want the client application to be able to access.
