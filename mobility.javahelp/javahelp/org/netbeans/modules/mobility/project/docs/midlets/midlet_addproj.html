Adding a New MIDlet to a Project

See Also

To add a New MIDlet to a project:

  1. Select a Project and choose File > New File.
  2. Under Categories, choose MIDP. Under File Types, choose MIDlet.
  3. Choose the appropriate file template for your needs.
