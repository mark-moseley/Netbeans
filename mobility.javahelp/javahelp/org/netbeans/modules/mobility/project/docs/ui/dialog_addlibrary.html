Add Libraries

See Also

Use the Add Libraries dialog to add libraries or resources to your project's classpath. You can also use this dialog to edit libraries by adding or removing JAR files, library sources, and javadocs.

You open the Add/Edit Libraries dialog by choosing File > <active project name> Properties. In the Project Properties dialog, select Build Libraries & Resources and click the Add Library button.

To add a library:

Select a library from the list of installed libraries and click Add Library.

To edit a library:

  1. Select a library from the list of installed libraries and click Edit Library.
    This opens the Library Manager, where you can edit the library.
  2. Click on one of the following tabs to edit the library's contents:
    • Classpath.
    • Sources.
    • Javadoc.
