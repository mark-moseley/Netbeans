This topic describes common tasks you can perform in the Game Builder:

To perform this task Follow these steps
Create a new project that uses the game builder sample as a foundation for your project.
  1. Choose File > New Project.
    This opens the New Project wizard.
  2. Under Categories, Choose Samples - Mobility - MIDP 2.0
  3. Under Projects, choose the Simple Game created with Game Builder.
  4. Complete the Name and Location page to suit your system preferences. Click Next
  5. Under Emulator Platform, choose the desired Emulator Platform.
  6. Under Device, choose desired target device.
  7. Device Configuration should be CLDC 1.1 and Device Profile should be MIDP 2.0 or better. Click Finish.
Designing a new game based on the game builder sample.
  1. Open the game builder sample in the IDE .
  2. Expand the Source Packages folder and select demo - GameDesign.java and open the file in the IDE.
  3. Click the link to Creates Scenes, Tiled Layers and Sprites. You can also create either of these new elements by clicking the icons in the top right corner of the source editor.

Create a file within a project that uses the game builder.
  1. Choose File > New File.
    This opens the New File wizard.
  2. Under Categories, Choose MIDP. Under File Types, choose either the Visual MIDlet or Hello Visual MIDlet template.
  3. In the Name and Location page, specify the name and location for the new MIDlet.
