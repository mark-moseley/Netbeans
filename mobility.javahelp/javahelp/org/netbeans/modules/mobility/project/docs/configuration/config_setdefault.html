Setting the Active Project Configuration

See Also

If you have more than one project configuration, you can change the active configuration any time.

To set the active configuration:

  • Right-click the Project node and choose Set Active Project Configuration > the desired configuration.
  • Or, if you have only one configuration created, you can choose Set Active Project Configuration > Add Configuration to Project. This opens the Add Configuration dialog and the Project Properties Sheet.
  • You can also change the configuration using the Configuration drop-down menu in the Toolbar.
