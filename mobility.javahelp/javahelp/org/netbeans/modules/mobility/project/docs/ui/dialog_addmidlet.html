Add/Edit MIDlet

See Also

Use the Add/Edit MIDlets dialog to add or edit an existing MIDlet to a project configuration.

You open the Add/Edit MIDlet dialog by clicking the Add or Edit buttons in the MIDlets page of the Project Properties.

The dialog displays the following fields:

  • MIDlet Name. The displayable name of the MIDlet that the user sees when the MIDlet is run on a mobile device. The combo box lists all available MIDlets.
  • MIDlet Class. The java class that for the MIDlet. The combo box lists all available MIDlet classes of the project and its libraries.
  • MIDlet Icon. An icon (a .png file), representing the MIDlet, that the user sees when the the MIDlet is run on a mobile device. The combo box lists all available PNG images of the project and its libraries.

To add a MIDlet to a suite:

  1. In the MIDlets page of the Project Properties, click the Add button.
    This opens the Add MIDlet dialog. The dialog lists the MIDlets available in the project.
  2. Use the Add MIDlet dialog to enter a name, then select a MIDlet class from the dropdown menu. You can also choose an icon for the MIDlet from the MIDlet icon dropdown menu.
  3. Click OK to close the dialog. The MIDlet is listed in the table.

To Edit a MIDlet in the suite:

  1. Select a MIDlet and click the Edit button.
    This opens the Edit MIDlet dialog.
  2. Use the Edit MIDlet dialog to Edit the name, MIDlet class, and MIDlet icon.
