New WSDL Document Wizard: Name and Location

See Also

The Name and Location page of the New WSDL Document wizard prompts you to supply basic information about the WSDL file.

The basic information includes the file name and the folder where the file will be located. The default values of other fields in the wizard are derived from the file name.

  • File Name. Supply a name for the file. You cannot enter a name that already exists in the project.
  • Folder. If you want to change the default folder, then click Browse and select the new folder. Any WSDL file that is not located in the default folder will not appear in the Projects window. However, the file will appear in the Files window.

You must also specify the target namespace. You can optionally import one or more XML Schema files into the WSDL file.

  • Target Namespace. Enter the target namespace for the WSDL file. The target namespace will appear in the root element.
  • Import XML Schema File(s). If you want to import one or more XML Schema files into the WSDL file, then select this check box. The XML Schema files must already be part of the same project.
  • XML Schema(s). If you selected the Import XML Schema File(s) check box, then click Browse and select one or more XML Schema files.

If you click Next, then the Abstract Configuration page appears.

If you click Finish, then the WSDL file is created. The WSDL Editor appears in WSDL view.
