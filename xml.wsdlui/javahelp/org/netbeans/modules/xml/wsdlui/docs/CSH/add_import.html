Add Import Dialog Box

See Also

You use the Add Import dialog box to import XML schemas or WSDL files into a WSDL file.

This dialog box has the following columns:

  • File. This column provides two ways of viewing the available XML schemas or WSDL files: by file and by namespace. If a file cannot be imported or has already been imported, then the file is crossed out.
  • Import. If you can select the XML schema or WSDL file, then a checkbox appears in this column.
  • Prefix. This column enables you to view and change the prefix that will be associated with the namespace of the XML schema or WSDL file.

The Namespace field displays the namespace of the selected XML schema or WSDL file. This field is read only.
