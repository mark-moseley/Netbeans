Add Import Dialog Box

See Also

You use the Add Import dialog box to import XML schemas and WSDL files into a WSDL file.

This dialog box has the following fields:

  • File. You can view the available files by file name or by namespace.
  • Import. For any file that you can select, a checkbox appears in this column.
  • Prefix.
  • Namespace. Displays the namespace of the selected file. This field is read only.
