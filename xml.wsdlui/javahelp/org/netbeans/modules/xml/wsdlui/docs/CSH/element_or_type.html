Select Element or Type Dialog Box

See Also

You use the Select Element or Type dialog box to specify the type of content for a message part.

This dialog box has the following nodes:

  • By File. Enables you to select an element or type from an XML schema file. If the project does not include an XML schema file, then no elements or types are available.
  • Built-In Schema Types. Enables you to select any of the built-in types in W3C XML Schema. For each type, the base type is shown in parentheses. For example, the NCName type is derived from the Name type. Therefore, the node's label is NCName (restricts :Name).
