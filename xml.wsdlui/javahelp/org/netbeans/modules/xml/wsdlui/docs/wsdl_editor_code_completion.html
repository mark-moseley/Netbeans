Using Code Completion in WSDL Files

See Also

You can use the following types of code completion in WSDL files:

  • Element code completion. When you type an opening left angle bracket (<) inside a parent element, the IDE displays a list of child elements for the parent element.
  • Attribute code completion. When you type the space character inside an element tag, the IDE displays a list of attributes for the element.
