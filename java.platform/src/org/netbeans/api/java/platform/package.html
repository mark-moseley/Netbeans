Representation of Java platform installations.

Each {@link org.netbeans.api.java.platform.JavaPlatform} is one installation of a Java platform, such as J2SE, J2EE, or J2ME. It has a specification name like J2SE and a version such as 1.4. A platform like J2ME may also have associated profiles like MIDP 1.0.

{@link org.netbeans.api.java.platform.JavaPlatformManager} keeps track of installed platforms.
