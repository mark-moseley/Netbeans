Creating the Hibernate Configuration File

See Also

To use Hibernate your application must have the hibernate.cfg.xml configuration file. The hibernate.cfg.xml configuration file contains the database connection details enabling the application to connect to a database. The file also contains information about the location of Hibernate mapping files. Edit this file if you need to change the database connection details.

The Hibernate configuration file is created automatically if you create a web application and you select the Hibernate framework in the New Project wizard.

You can also create this file using the New File wizard if you want to use Hibernate in your application, for example in a Java SE application.

To create hibernate.cfg.xml using the New File wizard:

  1. Open the New File wizard.
  2. Select Hibernate from the Categories list and Hibernate Configuration File from the File Types list. Click Next.
  3. Type hibernate.cfg for the class name. Click Next.
  4. Select a Database Connection from the dropdown list. Click Finish.

When you click Finish, the file hibernate.cfg.xml is created in a source package with the name <default package> and the file opens in the Source Editor.

  • In a web application the hibernate configuration file is also visible under the Configuration Files node in the Projects window.
