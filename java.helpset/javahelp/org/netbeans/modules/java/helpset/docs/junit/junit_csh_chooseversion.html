Select JUnit Version Dialog Box

See Also 

This dialog box lets you specify the version of JUnit to use for creating JUnit test skeletons. You see this dialog box the first time you create a JUnit test or test suite for your project.

If you are developing your project on the Java 5 platform, you can choose between the following versions of JUnit:

  • JUnit 3.x
  • JUnit 4.x
  • To choose JUnit 4, your project must be running on the Java 5 platform or higher.
