Standard Java SE Project Properties Dialog Box: Application

See Also 

The Application page lets you define properties for the presentation of your final application. For example, you can specify an image to serve as the application's splash screen.

You open the Standard Java SE Project Properties dialog box by right-clicking any  and choosing Properties.
