Generating Hibernate Mapping Files and POJOs from a Database

See Also

In addition to writing Hibernate mapping files and POJOs and from scratch, you can also use reverse engineering to generate the mapping files and POJO from a database. You can use the Hibernate Mapping Files and POJOs from Database wizard to generate a hibernate.reveng reverse engineering file from a connected database. The IDE can use the hibernate.reveng file to create the mapping files and POJOs.

To generate Hibernate mapping files and POJOs from a database:

  1. Right-click the project node in the Projects window and choose New > Other to open the New File wizard.
  2. In the New File wizard, select Hibernate Mapping Files and POJOs from Database from the Hibernate category.
  3. Specify the Location for the generated hibernate.reveng file. Click Next.
  4. Select the Configuration File from the dropdown list.
    • The configuration file contains the database connection details. If a connection to the database is not open you may be prompted to supply a username and password to connect to the database. The database must be running.
  5. Select any tables in the left pane and click the Add button. Any tables related to the tables you select are automatically added to the list in the right pane. The IDE will generate Hibernate mapping files and POJOs for each table listed in the right pane.
    • Deselect Include Related Tables if you do not want entity classes created for related tables.
  6. Click Next.
  7. Specify any options for the generated files.
  8. Specify if you want to the IDE to automatically generate the mapping files and POJOs.
  9. Select an existing package from the Package drop-down menu or type the name of a new package.
  10. Click Finish.

When you click Finish, the IDE generates the hibernate.reveng reverse engineering file in the specified location. If you selected Domain Code in the wizard the IDE also generates POJOs based on the reverse engineering file. If you selected Hibernate XML Mappings in the wizard the IDE also generates Hibernate mapping files based on the reverse engineering file.
