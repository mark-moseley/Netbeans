Working with Hibernate

The following table outlines the basic process of working with Hibernate.


Create a Project

  • Create the project.
  • Open a database connection in the Services window.

Configure the Project

  • Create a Hibernate configuration file.
  • Edit the configuration file.
  • Create the Hibernate utility class.

Create Classes

Create the classes by doing either of the following:

  • Create individual POJOs and mapping files.
  • Use the POJOs and Hibernate mapping files from database wizard.

Code Application

  • Create HQL queries.
  • Edit source files.
