New Breakpoint Dialog Box

See Also

The New Breakpoint dialog box enables you to set a breakpoint in your code to stop execution of your program.

To open this dialog box, from the main menu, choose Debug > New Breakpoint.

The fields in the dialog box change depending on the breakpoint type you select from the Breakpoint Type drop-down list.

Method

  • Breakpoint Customizer: Method

Exception

  • Breakpoint Customizer: Exception

Field

  • Breakpoint Customizer: Field

Thread

  • Breakpoint Customizer: Thread

Class

  • Breakpoint Customizer: Class
