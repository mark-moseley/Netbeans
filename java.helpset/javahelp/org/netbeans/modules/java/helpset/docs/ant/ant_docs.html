Installing Ant Documentation

See Also 

The Ant Documentation module installs the Ant manual in your IDE.

To install the Ant Documentation module:

  1. Choose Tools > Update Center from the main window.
  2. Select the NetBeans Update Center and click Next.
  3. Select the Ant Documentation module, click the Add button, and click Next.
  4. Follow the wizard instructions to complete installing the module.

Notes:

  • If you receive the Unable to Connect to the Update Center Server error message, click OK in the dialog box to close it. Click the Proxy Configuration button in the wizard and set a proxy host and port number. Click OK to set the values and continue with your update.
  • The official release of the Ant Documentation module does not always coincide with the release of the IDE. If you do not see the Ant Documentation module on the NetBeans Update Center, it may be posted on the NetBeans Beta Update Center.
