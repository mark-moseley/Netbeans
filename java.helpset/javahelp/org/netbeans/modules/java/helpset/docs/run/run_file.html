Running a Single File

  1. Select the file in the Source Editor or Projects window.
  2. Choose Run > Run File > Run my_class.

    The IDE initiates execution of the selected file.

  • In , this command is disabled by default. To enable this function, you have to write an Ant target for running the file in the IDE and map it to the IDE's Run Class command. This command is not available for EJB projects.
