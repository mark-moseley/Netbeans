Inspector Window

See Also

The Inspector window displays a tree hierarchy of all components contained in the currently opened form. Displayed items include visual components and containers, such as buttons, labels, menus, and panels, as well as non-visual components such as timers and data sources.

To open the Inspector window, choose Window > Inspector from the main menu.
