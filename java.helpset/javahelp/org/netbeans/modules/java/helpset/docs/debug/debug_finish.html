Finishing a Debugging Session

See Also

If necessary, you can interrupt the current debugging session using the Shift-F5 shortcut. You can also close a specific debugging session using the Sessions window.

To finish the current debugging session

  1. Choose Run > Finish Debugger Session (Shift-F5).

To finish one of several debugging sessions:

  1. Open the Sessions window by choosing Window > Debugging > Sessions (Alt-Shift-6).
  2. Right-click the debugging session you want to stop and choose Finish.
