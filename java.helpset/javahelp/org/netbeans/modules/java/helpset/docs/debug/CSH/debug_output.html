Debugger Console and Process Output Tabs in the Output Window

See Also

Once you start debugging, the Output window includes two tabs:

  • Debugger Console tab. Displays messages from the debugger for the current session and includes hyperlinks to the source code.
  • Process (debug) tab. Displays messages, such as compilation and deployment messages, from the process associated with the current session.

You can copy, save, and remove text from the Output window by right-clicking in either tab and choosing one of the following items from the pop-up menu.

Menu Item Description
Copy Copies the selected text to the clipboard.
Find Find a string in the window.
Find Next Find the next occurence of the search item specified in Find.
Wrap Text If selected, the text wraps to the window width. If not, you must expand the window to see text that is wider than the window.
Save As Prompts you for a file where you can save the window contents.
Clear Clears the contents of the selected tabbed window.
Close Closes the Output window completely.
