About Hibernate

See Also

Hibernate is a framework that enables your applications to interact with databasess using Object Relational Mapping (ORM). Hibernate can be used in web applications and Java SE applications

The IDE offers the following support for Hibernate.

  • Wizard for creating Hibernate configuration file
  • Wizard for creating Hibernate mapping files
  • Wizard for creating Hibernate utility class
  • Reverse engineering POJOs and mapping files from a database
  • Code completion when editing Hibernate mapping files
  • Hibernate configuration file visual editor
  • Hibernate Query Language (HQL) editor

Adding Hibernate Support

You can add support to an application by creating the hibernate.cfg.xml Hibernate configuration file. When you create the configuration file the IDE automatically adds the necessary Hibernate libraries to the application. You can create Hibernate configuration file using the New File wizard or by selecting the Hibernate framework in the New Project wizard when creating a web application project.
