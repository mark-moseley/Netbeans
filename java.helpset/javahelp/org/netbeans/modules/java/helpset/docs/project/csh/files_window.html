Java: Files Window

See Also

The Files window shows a directory-based view of your projects, including files and folders that are not displayed in the Projects Window. You can open the Files window by choosing Window > Files (Ctrl-2).

For , the Files window displays the following:

  • Project folder, including output folders
  • Source folder
  • Test folder

For , the Files window displays the following:

  • Project location
  • Project folder, if it is different from the project location
  • Source folders outside of the project location

Context-Sensitive Menu

Each node in the Files window has a context-sensitive pop-up menu of commands that you can run on that node. You open a node's pop-up menu by right-clicking the node. In the Files window, the pop-up menu for project nodes does not contain the major project commands, like Build, Run, and Debug. These commands are only available in the Projects window.
