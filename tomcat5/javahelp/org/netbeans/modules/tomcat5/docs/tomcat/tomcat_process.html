Working with the Tomcat Web Server

The following table shows how to work with the Tomcat Web Server.

Set up the Tomcat Web Server
  1. Register the Tomcat Web Server with the IDE.
  2. View the Tomcat Web Server in the IDE
  3. Do one or more of the following:
    • Customize the Tomcat Web Server
    • Set up a connection pool
    • Write Ant targets for deployment

Deploy to the Tomcat Web Server
  1. Set the Tomcat Web Server as the project's target server.
  2. Do one of the following:
    • To start the Tomcat Web Server automatically, deploy the web application.
    • To start the Tomcat Web Server manually, right-click the server instance in the Runtime window, choose Start/Stop Server, and click Start Server in the Server Status dialog box. Start the server manually when you, for example, use a debug target in a free-form project .
        Click Start Server (Debug) if you want to start the Tomcat Web Server manually in debug mode.
