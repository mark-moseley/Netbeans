About the Tomcat Plugin

See Also

The Tomcat Plugin is an IDE module that handles integration with Tomcat servers.

A Tomcat server is bundled with the IDE and is pre-configured to work with the plugin. The Tomcat server implements the JavaServer Pages 2.0 and Servlet 2.4 specifications and includes many features that are useful in the deployment of web modules and web module groups.

You can also configure the Tomcat plugin to integrate with other supported Tomcat servers. See Adding a Tomcat Server.

To see the Tomcat servers that the Tomcat Plugin manages, expand Server Registry in the Runtime window and expand the Tomcat 5 Servers node.

For more information, see the following web sites:

  • Extensive information about Tomcat web server internals is available at http://jakarta.apache.org/tomcat
  • The JavaServer Pages Specification is available at http://java.sun.com/products/jsp
  • The Java Servlet Specification is available at http://java.sun.com/products/servlet
