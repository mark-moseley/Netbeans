Tomcat 5 Servers Node

See Also

The Tomcat 5 Servers node represents the Tomcat Plugin module. It appears under the Server Registry's node in the Runtime window.

This node shows all the Tomcat servers that the plugin manages, including the bundled Tomcat server that is installed with the IDE.
