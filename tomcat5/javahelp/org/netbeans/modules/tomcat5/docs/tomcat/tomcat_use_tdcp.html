Setting Up a Tomcat Database Connection Pool

See Also

There are many ways of using a data source in a web application. The simplest way is to access your data source directly from a JSP file. You are recommended to use this method for testing purposes and in small web applications only.

  1. Modify the JSP file to enable the selection of database records. Make sure to specify the same JNDI name as specified in the data source above. To do so, paste or type code similar to the following below the taglib directives in a JSP file: 
        	       <sql:query var="queryresults" dataSource="jdbc/poolDB">
                       SELECT * FROM employee_data ORDER BY upper(f_name)
        	       </sql:query>

  2. Make the database driver available to Tomcat connection pooling by copying the database driver's JAR file into Tomcat's common/lib folder. By default, this folder is within the C:\Program Files\netbeans-4.0\nb4.0 folder.

    If you have already started the Tomcat Server, make sure that you restart it after you copy the database driver's JAR file to Tomcat's common/lib folder, so that the Tomcat Server can load the JAR file.
