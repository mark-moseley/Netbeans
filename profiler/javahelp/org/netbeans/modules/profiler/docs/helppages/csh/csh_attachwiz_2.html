Attach Wizard: Review Attach Settings

See Also

The Attach Wizard helps you configure the settings for profiling an application using the attach method. The attach settings are asociated with the project.

You use this page of the Attach wizard to review the settings you specified in the previous page. If the settings are correct, do one of the following:

  • Click Finish if the application or remote JVM is already configured to accept attachment and you are familiar with the instructions.
  • Click Next if you are not familiar with how to configure the application or remote JVM to accept attachment. The manual integration instructions describe the steps necessary to prepare the application or remote JVM so that you can attach the IDE.
