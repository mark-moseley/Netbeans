Attach Wizard: Manual Integration

See Also

The Attach Wizard helps you configure the settings for profiling an application using the attach method. The attach settings are asociated with the project.

You use this page of the Attach wizard to specify the Java platform for the target application or server. When you choose the platform you receive instructions for configuring the target platform so that you can attach the IDE.

  • To profile an application on a remote machine you need to generate a Remote Pack and add a startup option when starting the remote application.

To complete the wizard and attach the IDE, perform the following steps:

  1. Select the Java platform from the drop-down list.
  2. Read and follow the steps described in the pane to configure and start the remote JVM or server on the generated Profiler Remote Pack.
  3. Click Finish to exit the Attach Wizard.
  4. Select a profiling task in the Select Profiling Task dialog box and click Attach.

After you click Attach, the IDE will attach to the remote JVM and the application will start.
