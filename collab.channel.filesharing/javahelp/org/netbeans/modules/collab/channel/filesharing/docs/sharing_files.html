Sharing Files

See Also

You can use the Developer Collaboration feature to share files with other developers within a conversation. Files are shared using the Shared Files collaboration channel visible within each conversation window.

  Files shared within a conversation are separate from the files you keep and edit locally. If existing local files are shared, copies of these files are made and these copies are shared instead of the originals. This ensures that no matter what happens during a conversation (accidental editing, network failure, and so on), the original files are always safe.

To share files:

  • To share an existing file, drag or copy the file into the File Sharing channel of a conversation.
  • To share a new file, select a folder node in the File Sharing channel of a conversation and use the contextual menu or File > New to create the desired file.

The files are sent to the other conversation participants automatically.

  When other participants share files, their files appear in your File Sharing channel after a short delay. Each file shows the owner, or person who originally shared the file, after the file name.
Most files shared within a conversation can be collaboratively edited in real time. In general, if the IDE can edit the content of the file locally, it will be editable once shared. See Editing Shared Files for more information.

To delete a shared file or folder:

  • Select the file or folder node and use the Delete key or the contextual menu to remove the file or folder.
    You can only delete a shared file or folder if you are the owner of the file or folder.

To save changes made to a shared file:

  • Drag or copy the shared file from the File Sharing channel back to a local filesystem.
  When a conversation is ended (all participants leave), all shared files are discarded and are no longer accessible. However, backups of shared files are kept for a time in your IDE user directory under the collab/shared_files folder. The number of conversation backups to keep is configurable in the File Sharing channel options under Tools > Options, in the Collaboration > Channels category.
