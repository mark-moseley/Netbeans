Sharing Form Files

See Also

You can share files that consist of a form in a Collaboration session. When you share form files, only the file owner can make changes; remote file sharers cannot make changes to the form files.

To share a form file:

  1. Double-click the shared file or right-click the file and choose Open from the contextual menu.
    The opened file has an annotation on its document tab to show that it is a shared file.
  2. Begin editing the form file.
  3. When you are finished, click the Save button.
    A message appears asking you whether you want to synchronize files ("Sync Files").
  4. Click OK to allow other participants in the conversation to see the changed form file.
