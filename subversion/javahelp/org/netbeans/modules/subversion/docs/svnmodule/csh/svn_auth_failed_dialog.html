Subversion: Authentication/Connection Failure Dialog Box

See Also

The Subversion: Authentication/Connection Failure dialog box alerts you to any connection failure that occurs when trying to access the repository. It also allows you to verify and possibly modify your username and password, repository URL, and proxy configuration.

This dialog box displays when the IDE attempts to connect to the specified repository (e.g. when you use the Check Out wizard or Import wizard) but cannot do so.

Various reasons for authentication or connection failure can occur, such as:

  1. Incorrect repository URL
  2. Incorrect username or password
  3. Malformed proxy configuration
  4. Modified repository settings
  5. Repository failure
