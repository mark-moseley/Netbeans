Subversion Checkout Wizard: Panel 1

See Also

The first panel of the Subversion Checkout wizard enables you to specify the root path for your Subversion repository when checking out files from, or importing files into, the repository.

You access this panel when you do the following:

  • Checkout a project using the Checkout wizard by choosing Versioning > Subversion > Checkout from the main menu.

When checking out files or importing a project into a repository, you need to specify the location of the repository as defined by the Subversion root. To specify the Subversion root, do the following:

  • Repository URL. Select a Subversion root type and location from the drop-down list. The drop-down list contains a list of previous Subversion root locations used in the IDE.

    • If no root locations are listed in the drop-down list, you can enter the information directly in the text field.

    The IDE supports the following Subversion protocol types:

    • file.file:///repository_path[@REV]
    • http.
    • https.
    • svn.
    • svn+ssh. Requires an external tunnel executable.
  • Password. Enter the password for the user to access the repository.
  • Proxy configuration. Click the Proxy configuration button to specify details of your proxy configuration to connect to the CVS repository.
