Subversion: Select Dialog Box

The Subversion Select dialog box enables you to specify which file or folder you want to use to perform the action associated with the dialog.

You can specify the item by choosing one of the listed files or folders in the panel, then clicking Select.

This dialog box displays when you invoke an action from one of the various Subversion dialogs and windows (i.e., Copy, Switch, Merge, Relocate, Revert, Property Editor) and the IDE identifies multiple files or folders associated with your project that correspond to the node you previously selected.

This can occur if you are working with external libraries (i.e., your project uses a shared libraries folder) or components in a freeform project that are separately versioned (i.e., your project has source roots in different disk folders).
