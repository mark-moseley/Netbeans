Versioning: Open Projects Dialog Box

The Open Project dialog box enables you to open a project in the IDE when you have completed a checkout (CVS, Subversion) or clone (Mercurial).

This dialog box appears when the versioning operation you are currently running successfully completes, and the sources obtained from the operation contain more than one NetBeans project.

The dialog box displays any NetBeans projects that were identified during the operation. Click on a project to select it, or Ctrl-click to select multiple projects simultaneously.

When you click OK, the selected project(s) open in the IDE and can be managed from the Projects window (Ctrl-1).

  • Select the Open Required option to ensure that any sub-projects or dependencies (e.g., libraries or module dependencies) that the selected project(s) require are also opened.
