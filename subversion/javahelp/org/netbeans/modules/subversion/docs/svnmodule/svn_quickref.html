Subversion Tasks: Quick Reference

See Also

The following table describes common Subversion tasks.

To perform this task: Follow these steps:
Check out sources
  1. Choose Versioning > Subversion > Checkout from the main menu.
  2. In the Subversion Checkout wizard, specify the location of the Subversion repository, enter your username, password, and proxy (if required), and click Next.
  3. Specify the folders you want to check out in the Repository Folder(s) field.
  4. In the Local Folder field, specify the location on your computer where you want the checked out sources to reside.
  5. Click Finish to check out the files.
View versioning information
  1. Select a project, directory, or file that is in a Subversion working directory
  2. Choose Subversion > Show Changes from the right-click menu. The Versioning window opens to display versioning information on the selected item(s).
Update local file versions
  1. Select the project, file, or folder that you want to update
  2. Choose Subversion > Update from the right-click menu.
Diff files
  1. Select the file that you want to perform a diff on.
  2. Choose Subversion > Diff from the right-click menu.
Commit local changes
  1. Select the file or directory that you want to commit.
  2. Choose Subversion > Commit from right-click menu.
Merge revisions
  1. Select the files or folders that you want to merge into your local working copy and choose Subversion > Merge Changes from the right-click menu.
  2. Select the branch you want to merge the changes from.
  3. Click Merge.
Create a patch
  1. Select the locally modified file or folder and choose Versioning > Export Diff Patch from the main menu.
  2. Enter a name for the patch file and specify the location where you want to save the patch.
  3. Click Export.
Apply a patch
  1. Select the locally modified file or folder and choose Versioning > Apply Diff Patch from the main menu.
  2. Type the path or navigate to the patch file you want to apply.
  3. Click Patch.
