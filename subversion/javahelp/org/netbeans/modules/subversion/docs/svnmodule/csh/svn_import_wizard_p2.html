Subversion Import Wizard: Repository Folder

See Also

The Folder to Import panel of the Subversion Import wizard enables you to:

  • specify the folder or project to import from your working directory, and
  • designate a target location in the repository.

You access this panel by arriving at the second panel of the Subversion Import wizard (Versioning > Subversion > Import into Repository).

When importing into the repository, specify the following:

  • Repository Folder. Specify a location in the repository by typing in a folder name. Note that you can create a new repository folder by entering a new folder name. By default, the IDE suggests a folder containing the name of your project. Alternatively, click Browse to open a dialog enabling you to view and choose from existing repository folders.
  • Specify the Message. Enter a message describing the project you are importing.
  • Click Next to preview all files that are to be imported and specify additional actions (e.g. exclude individual files from the import), otherwise click Finish to initiate the import.
