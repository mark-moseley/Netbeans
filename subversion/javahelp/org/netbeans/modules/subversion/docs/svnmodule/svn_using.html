Using Subversion in the IDE

See Also

The IDE's Subversion support enables you to manage the evolution of changes in version-controlled files by displaying status information directly in the IDE's various windows (Projects, Files, Versioning, etc.) so that you can call Subversion commands as you work.

Tracking Changes in the Versioning Window

The Versioning window presents a real-time view of the changes in selected directories. It opens at the bottom of the IDE whenever you view changes made in version-controlled files. By default, the Versioning window displays a list of all modified files within the selected package or folder.

You can open the Versioning window by choosing Versioning > Subversion > Show Changes from the main menu. Alternately, you can right-click any versioned project, directory, or file and choose Subversion > Show Changes.

Using the filter buttons in the toolbar, you can limit the list of displayed files to either locally or remotely modified files. The Versioning window toolbar also includes buttons that enable you to invoke the most common Subversion tasks on all files displayed in the list, such as committing, updating, refreshing status, and performing diff's.

The following Subversion commands are available in the Versioning window's toolbar:

Icon Name Function
Refresh Status Refreshes the status of the selected files and folders. Files displayed in the Versioning window can be refreshed to reflect any changes you have made.
Diff All Opens a Diff window providing you with a side-by-side comparison of your local copies and the versions maintained in the repository.
Update All Updates all selected files in the repository.
Commit All Enables you to commit local changes to the repository.

Calling Version Control Commands

You can call Subversion commands on versioned-controlled files (those existing within a registered working directory) using the Subversion menu in the IDE's main window. In addition, many Subversion commands are available in the context-sensitive menu for a selected file or directory.

You can access the Subversion commands context-sensitive menu by right-clicking a file or directory in any of the following:

  • Projects window
  • Files window
  • Favorites window
  • Versioning window
  • Search Histories window
  • Search Results window
  • Source Editor
  • The IDE's status bar, located in the bottom right of the interface, displays the current command's status.
