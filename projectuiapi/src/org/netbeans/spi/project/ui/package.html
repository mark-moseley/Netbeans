Ability to provide a tailored UI for a project.

Normally a project type provider will supply some special UI representation of the project making use of domain-specific features. {@link org.netbeans.spi.project.ui.LogicalViewProvider} permits a project to display itself however it likes in the Projects Explorer tab. {@link org.netbeans.spi.project.ui.CustomizerProvider} permits the project to be customized in an arbitrary way, typically opening a GUI dialog.

{@link org.netbeans.spi.project.ui.ProjectOpenedHook} may be used by projects wishing to take some special action upon being opened in the GUI.
