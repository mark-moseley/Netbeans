Building XSLT Module Projects

See Also

Building an XSLT Module project creates a project .jar file named SEDeployment.jar. Referencing this project .jar in a Composite Application project makes it available as a Service Unit in a deployable Service Assembly.

To build an XSLT Module project:

  • In the Projects window, right-click the project node and choose Build Project from the pop-up menu.
    In the Output window that opens in the lower part of the IDE, watch for the BUILD SUCCESSFUL message. If the build fails, the Output window shows error messages. Rectify the errors and repeat the procedure.
