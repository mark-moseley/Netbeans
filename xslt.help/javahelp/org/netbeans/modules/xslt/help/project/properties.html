XSLT Module Projects Properties

See Also

You open the Properties dialog box for an XSLT Module project by right-clicking the XSLT Module project node and choosing Properties. The tree in the left pane shows that you have access to three pages:

  • General
  • Project References
  • XML Catalog

The General page allows you to view the path of the folder containing the project's files, to view or modify the prefix for the project service engine (such as sun-xslt-engine), and to supply aliases and descriptions for the project's Assembly Unit and Application Subassembly.
