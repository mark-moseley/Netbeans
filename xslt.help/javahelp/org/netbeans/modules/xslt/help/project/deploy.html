Deploying XSLT Module Projects

See Also

XSLT Module projects are not directly deployable. An XSLT Module project is deployed as a JBI component of a Composite Application project.

To deploy an XSLT Module project:

  1. Confirm you have an existing Composite Application or create a new one by following the instructions in Creating Composite Application Projects.
  2. Add the XSLT Module project as a JBI component to the Composite Application project. Follow the instructions in Adding JBI Modules to Composite Application Projects.
  3. Deploy the Composite Application as described in Deploying Composite Application Projects.
