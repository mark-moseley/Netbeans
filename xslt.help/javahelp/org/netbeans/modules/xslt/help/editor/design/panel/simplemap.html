Creating Simple Data Mapping

See Also

Simple data mapping is used when you want to copy the content of a node in the Source tree pane to a node in the Destination tree pane.

To create a simple data mapping:

  • Select a node in the Source tree pane and drag the pointer to a node in the Destination tree pane. An arrow snaps in place to show the data flow.

To delete a simple data mapping:

  • Select the link and press Delete.
