About the Source View of the XSL Transformation Editor

See Also

The Source view of the XSL Transformation Editor enables you to manually edit the XSL stylesheet and provides the following features:

  • Syntax coloring
  • Indentation and reformatting
  • Code completion
  • Navigator window. The Navigator window (Ctrl-7) displays the structure of your XSL stylesheet. The Navigator window displays the name, attributes, and text content of each element. You can double-click any element node to jump to its location in the Source view of the XSL Transformation Editor.
  • Code folding
