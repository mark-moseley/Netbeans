About the Transformations Pane

See Also

The Transformations pane is located in the middle of the Design view and contains a canvas for creating mappings.

For a simple data mapping, the Transformations pane shows data flow arrows from the Source tree pane to the Destination tree pane.

If you add XPath functions to the stylesheet, the Transformations pane shows function boxes. The function boxes have one or several connectors used to map data.
