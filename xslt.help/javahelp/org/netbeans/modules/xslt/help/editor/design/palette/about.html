About the Design View Palette

See Also

The Design view Palette contains a collections of XPath functions. The functions are based on the XPath 1.0 specification.

Each function has zero or more arguments. Each function returns a single result.

The functions are categorized by argument type. The Palette contains the following sections:

  • The Operator Section
  • The String Section
  • The Number Section
  • The Boolean Section
  • The Nodes Section
