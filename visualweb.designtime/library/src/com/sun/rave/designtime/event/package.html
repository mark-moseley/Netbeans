This package is for the event interfaces and adapter base classes of the Creator Design-Time API

Package Specification

The Creator Design-Time API is the 'hook' for component authors to get access to the rich design-time functionality supplied by Creator. Each class or interface is clearly marked as to wether it is to be implemented by the component author or is supplied by the Creator IDE. This package contains the event interfaces and adapters base classes of the Creator Design-Time API.
