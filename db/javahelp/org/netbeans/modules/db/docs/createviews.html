Creating Views

The Database Explorer enables you to create SQL views that are part of the database itself and are available to application programs and other clients of the database.

To create views:

  • Select Add View in the contextual menu for Views (or an existing view). A dialog box appears.
  • Enter the name of the view in View Name. The name is stored in a database.
  • Enter the SQL (SELECT) query into the text area below and click OK. The new view is added to the list under Views.
