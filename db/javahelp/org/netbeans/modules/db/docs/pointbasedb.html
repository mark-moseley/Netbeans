Connecting to a Pointbase Database

See Also

When you install the SJS Application Server, the Pointbase Database Server is installed into the Sun/AppServer/pointbase directory. Two sample databases are installed with the Pointbase installation:

  • sample
  • sun-appserv-samples
To connect to the Pointbase Database Server:
  1. Make sure the Pointbase Database Server is Running by going to Tools  Pointbase Database   Start Local Pointbase Database.
    2. Once connected to the database, you can connect to the database. see connecting to a database you can only connect to one Pointbase database at a time
log / pw pbpublic/pbpublic

If the Pointbase Server is not available, make sure that your Application Server is registered. For more see Registering the server Registering a server In the Runtime window, you will be prompted for your username and password
