Viewing Data

Viewing data is an important function you can perform with the Database Explorer.

To view data of a table or view:

  • Right-click on the node of the table or view.
  • Select View data from the context menu. A dialog with a default SQL command
    to select all data from the table will appear.
  • Edit the SQL command or enter a different command.
  • Press Fetch to run the query.
  • Examine the results in the spreadsheet-like view. Forte for Java remembers all commands you
    enter in a drop-down box for later use.

Each column node has the same View data item in its pop-up menu.
This functionality is similar to the functionality for tables, however,
when you view tables, the SQL query for viewing data fetches only the one column.
