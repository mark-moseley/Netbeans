Viewing Data

Viewing data is an important function you can perform with the Database Explorer.

To view data of a table or view:

  • Right-click the node of the desired table or view.
  • Select View Data from the contextual menu. A dialog box with a default SQL command to select all data from the table appears.
  • Edit the SQL command, or enter a different command.
  • Press Fetch to run the query.
  • Examine the results in the spreadsheet-like view. Forte for Java records all the commands you enter in a combo box for later use.
Each column node has the same View Data item in its contextual menu. This feature is similar to the feature for tables; however, when you view tables, the SQL query for viewing data fetches only the one column.
