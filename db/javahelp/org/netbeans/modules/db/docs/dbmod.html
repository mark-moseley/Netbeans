Executing SQL Commands

See Also

You can execute SQL queries within the IDE on the connected database.

To execute a SQL command:

  1. Expand the Databases node in the Runtime window.
  2. Right-click the node for the connected database and choose Execute Command from the contextual menu.
  3. Enter a new (DDL or DML) command in the Command input box in the new window.
  4. Click Execute to run the Query.

The results are displayed in a table view in the window. The IDE keeps a record of all the commands you enter and presents them in a combo box for your subsequent use.

You can access the Execute Command contextual menu item from any subnode of the Database Connection node.
