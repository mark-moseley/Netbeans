Executing SQL Commands Using the Database Explorer

  1. Right-click the one node under the Connection node and choose Execute from the contextual menu.

    A dialog box with an empty SQL command appears.

  2. Type a new (DDL or DML) command.
  3. Click Execute to run the Query.

    The results are displayed in the spreadsheet-like view. The IDE keeps all commands you enter and presents them in a combo box for your subsequent use.
