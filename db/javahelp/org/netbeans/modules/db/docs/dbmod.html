Executing SQL Commands Using the Database Explorer

You can use the Database Explorer to execute a SQL command. To execute a SQL command:

  1. Right-click the one node under the Connection node.
  2. Select Execute command ... from the contextual menu. A dialog box with an empty SQL command appears.
  3. Enter a new (DDL or DML) command.
  4. Press Execute to run the Query.
  5. Examine the results in the spreadsheet-like view. Forte for Java keeps all commands you enter and presents them in a combo box for your subsequent use.
