Executing SQL Commands

See Also

You can use the SQL Editor to write and execute SQL queries within the IDE. The SQL commmand is executed on the database you select in the Connection drop-down list in the toolbar. The IDE opens a connection to the database if the database connection is closed.

To execute a SQL command:

  1. Expand the Databases node in the Runtime window.
  2. Right-click the node for the connected database and choose Execute Command from the pop-up menu to open the SQL Editor.
  3. Make sure your current Database connection is selected in the Connection drop-down list.
  4. Enter a (DDL or DML) command in the upper pane of the SQL Editor window.
  5. Click Execute SQL (   ) to run the query.

The results are displayed in a table view in the lower pane of the SQL Editor window.

You can access the Execute Command pop-up menu item from any subnode of the Database Connection node in the Runtime window.
