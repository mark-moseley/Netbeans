Documentation Customizer Dialog Box

See Also

You use the Documentation Customizer dialog box to define or modify a documentation sub component of an annotation schema component in an XML schema file. You use the documentation schema component to store text comments in your schema. The documentation component must go inside an annotation schema component.

You access the Documentation Customizer dialog box by selecting a documentation node, right-clicking and choosing Customize from the pop-up menu.

This dialog box has one text box where you enter the text comments.
