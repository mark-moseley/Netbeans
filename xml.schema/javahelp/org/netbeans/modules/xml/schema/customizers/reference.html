Reference Dialog Box

See Also

You use the Reference dialog box to define a reference to a schema component.

You open the Reference dialog box by right-clicking a schema component to which you can add a reference and choosing Add > component Reference from the pop-up menu.

  • The title of the dialog box changes depending on the reference you are adding. The title reflects the referenced schema component: component Reference.
    For example, Element Reference and Attribute Reference.

This dialog box has only one field, a selection box where you can select the schema component you want to reference. This selection box has two nodes:

  • Component node. This node and its contents change depending on the type of reference you are adding: Attributes, Attribute Groups, Elements, or Groups. You can expand these nodes to select a component of that type that exists in your schema.
  • Referenced Schemas node. This node lets you select a component of the type you are referencing that exists in a schema that is referenced in your schema.
