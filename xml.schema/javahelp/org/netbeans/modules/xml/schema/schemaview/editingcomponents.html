XML Schema Editor: Editing XML Schema Components Using the Schema View

See Also

You can edit schema components in the Schema view using customizers and the Properties window.

To edit an XML schema component using a Properties dialog box:

  1. Navigate to and select the node that represents the schema component.
  2. Right-click your selection and choose Properties.
  3. In the Properties dialog box, make your changes and click OK.

To edit an XML schema component using the Properties window:

  1. From the main menu, choose Window > Properties to show the Properties window.
  2. Navigate to and select the node that represents the schema component.
  3. In the Properties window, make your changes.

To edit an XML schema component using a customizer:

  1. Navigate to and select the node that represents the schema component.
  2. Right-click your selection and choose Customize.
    The component Customizer dialog box opens.
    • The Customize option is not available for all schema components.
  3. In the component Customizer dialog box, make your changes and click OK.
