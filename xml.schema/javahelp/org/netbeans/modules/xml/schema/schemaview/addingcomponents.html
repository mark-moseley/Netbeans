When you add XML schema components in the Schema view, a dialog box that lets you customize the new component may open automatically. The following table lists the main components that you can add in the Schema view.

  • For those schema components not listed in the following table, use this step as a starting point: In the Schema view, right-click the node of the schema component where you want to add the new component and choose Add > component from the pop-up menu.
