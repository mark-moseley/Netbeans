Opening XML Schema Files

See Also

You can open XML schema files directly in the Schema view or in the Source view.

To open an XML schema file in the Schema view:

  1. In the Projects window, expand the tree until the .xsd file you want to edit is visible.
  2. Do one of the following:
    • Double-click the file-name.xsd node.
    • Select the file-name.xsd node, right-click and choose Open.

To open an XML schema file in the Source view:

  1. In the Projects window, expand the tree until the .xsd file you want to edit is visible.
  2. Select the file-name.xsd node, right-click and choose Edit.
