Analyzing XML Schemas

See Also

You use the Schema view to perform introspections of XML schemas. You can display a filtered or unfiltered graphical representation of your schema that shows relationships between schema components. For example, you can quickly find where a certain global component is reused in different schema files. You can identify possible problems in a schema by using some of the predefined queries.

Using XML schema tools, you can:

  • Find usages of a schema component
  • Find unused global components in an XML schema
  • Find derivations of a global complex type in an XML schema
  • Find substitution groups in an XML schema

When you execute a query, the status bar provides information about the query results.
