About XML Schema Tools

See Also

The XML schema tools available in this release of NetBeans Enterprise Pack provide tools for editing, visualizing, and refactoring XML Schema, WSDL, and BPEL documents. The XML schema tools focus on complex information display and real-world use issues (for example, scalable visualization and editing of massive XML schemas). In addition, the XML Schema Editor reduces the complexity of creating and editing XML schemas, thus allowing someone who is not a schema expert to create and modify XML schema and other XML documents. Using XML schema tools, you can reference external schemas and run advanced queries to analyze your schemas.

Tasks

Using the XML Schema Editor, you can:

  • Create and open XML schema files
  • Add XML schema components using the Schema view or using the Design view
  • Edit XML schema components using the Schema view or using the Design view
  • Analyze XML schemas
  • Apply design patterns to an XML schema
  • Reference external schemas
  • Retrieve XML resources
  • Use code completion in XML files constrained by an XML schema
  • Use refactoring to rename or delete XML schema components
  • Validate XML schema files

Views in the Source Editor

When you open an XML schema file in the IDE, the Source Editor opens that file in a tab that contains several views. The Source Editor tab for the schema file has three buttons that you use to access the different views: Source, Schema, and Design.
