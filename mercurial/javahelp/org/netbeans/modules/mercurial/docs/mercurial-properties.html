Setting Mercurial Project Properties

Open the Mercurial Properties Dialog:

  • Choose Team > (Mercurial >) Properties from the IDE's main menu, or
  • Choose Mercurial > Properties from the right-click menu of a Mercurial-versioned project.

The following table describes the Mercurial properties that can be set.

Property Comment
default-pull Specify the local repository path to Pull changes from when choosing Mercurial > "Pull from - default" from the Mercurial controled project's right click menu.
default-push Specify the local repository path to Push changes to when choosing Mercurial > "Push to - default" from the Mercurial controled project's right click menu.
username Set the username to record when commiting changes to the repository. Will override the global Mercurial User Name Option for this project.

Notes:

  • To Push to other external locations choose Team > (Mercurial >) Share > Push Other.
  • To Pull from other external locations choose Team > (Mercurial >) Share > Pull Other.
