About Collaboration

The Developer Collaboration feature in the IDE provides you the ability to work remotely on projects with other developers in real time.

After you create a collaboration account, you can do the following:

  • Login to the collaboration server
  • Establish and locate contacts with whom to collaborate
  • Have a conversation with other developers
  • Share files with other developers
  • Collaboratively edit files you share with other developers in real time
