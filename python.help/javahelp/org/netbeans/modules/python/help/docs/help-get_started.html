Getting Started with Python development

  • Creating a new Python project

    • You can create a new Python project from scratch or use your existing Python sources to create a new project in NetBeans.

    start the New Project Wizard (Ctrl+Shift+N)

    • To create a new Python project from scratch:
      1. File > New Project > Python Project
      2. In the Name and Location page, you have to set the following properties:
        • Project Name. Specifies the display name of the project. This name is also used to create the project folder.
        • Project Location. Specifies the folder where you want to store the project metadata and source code.
        • Set as Main Project. Sets the newly created project as the main project.
        • Create Main File. Specifies the name of the main file.
        • Python Platform. You can select the Python platform to use for your project. (See Python Platforms)
          • When you click Finish, the project is created on your computer and opened in both the Projects window and the Files window
    • To create a new Python project from existing sources:
      1. File > New Project > Python Project with Existing Sources
      2. In the Name and Location page, you have to set the following properties:
        • Project Name. Specifies the display name of the project. This name is also used to create the project folder.
        • Project Location. Specifies the folder where you want to store the project metadata and source code.
        • Set as Main Project. Sets the newly created project as the main project.
        • Python Platform. You can select the Python platform to use for your project. (See Python Platforms)
      3. In the Existing Sources page, you have to set the following properties:
        1. Source Root Folders. Specify the root directory where your existing source lives
        2. Test Root Folders. Specify the root directory where your existing tests live
        4. When you click Finish, the project is created on your computer and opened in both the Projects window and the Files window

  • Adding a new Python module. To add a new Python module, Right-Click on the project and select New > Empty Module

    • In the Name and Location page, you have to set the following properties:

    • File Name. Name of the module
    • Project. The project to which this module will be added
    • Folder. The directory in which you want the module to live
    When you click Finish, the module is added to the project with a default template code already added.

  • Adding a new Python package
    • . To add a new Python module, Right-Click on the project and select New > Python Package

    In the Name and Location page, you have to set the following properties:

    • File Name. Name of the package
    • Project. The project to which this module will be added
    • Folder. The directory in which you want the module to live
    When you click Finish, the package is added to the project with a default file already added.
