Using the Stream and Project Filter Operator Property Window

See Also

The Stream and Project Filter operator enables you to obtain a stream as it comes in and do one of three things:

  • Take existing attributes and add other additional attributes
  • Join a static table to an existing set of events to obtain additional attributes or the result. For example, you have information relating to an existing stock and its price, but you want to join a static SEC table to provide CEO and company size information.
  • Add or remove dimensions to an event; that is, provide filtering capability on attributes to add or remove attributes. For example, you have information on a company that includes stock symbol and price but you want to filter out the latter.

To define the attributes for a Stream and Project Filter operator:

  1. Drag a Stream and Project Filter operator from the Palette to the Design area.
  2. Click the Stream and Project Filter operator icon.The Property Editor opens.
  3. Provide the appropriate SQL statements, as below.
  4. Click Ok.

To define the SQL statements to use with a Stream and Project Filter operator:

  1. In the Expression box, enter the expression for the SQL Select statement which is used to specify the input attribute upon which the Select is performed. Add atrribute names, data types. and sizes, as applicable.
  2. In the From box, define the input on which to perform the selection.
  3. In the Where clause box, provide additional filtering on records.
  4. In the Group by box, provide a comma-separated list of attribute names.
