Using the Stream Projection and Filter Operator Property Window

See Also

The Stream and Project Filter operator enables you to obtain a stream and for each event in the input stream, make the event bigger by adding new attributes, or make the events smaller by eliminating attributes, or rename attributes. You can also join an existing set of events with one or more static tables, in which case the output stream can have a different schema than the input stream's schema. For example, you have information relating to an existing stock and its price, but you want to join a static SEC table to provide CEO and company size information--you are, in effect, adding a new dimension to the set of events in the input stream

When you add or remove dimensions to an event; you are using the operator to provide a filtering capability on attributes to add or remove attributes.For example, you have information on a company that includes stock symbol and price but you want to filter out one of these.To do, you can use the Stream Projection and Filter Operator.

To create a Stream Projection and Filter operator:

  1. Drag a Stream and Project Filter operator from the Palette to the Design area.
  2. Click the Stream and Project Filter operator icon.The Property Editor opens.
  3. In the Expression box, enter the expression for the SQL Select statement which is used to specify the value of each output attribute.Add attribute names, data types. and sizes, as applicable.
  4. In the From box, define the input on which to perform the selection which table--
  5. In the Where clause box, provide filtering criteria on records and specify condition of a join if you are doing a join
  6. Click OK.
