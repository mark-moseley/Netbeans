Receiving and Sending Events

See Also

The IEP service engine sends business event notification to any other service engines and binding components installed in the same JBI container. The IEP service engine can also store information into the database to which it is connected.

The IEP service engine can receive from and send data to any business event notification from any other service engines and binding components installed in the same JBI container.

The schema you create in the input and output operators of the specific IEP process define the format of the input and output messages. You can see the XML format of the messages by examining the WSDL that each IEP process automatically generates.
