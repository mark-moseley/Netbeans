Using the Table Operator Property Editor

See Also

The Table operator provides a snapshot of the current state of a relation and a visualization of the relation at a moment in time which can be used as input to an output operator.

To create a Table operator:

  1. Drag a Table operator from the Palette to the Design area.
  2. Click the Table operator icon.The Property Editor opens.
  3. The Details section of the property editor displays the name of the operator and the outpt schema.
  4. The Attributes section provides a picture of the current state of a relation.
