Understanding Intelligent Event Processing Module Projects

See Also

You use IEP Module Projects with the Service-Oriented-Architecture/Composite-Application project system to generate service-engine deployment assembly for event processor deployment.

To create a new IEP Project:

  1. Select the File menu and select New Project.
  2. Select Service-Oriented Architecture from Categories
  3. Select Intelligent Event Processing Module from Projects.
  4. Click the Next button.
  5. Input the name of the Project. Click Finish.

The IDE creates the source file under the Source Packages, opens the source file in the IEP design view, and opens the IEP Palette

You can then drag operators from the Palette to the design view.

After you have saved the project, you can create and deploy a composite application that contains the IEP project.
