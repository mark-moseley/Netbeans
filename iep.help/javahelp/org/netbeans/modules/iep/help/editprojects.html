Editing Intelligent Event Processing Module Projects

See Also

You can edit Intelligent Event Project modules you develop.

To edit an IEP project node using a Property Editor:

  1. After dragging an operator to the Design view, click on the operator. The Property Window opens
  2. In the Properties window, make your changes.
  3. Cick OK.
