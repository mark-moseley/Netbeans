Using the Delete Stream Operator Property Editor

See Also

The Delete Stream operator provides deleted (changed or new) records from a current event triggering relation event.

To create an Delete Stream operator:

  1. Drag a Delete Stream operator from the Palette to the Design area.
  2. Click the Delete Stream operator icon.The Property Editor opens.
  3. The Details section of the property editor displays the name of the operator and the output schema.
  4. The Attributes section provides a picture of the current state of a stream.
