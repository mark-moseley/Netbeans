Understanding WSDL Creation

See Also

Once you save an IEP Event Processor, you generate a WSDL. By default, the WSDL that is created specifies an HTTP Binding for every Stream Input Operator and a File Binding for every Srteam Output operator.

The namespace of the generated WSDL is the Event Processor's name. Once you modify the WSDL manually, the IEP editor asks you to confirm overridding previous changes you have made to the WSDL.
