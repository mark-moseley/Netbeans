Understanding WSDL Creation

See Also

Once you save an IEP Project, you generate a WSDL. By default, the WSDL created specifies a place for an HTTP Binding Component as Input and a File Binding Component as output.

Operator input and output conventions you use when using the IEP Operator Design Canvas are the conventions used in the generated WSDL.
