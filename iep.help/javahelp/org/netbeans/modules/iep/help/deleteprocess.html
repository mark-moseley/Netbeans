Deleting an IEP Process

See Also

You can delete an IEP Process. To do so, you can either:

  • Delete the IEP process from within the Runtime window, or
  • Undeploy the composite application project that is the container for the IEP process.

In addition, deleting the .iep file and the IEP WSDL will delete the process..
