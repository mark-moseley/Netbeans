Using the Stream Output Operator Property Editor

See Also

The Stream Output operator passes individual records into the stream and does not act as a filter or convert stream data that it receives as input. The operator provides the IEP the ability to send notification via a message to a normalized message router for another binding component or service engine.

The IEP uses stream output and WSDL generation to define a web service that another application can implement to receive events from by an event processor. The stream output’s schema is used to generate the XSD of the web service’s input message.

To create a Stream Output operator:

  1. Drag a Stream Output operator to the Design view.
  2. Click the Stream Output operator icon.
  3. Click the Include Timestamp checkbox if you want to include a timestamp on the output from the Stream Output.
  4. Click Ok.
