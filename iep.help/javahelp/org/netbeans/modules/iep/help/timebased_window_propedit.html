Using the Time Based Window Operator Property Editor

See Also

The Time Based Window operator retains all records within the specified period of time.

You can combine the Time Based operator with a subsequent Relation Output operator to provide a timestamp record for the records retained.

To create a Time Based Window operator:

  1. Drag a Time Based Window operator to the Design area.
  2. Click the Time Based Window operator icon.The Property Window opens.
  3. In the Start field, type the amount of time to retain records in numbers in the first Size field.
  4. Choose the units from the pull down menu next to the Size field.
  5. In the Increment field, enter the time increment for the time based aggregator to perform the analysis.
  6. In the Size field, enter the time range within for the time based aggregator to perform the analysis.
