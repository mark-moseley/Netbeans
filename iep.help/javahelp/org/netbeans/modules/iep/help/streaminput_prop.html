Using the Stream Input Operator Property Editor

See Also

The Stream Input operator enables event collection. It enables you to to use the default schema name or to use rename the output data stream schema. A customized schema name can be helpful in enabling you to identify stream origins in complex IEP processes.

To create a Stream Input operator:

  1. Drag a Stream Input operator from the Palette to the Design area.
  2. Click the Stream Input operator icon.The Property Editor opens.
  3. The Details section of the property editor displays the name of the operator and the output schema.
  4. In the Attributes section, add one or more attributes, or data fields.
  5. In the Data Type column in the Attributes section, select the appropriate data type from the list of data types for each attribute you add.
  6. In the Size column, you can optionally specify the size of the attribute.
  7. Click OK to save your edits.
