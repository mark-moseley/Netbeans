Using Property Editors to Customize IEP Operators

See Also

You use property editors to customize IEP operators. To open a property editor:

  1. Double-click on an operator on the design canvas. A property editor opens.
  2. You can add Attribute information in the Property Editor for:
    1. Attribute Name
    2. Data
    3. Size
    4. Scale
    5. Comment
  3. You can also add additional attributes via the property editor.
