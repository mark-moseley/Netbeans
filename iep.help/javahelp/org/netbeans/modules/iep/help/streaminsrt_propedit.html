Using the Insert Stream Operator Property Window

See Also

The Insert Stream operator converts a relation into a stream. The Delete Stream operator converts the deleted records of a changed relation into a stream. The operator takes all records in the current table but not in the immediate table and places them into a stream with a timestamp.

When the insert stream operation is triggered, all of the changed or new records are issued. In other words, the Insert Stream operator passes new records, unlike the Relation Output operator, which issues individual records tagged with either a plus, meaning a new record, or minus, meaning a record was issued previously, but is no longer part of the relation result.

If you want to output data from the IEP process using the Insert Stream operator, connect it to a Stream Output operator.

To create an Insert Stream operator:

  1. Drag an Insert Stream operator from the Palette to the Design view.
  2. Connect it to an operator with a relation output.
  3. Click the Insert Stream operator icon.The Property Editor opens.
  4. The Details section of the property editor displays the name of the operator and the output schema.
