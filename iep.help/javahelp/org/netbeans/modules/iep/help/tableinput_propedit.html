Using the Table Input Operator Property Editor

See Also

The Table Input Operator provides extra event information using a relational table, and also providing an unique endpoint operator that supplies relational table data downstream to other operators.

To create a Table Input operator:

  1. Drag a Table Input operator from the Palette to the Design area.
  2. Click the Table Input operator icon.The Property Editor opens.
  3. The Details section of the property editor displays the name of the operator and the outpt schema.
  4. In the Attributes section, you can add or delete attributes of this table inpu, or move the attributes up or down in order.
  5. In the Data Type column in the Attributes section, select the appropriate data type from the list of data types.
  6. In the Size column, you can optionally specify the size of the attribute.
  7. Click OK to save your edits.
