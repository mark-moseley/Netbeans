Deploying a Composite Application that Contains the IEP Project

See Also

Once you have saved your IEP project, you can create and deploy a composite application that contains the IEP Project.

To create the Composite Application project and add the JBI Module:

  1. Select New Project from the File Menu
  2. Select Service Oriented Architecture from Categories
  3. Select Composite Application from Projects and click Next.
  4. Input the name of the project and click Finish.
  5. Right-click on the folder that contains the Composite Application, and select Add JBI Module.
  6. Navigate to the directory under which you created your IEP project, select the folder containing that project , and click the button Add Project JAR Files.

To deploy a composite application that contains the IEP project:

Right-click on folder containing the Composite Application, and select Deploy Project.
