Deploying a Composite Application that Contains the IEP Project

See Also

Once you have saved your IEP project, you can create and deploy a composite application that contains the IEP Project.

To create the Composite Application project and add the JBI Module:

  1. From the IDE main menu, choose File > New Project.
  2. In the Categories list, select Service Oriented Architecture. In the Projects list, select Composite Application and click Next.
  3. Type the name of the project and click Finish.
  4. Right-click on the folder that contains the Composite Application, and select Add JBI Module.
  5. Navigate to the directory under which you created your IEP project, select the folder containing that project , and click the button Add Project JAR Files.

To deploy a composite application that contains the IEP project:

In the popup window, navigate to the directory where you have your IEP project, select the folder containing the Composite Application, and select Deploy Project.
