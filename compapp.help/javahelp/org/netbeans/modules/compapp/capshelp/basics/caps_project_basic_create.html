New Composite Application Project Wizard: Name and Location

See Also

The New Composite Application Project Wizard steps you through the process of creating a new composite application project. To access the wizard, open the New Project wizard and select the SOA (Service Oriented Architecture) category and the Composite Application project.

The Name and Location page of the wizard prompts you to supply information characterizing the Composite Application project you are creating:

  • Project Name. Supply a name for the folder that will contain your Composite Application files.
  • Project Location. Keep the default path to your other projects, or specify a new one.
  • Project Folder (not settable). The system derives this value by concatenating the previous two entries.
  • Set as Main Project. Leave this check box selected to make this composite application your main project.

When you click Finish, the project is created on your computer and appears in both the Projects window and the Files window.
