Java EE Server Resource Usage

The Resource Usage window contains a list box with occurrences of available resources. Items are listed with a Java filename. Double click a listed item to open an editor that contains the file.

Other server resource functions include:

  • Delete Java EE Module - Select this option to delete the Java EE module from the JBI Modules folder.
  • Deployment Settings - Select this option to open the Deployment Settings window. Uncheck the Deploy with Composite Application checkbox to prevent the deployment of any server resources in the EJB module or web application.
