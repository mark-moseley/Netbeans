Java EE Module Options

When you right-click a Java EE module, listed in a composite application's JBI Module folder in the Projects window, the pop-up menu displays three resource options.

  • Delete: Select this option to delete the Java EE module from the JBI Modules folder.
  • Deployment Settings: Select this option to open the Deployment Settings window. The Deployment Settings window allows you to disable or enable the deployment of server resources in the Java EE module.
  • Verify Server Resources: Select this option to open the Verification Results window. The Verification Results window verifies whether resources used by Java EE module are available in the Server associated with Composite Application project.
