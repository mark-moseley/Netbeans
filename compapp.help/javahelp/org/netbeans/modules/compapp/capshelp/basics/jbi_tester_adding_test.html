Adding/Binding a Test Case

See Also

To add a test case and bind it to a BPEL operation:

  1. In the IDE Projects tab, open the composite application project to expose its Test folder.
  2. Right-click Test, and choose New Test Case from the pop-up menu.
    The Create New Test Case wizard appears.
  3. In the Enter the Test Case Name step, enter a name for the test case and click Next.
  4. In the Select the WSDL Document step, open a module project, select the WSDL file that contains the operation you want to test, and click Next.
  5. In the next step, select the operation you want to test, and then click Finish.
    The test case you created now appears under the composite application's Test folder. The new test case contains two files: Input.xml and Output.xml.
    If you view the test case in the Files tab, you will see Concurrent.properties as a third file.
