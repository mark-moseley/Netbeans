Adding JBI Modules to Composite Application Projects

See Also

To add a JBI Module project to the Composite Application project:

  1. In the Projects window, right-click the Composite Application project and choose Add JBI Module.
  2. In the Select Project dialog box, select a module project folder (BPEL, IEP, SQL, or XSLT), make sure that the project's JAR file has appeared in the Project JAR Files list, and click Add Project JAR Files.
    To verify that the JBI module has appeared, in the Projects window, expand Composite Application project > JBI Modules. You should see the newly added JAR file.
