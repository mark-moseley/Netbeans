Adding JBI Modules to Composite Application Projects

See Also

To add a JBI Module project to the Composite Application project:

  1. In the Projects window, right-click the Composite Application project and choose Add JBI Module.
  2. In the Select Project dialog box, select a module project folder (BPEL, IEP, SQL, or XSLT), make sure that the project's JAR file appears in the Project JAR Files list, and click Add Project JAR Files.

    To verify that the JBI module has has been added to your project, expand the JBI Modules node under your composite application in the Projects window. The new JAR file should now be added to the JBI Modules list.

You can also add JBI Modules as follows:

  • Right-click the JBI module portion of the CASA window and select Add JBI Module from the pop-up menu. The Select Project dialog box appears, from which you can browse to and select the appropriate JAR File to add to your project
  • Drag and drop a project from the Projects window to the JBI module portion of the CASA window.

    Note: From the CASA Editor toolbar, click Build to display all of the project's components in the CASA window.
