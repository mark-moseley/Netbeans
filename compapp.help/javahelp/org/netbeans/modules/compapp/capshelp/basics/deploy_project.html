Deploying Composite Application Projects

See Also

Before you can test-run a business process, make sure that you start the Application Server, build the BPEL module project, and add it as a JBI Module to your Composite Application project.

To deploy a Composite Application project to the JBI server:

In the Projects window, right-click the Composite Application project's node and choose Deploy Project.

The Deploy action compiles the files in the Composite Application project, packages the compiled BPEL and related web service artifacts (.wsdl and .xsd files), and deploys them to the BPEL Service Engine.

To verify that the project has been successfully deployed to the BPEL service engine, open the Service window, expand the JBI node, right-click the Service Assemblies node, and choose Refresh. The name of the Composite Application project for the deployed service assembly appears under the Service Assemblies node.
