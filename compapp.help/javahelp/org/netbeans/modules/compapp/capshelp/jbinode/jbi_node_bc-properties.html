About Binding Component Properties

See Also

Binding component properties can be accessed from the JBI/Binding Component node. Right-click a binding component under the JBI/Binding Components node and select Properties to display the components properties in the Properties window, or double-click a binding component to open the a properties dialog box.

Binding component properties are catagorized as:

  • General: lists the name, state, type, and description of the component (uneditable).
  • Identification: lists the component's build number and specification version (uneditable).
  • Configuration: lists the component's runtime property settings.
  • Statistics: lists component statistics that are collected for actions such as endpoints activated, average response time, completed exchanges, and so forth (uneditable).
  • Loggers: lists the component activities that can be recorded by server.log. Logger properties allow you to specify the level of logging for each event, from FINEST (trace information for debugging), to SEVERE (task cannot continue), as well as OFF.
