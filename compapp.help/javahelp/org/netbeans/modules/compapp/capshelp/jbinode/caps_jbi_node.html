JBI Node (Under Runtime/Servers)

See Also

The JBI node holds all installed or deployed JBI components in one of the following subfolders:

  • Service Engines: Perform transformations.
    You can use the folder's pop-up menu to install a new Service Engine or to refresh the current list.
  • Binding Components: Connect to external systems.
    You can use the folder's pop-up menu to install a new Binding Component or to refresh the current list.
  • Shared Libraries:Components that provide common code shared by multiple JBI components deployed on the JBI meta-container. Shared libraries are required by some SEs and BCs to function properly.
    You can use the folder's pop-up menu to install a new shared library or to refresh the current list.
  • Service Assemblies: Deployable SOA applications.
    You can use the folder's pop-up menu to deploy a new Service Assembly or to refresh the current list.
