JBI Node (Under Services/Servers)

See Also

The JBI node holds all installed or deployed JBI components in the following subfolders:

  • Service Engines: Perform transformations.
    From the folder's pop-up menu you can install, or install and start, a new service engine, or refresh the current list.
  • Binding Components: Connect to external systems.
    From the folder's pop-up menu you can install, or install and start, a new binding component, or refresh the current list.
  • Shared Libraries: Provide common code shared by multiple JBI components deployed on the JBI meta-container. Shared libraries are required by some service engines and binding components to function properly.
    From the folder's pop-up menu you can install a new shared library or refresh the current list.
  • Service Assemblies: Deployable SOA applications.
    From the folder's pop-up menu you can deploy, or deploy and start, a new service assembly, or refresh the current list.
