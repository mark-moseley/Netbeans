Service Engines Node (Under Services/Server/JBI)

See Also

The Service Engines node is a folder containing all JBI service engines that are currently installed to this server. A service engine is the JBI component type that performs data transformations.

The pop-up menu of each Service Engines subnode allows you to start, stop, shut down, or uninstall it.
