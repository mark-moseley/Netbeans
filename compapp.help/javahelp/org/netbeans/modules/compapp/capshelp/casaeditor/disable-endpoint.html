Disabling and Enabling Endpoints in an Application

See Also

The endpoint property, ConfigEndpointExtension, allows you to disable or enable a specific endpoint in an application. In the CASA editor, when the ConfigEndpointExtension property's disableInBC check box is selected in the endpoint properties, that endpoint will not be activated the next time the composite application is deployed.
