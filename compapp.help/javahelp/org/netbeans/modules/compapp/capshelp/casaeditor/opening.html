Opening the CASA Editor

See Also

To open the CASA Editor:

  1. In the Projects window, select and expand the project node of a Composite Application project.
  2. Double-click the Service Assembly node under the project node, or right-click the Service Assembly node and select Edit.
    The CASA Editor opens with the Design View as the active view in the editor.

    Note: You must build the project before the components are displayed in the CASA editor (see Building a Composite Application).
