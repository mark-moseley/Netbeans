CASA Editor: About the Design View

See Also

You can use the Design view of the CASA editor to edit the deployment configuration for a JBI Service Assembly. When you open the CASA editor, the Design view is the default view.

The Design view has the following parts:

  • Editor toolbar. The Editor toolbar is located at the top of the view, just below the tab for the casa file. In the Design view, this toolbar has two navigation buttons: the Source and Design buttons let you switch to those views of the casa file. In addition, you can use the Auto Layout button,   to let the IDE rearrange the information in the Design view. You use the Build Project button,   to build the current project.
  • Design area. The design area is divided into three panes: WSDL Ports, JBI Modules, and External Modules.

Palette

When the Design view is active, the Palette contains three sections: WSDL Bindings, Service Units, and Endpoints.
