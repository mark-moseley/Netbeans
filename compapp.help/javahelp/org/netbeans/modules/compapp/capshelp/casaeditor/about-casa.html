The Composite Application Service Assembly (CASA) editor allows you to edit the deployment configuration for a JBI Service Assembly from the Design view of the CASA editor.

You can perform the following tasks:

  • Open the CASA editor
  • Add JBI modules
  • Build a project
  • Create connections
  • Check WSDL endpoint properties
  • Add external service units
  • Remove connections
