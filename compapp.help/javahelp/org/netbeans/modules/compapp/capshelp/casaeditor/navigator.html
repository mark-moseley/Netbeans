Viewing and Editing the Composite Application in the Navigator Window

See Also

The Navigator window provides three separate views of a composite application:

  • Logical View
    Provides a hierarchical view of WSDL Ports, Service Units, and Connections.
    Elements selected in the Logical View are also selected in the Design Area of the CASA Editor.
    When you select an element, the Properties window displays the properties of that element.
  • XML View
    Provides a tree view of the XML representation of a composite application contained in the application's .casa file.
  • Satellite View
    Provides an overall view of the Design Area, highlighting the visible portion. You can drag the visible portion selection to navigate through the diagram.
