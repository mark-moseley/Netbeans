Building a Composite Application

See Also

When you build a composite application, you create a Service Assembly. The Service Assembly contains Service Units that represent the processing logic for the run-time Service Engines or Binding Components. After building the composite application, you can deploy the application to the application server.

You can build a composite application from within the CASA Editor or by using the JBI Manager in the NetBeans Project window.

To build a composite application in the CASA Editor:

  1. From the Editor Toolbar, select the Build Project button.
    If this is the first time the composite application has been built, the CASA Editor arranges the components and their connections in the Design Area of the editor.

    Note: The build process from the CASA Editor is identical to building the composite application using the JBI Manager in the Projects window.
