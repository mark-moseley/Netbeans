About Composite Applications

Using Composite Application projects, you can assemble the business logic of several project types into one composite application targeting the Java Business Integration (JBI) server. The project system allows you to easily:

  • Assemble an application utilizing multiple project types, such as BPEL, XSLT, and others.
  • Configure external or edge access protocols, such as soap, jms, smtp, and others.
  • Build JBI deployment packages.
  • Deploy an application image to a target JBI server.
  • Monitor JBI server components and applications status.
  • Define and run test cases.
