About Cross-Project XML File References

See Also

The IDE supports cross-project XML file references that are live, constant, and resolvable at design time and deployment time. XML file references means references to XML Schema and WSDL files. Cross-project means that you can reference files between IDE projects.

BPEL Module Project and XSLT Module Project Referencing

BPEL Module projects and XSLT Module projects can cross-reference files in the following types of projects:

  • BPEL Module projects
  • EJB Module projects
  • Java Application projects
  • SQL Module projects
  • Web Application projects
  • XSLT Module projects
