BPEL Designer: Switching Between Design and Source Views

See Also

The BPEL Designer synchronizes the Design and Source views as follows:

  • Changes you make in a diagram are reflected immediately in the corresponding source code.
  • Changes you make in the source code are reflected on the diagram when you switch to the Design view.
  • Changes are synchronized when you save either the source code or the diagram.

To view the code for the diagram:

  • When a diagram is open in the Design view, click the Source button in the Editor toolbar.
    You can also press Alt-Shift-Right and Alt-Shift-Left to switch from one view to another.

To save changes in either the diagram or the source:

  • Choose File > Save or press Ctrl-S.

You can erase a series of last changes you have in both the Design and Source views. Use the Redo command to reverse a series of Undo commands.

To undo a change:

  • Choose Edit > Undo (Ctrl-Z).

To reverse an Undo:

  • Choose Edit > Redo (Ctrl-Y).
