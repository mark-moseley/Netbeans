BPEL Designer: Operations with Elements in the Design View

See Also

In the Design view, you can add, reorder, rename, and delete elements on your business process diagram, configure the properties of elements, and create message flow links.

  • To invoke actions for some elements, such as adding variables, correlation sets, and event handlers, you can use the quick action buttons which appear when you select an element on a diagram.

To add elements to a diagram:

  • Drag element icons from the BPEL Designer Palette and drop them on the target place in the Design view.
    The BPEL Designer shows placeholders to mark the spaces where you can insert a new element.
  • You can add elements inside containter elements also by using the Add from Palette action in the pop-up menu.

To edit the name of an element:

  • Do one of the following:
    • Double-click the name of an element on the diagram and type in the new name.
    • Select an element on the diagram and in the Properties window, place the cursor in the Name row and type in the new name.
    • Select an element on the diagram and press F2 to change the name on the diagram.

To view the source code for an element:

  • Do one of the following:
    • Right-click an element on the diagram and choose Go to Source (Alt-O).
    • In the BPEL Logical View of the Navigator window, right-click an element and choose Go to Source.

To reorder elements on a diagram:

  • Drag and drop elements to a new place on a diagram.
    The BPEL Designer shows placeholders to mark the spaces where you can move elements.

To remove an element from the diagram:

  • Do either of the following:
    • Right-click the element and choose Delete from the pop-up menu.
    • Select an element on the diagram and press the Delete key on the keyboard.
    • In the BPEL Logical View of the Navigator window, right-click an element and choose Delete from the pop-up menu.
  • You can undo one or several delete operations by choosing Edit > Undo (Ctrl-Z).
