BPEL Designer Navigator: XML View

See Also

In the XML View of the Navigator window, you can navigate to a certain line of code and select whether to hide or show the attributes of elements and their contents.

To navigate to a line of source code:

  1. Open a BPEL file (.bpel) in the Projects window in any of the views, Design or Source.
  2. In the Navigator window, select XML View from the drop-down list.
  3. In the Navigator, double-click the element you want to navigate to.
    The line of source code is selected and the cursor is placed at the beginning of this line in the Source view.

To hide or show the attributes or content for elements:

  1. Open a BPEL file and switch to the XML View in the Navigator.
  2. Right-click inside the Navigator window and specify the following options:
    • Show Attributes. Displays the attributes of elements.
    • Show Content. Displays the text content of elements.
  • You can also choose the Show Attributes of XML Tags and Show Content of XML Tags commands using the Filters icons at the bottom of the Navigator window.
