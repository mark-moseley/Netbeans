Editing BPEL Module Project Files

See Also

To edit different types of files, you use different editors:

  • To edit .bpel files, use the Design view of the BPEL Designer
  • To edit .wsdl files, use the WSDL Editor in the IDE
  • To edit .xsd files, use the XML schema editor in the IDE
