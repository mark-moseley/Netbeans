About the BPEL Designer Navigator

See Also

The Navigator window provides a structured view of the currently opened file and simplifies navigation and work with business process diagrams. The Navigator presents a BPEL process in the following views:

  • XML View. This view of the BPEL source code is standard for all XML documents opened in the NetBeans IDE. From this view, you can navigate to a specific place in your BPEL source code. You can also choose to have this view display the attributes and text content of XML tags.
  • BPEL Logical View. Provides an overview of the BPEL process and its structure. In this view, you can perform a variety of operations on BPEL activities, variable, and correlation sets.

If the Navigator window is not open, choose Window > Navigator or press Ctrl-7 to open it.
