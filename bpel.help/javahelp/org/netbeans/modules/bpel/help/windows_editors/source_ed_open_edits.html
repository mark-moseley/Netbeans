Opening Source Files

See Also

By default, BPEL source files open in the Design view of the BPEL Designer, WSDL files open in the WSDL view of the WSDL Editor, and XML schema file open in the Schema view of the XML schema editor.

To open files in the Source view:

  1. In the Projects window, expand the BPEL Module project and the Process Files node and double-click the source file (.bpel, wsdl, or .xsd).
  2. Click the Source button in the editor toolbar.
    The editable source code appears in the Source view.
