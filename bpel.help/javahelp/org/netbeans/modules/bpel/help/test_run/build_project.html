Building BPEL Module Projects

See Also

When you build a project, the IDE checks the BPEL source file and packages the BPEL file and related web-service artifacts (.wsdl and .xsd files) into a JAR archive.

To build a BPEL Module project:

  • In the Projects window, right-click the BPEL Module's node and choose Build Project.
    Watch for the BUILD SUCCESSFUL message in the Output window.
  • You can also perform a clean build by right-clicking the BPEL Module's node in the Projects window and choosing Clean and Build Project.
