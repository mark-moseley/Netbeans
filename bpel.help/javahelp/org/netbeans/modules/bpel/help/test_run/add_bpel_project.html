Adding BPEL Module to Composite Application Projects

See Also

Before you deploy a BPEL Module project, you must first add it as a JBI module to a Composite Application project. Therefore, before adding a BPEL project as a JBI module, make sure that you have done the following:

  • Created a Composite Application project.
    For sample applications, a Composite Application project is created automatically when you create a new sample project.
  • Built a BPEL Module project.

To add a BPEL Module project as a JBI module:

  1. In the Projects window, right-click the Composite Application project's node and choose Add JBI Module.
  2. In the Select Project dialog box, select the BPEL Module project folder, make sure that the project's JAR file has appeared in the Project JAR Files list, and click Add Project JAR Files.
    To verify that the BPEL Module has been added as a JBI Module, in the Projects window, expand Composite Application project > JBI Modules. You should see the JAR file of the BPEL Module project.
