Deploying BPEL Processes

See Also

You deploy BPEL Processes to the BPEL Service Engine within the Composite Application projects. To prepare for deployment, make sure that you have done the following:

  • Started the Application Server
  • Created a Composite Application project.
  • Built the BPEL Module project.
  • Added the BPEL Module project as a JBI module

See Deploying Composite Application Projects for instructions on how to deploy Composite Application projects.
