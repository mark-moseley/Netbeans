The For Properties Dialog Box

See Also

You use the For Properties Dialog Box to define the duration for the process to wait. The For expression specifies the duration after which the process will be initiated. It is used for the On Alarm activity of the Event Handler and Pick elements and for the Wait activity.

This dialog box opens when you select For as the alarm type in the Properties window for the On Alarm or Wait elements and click the ellipsis button (...) next to the For parameter.

In this dialog box, enter the number of years, months, days, hours, minutes, or seconds after which the event specified within the On Alarm activity (or following the Wait activity) will be signaled.
