The Until Properties Dialog Box

See Also

You use the Until Properties Dialog Box to specify a specific point in time until the process waits (Wait activity) or when the alarm is to be fired (On Alarm of Pick and Event Handler).

The Until property defines the following:

  • A certain time when the alarm is to be fired (On Alarm of the Event Handler and Pick elements).
  • A certain time until which the process waits and then the subsequent operation is invoked (Wait activity)

This dialog box opens when you select Until as the alarm type in the Properties window for the On Alarm or Wait elements and click the ellipsis button (...) next to the Until parameter.

In this dialog box, specify the exact date and time until when the process should wait or the alarm is to be fired. By default, the fields in the dialog box contain the current system date and time.
