New Variable Dialog Box

See Also

You use the New Variable dialog box to create either an input or output variable for the Invoke, Receive, or Reply activities. This variable is used to store and transport data exchanged with partner services. This dialog box opens when you choose to create a new input or output variable from the Property Editor dialog boxes of the Invoke, Receive, and Reply activities.

  • Name. Specifies the name of the variable. By default, this field contains an automatically generated unique name. Edit the name if necessary.
  • Type. (Read-only) Specifies the variable type. The variable type depends on the partner link operation, which is selected in the Property Editor dialog box for the activity.
  • Scope. Specifies the scope for which the variable is added. If the process has more than one Scope, select the target Scope for this variable from the drop-down list.
