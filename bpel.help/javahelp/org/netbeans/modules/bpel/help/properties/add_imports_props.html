Create New Import Dialog Box

See Also

You use the Create New Import dialog box to import external .wsdl and .xsd files to a business process. This dialog box opens when you right-click the Import node and choose Add Import in the Navigator window.

  • File. Specifies the name of file to be imported.
    • Note that you can add as imports only the files stored in the source folder of the project directory. Thus, you first need to add them to the source folder within the project structure.
  • Namespace. (Read-only) Shows the namespace of the file you specified in the File field.
  • Type. (Read-only) Shows the type of the file you specified in the File field.
  • Note that you can manually enter the relative path to the file to be imported.
