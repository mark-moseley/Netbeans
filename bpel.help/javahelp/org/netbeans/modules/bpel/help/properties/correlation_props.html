Property Editor Dialog Boxes: Correlations Tab

See Also

You use the Correlations tab of the Property Editor dialog boxes to manually add correlation sets to the following activities: Receive, Reply, On Message of Pick, and On Event of Event Handlers. This tab is available in the Property Editor dialog boxes of these activities.

If you don't want to perform the task manually, use Correlation Wizard.

Before adding correlation to any of these elements, ensure a correlation set is defined for the Process element.

To add a correlation set to an element:

  1. On the Correlations tab, click Add.
  2. In the Choose a Correlation Set dialog box, expand the Correlation Sets node, select the correlation set and click OK.
  3. Choose an initiate attribute for this correlation set from the Initiate drop-down list.
  4. (Optional) Add more correlation sets as needed and click OK.

To edit the properties of the correlation set:

  • Select the correlation set in the list and click Edit.
