The Repeat Every Properties Dialog Box

See Also

You use the Repeat Every Properties Dialog Box to define the duration after which an activity will be repeatedly initiated. It is used for the On Alarm activity of the Event Handler element.

The Repeat Every expiration type can be used either along or in combination with either the For or the Until expressions.

  • If Repeat Every is specified alone, the clock for the very first duration starts at the point in time when the parent scope starts.
  • If Repeat Every is specified with the For or Until expressions, the first alarm is fired when the period of time specified by the For or Until expressions expires. After this, it is fired repeatedly at the interval specified by the Repeat Every expression. 

This dialog box opens when you select Repeat Every as the alarm type in the Properties window for the On Alarm element and click the ellipsis button (...) next to the Repeat Every parameter.

In this dialog box, enter the number of years, months, days, hours, minutes, or seconds after which the alarm will be repeatedly fired.
