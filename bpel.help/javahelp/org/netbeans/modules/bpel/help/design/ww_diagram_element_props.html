BPEL Designer: Configuring Element Properties

See Also

After you add BPEL activities to a diagram, you can configure their properties in the following areas:

  • IDE's Properties window
  • Property Editor dialog box (available only for some elements)

Use the IDE's Properties window to configure all BPEL element properties. The contents of the Properties window differs depending on the active element of the process. There are only two permanent fields of the Properties window available for every element:

  • Name. Shows the name of the element.
  • Documentation. Contains comments associated with the element. For more information see Documentation and Report Generation.

To configure element properties in the Properties window:

  1. Choose Window > Properties (Ctrl-Shift-7).
  2. Select an element on the diagram and view its properties in the Properties window.
  3. To edit properties in the Properties window, do one of the following:
    • Click ellipsis (...) buttons.
    • Use drop-down lists.
    • Enter values in the text editor.

To open the Property Editor dialog box for an element, do one of the following:

  • Right-click the element and choose Edit.
  • Double-click the element.
  • Right-click the element in the BPEL Logical View of the Navigator window and choose Edit.
