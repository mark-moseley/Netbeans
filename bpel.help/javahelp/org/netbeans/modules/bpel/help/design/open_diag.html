BPEL Designer: Opening a BPEL Diagram

You use the Design view to create and modify a diagram that represents a business process, and you use the Source view to display and edit the source file for that business process.

To open a diagram in the Design view:

  1. In the Projects window, expand your BPEL Module project's top node and the default Process Files node. The Process Files node contains an arbitrary hierarchy of nodes corresponding to the folder structure under the src directory. Expand the node hierarchy until you find the BPEL file (with the .bpel extension) that you want to edit.
  2. Double-click the .bpel file.
    The diagram opens in the Editing Mode of the Design view.

To display the diagram's source file:

  • Click the Source button to switch from the Design view to the Source view.
    The source code for the diagram appears in the editor.
