BPEL Designer: Changing Message Flows

See Also

You can create new or change existing message flows on a diagram. To do so, use the icon that appears when you select a web service activity on the diagram.

To create a message flow or a flow link:

  1. Select an element from which you want to draw a message flow link.
  2. Click the envelope icon that appears on the left of the selected element and point the flow line over the target element.
  3. Release the mouse button over the compatible target element.
    The elements are now connected by a new message flow.
  • The new message flow replaces any former message flows to or from one of those elements to avoid possible conflicts. The BPEL Designer allows you to create message flows only according to BPEL logic.
  • You can also change a message flow by specifying a different Partner Link and Operation in the Property Editor of the corresponding activity.
