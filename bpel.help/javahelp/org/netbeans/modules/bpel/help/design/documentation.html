BPEL Designer: Documentation

See Also

The BPEL Designer has a feature that allows to create comments (documentation) attached to the elements of the BPEL Process. This documentation will be included in the source code of the BPEL process and later can be extracted and included in a report.

To create documentation:

  • Do one of the following:
    • In Documentation field of the Properties window for the selected element type anything you consider useful.
    • Click on a small book icon near the selected element. A bubble will appear. Here you can write documentation for the element.
    The text will be included in the source code in the source code between tags and placed after the opening tag of the appropriate element.

    To read documentation:

    A small book icon near the element that is not selected indicates that the element has documentation . If you point at this icon with the cursor, a tool tip with the text written for the element will pop up. You can also see element's documentation in Documentation field of the Properties window.
    To get all the documentation of the process as a separate file you can generate a report.
