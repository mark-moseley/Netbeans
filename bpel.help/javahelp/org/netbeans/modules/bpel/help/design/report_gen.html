BPEL Designer: Report Generation

See Also

BPEL Designer has a feature that allows to get a document describing the BPEL Process. This is Report Generation feature.

To generate a report:

On the BPEL diagram toolbar press the Generate Report button.

In the Projects window among the Process files of the BPEL module a pdf file will appear. The report in pdf format contains the name of the process, the diagram, information about imported documents, information about partner links, a list of all variables defined in the process and information about process elements. If the elements have documentation written for them, it will be included in the report.

You can customise the report using Customise Report button on the diagram toolbar. This button provides two options:

  • Check Generate Verbose report check-box if you want to get a detailed report (elements' properties will be included in the report)
  • Check Include only elements with documentation check-box if you want only elements with documentation to be included in the report
