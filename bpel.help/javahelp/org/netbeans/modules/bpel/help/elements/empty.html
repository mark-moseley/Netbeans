Empty

See Also

The Empty activity instructs a business process instance to perform no operation. Use this element to insert an activity that does nothing, for example, when a fault needs to be caught and suppressed or when you need a placeholder activity.

To use the Empty element:

  1. In the Design view, drag the Empty element from the Palette to the diagram.
  2. Select the Empty element on the diagram.
    The Properties window shows the Empty properties (Ctrl-Shift-7).
  3. (Optional) In the Properties window, enter the name for the Empty element.
    You can also double-click the name on the diagram and enter the new name.
