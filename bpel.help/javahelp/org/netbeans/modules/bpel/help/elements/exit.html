Exit

See Also

The Exit activity immediately terminates the execution of a business process instance: either within the process, within a structured activity, or within a fault handler. All currently running activities are terminated without any additional handling behavior.

To use the Exit element:

  1. In the Design view, drag the Exit element from the Palette to the diagram.
  2. Select the Exit element on the diagram.
    The Properties window shows the Exit properties (Ctrl-Shift-7).
  3. (Optional) In the Properties window, specify the name for the Exit element by clicking the ellipsis button (...).
    You can also double-click the name on the diagram and enter the new name.
