BPEL Debugger Windows

See Also

Whenever a debugging session starts, the IDE displays debugger windows below the IDE editing area. The BPEL Partner Links, BPEL Process Instances, BPEL Process Execution, Sessions, Local Variables, windows display information related to the BPEL processes running within the current debugging session.

The Breakpoints and Watches windows are standard IDE debugger windows. They display all breakpoints and watches set in the IDE.

To open a debugger window:

  • In the main menu choose Window > Debugging > window-name (for example, Window > Debugging > BPEL Process Execution).

To switch between debugger windows:

  • Click the tab with the name of the window you want to access.
