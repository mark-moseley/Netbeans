Prerequisites
   
 - InstallShield MultiPlatform 5 (SP3)
 - Apache Ant 1.6.1
 - JDK 1.4.2 (not JDK 1.5 which uses an incompatible classfile format)
 - external unzip utility on $PATH
 - Unix

To build from command line run ant as usual with

 -Dismp.home=<home dir of ISMP5>
 -Dcluster.dir=/path/to/profiler
 -Dinstaller.basename=NetBeansProfiler-dev-20040701

Example:
ant -Dismp.home=/mnt/local/ismp5 -Dcluster.dir=/mnt/local/linux.profiler.cluster.dir \
-Dinstaller.basename=NetBeansProfiler-dev-20040701 build-linux
 
The installers will be generated and placed in the dist/ subdir

As the ${cluster.dir} is platform specific for profiler build for every
platform must be run separately. There is target for every supported platform:
build-windows, build-linux, build-solaris-sparc, build-solaris-x86.
There is no default target.
