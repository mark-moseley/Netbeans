Finding Answers to Your Question

While there is no specific help for this part of the IDE, the following sources might be able to answer your question:

  • To view a list of tutorials, open the following web page:
  • Press F1 to receive information about a related task or view one of the topics listed in the dynamic help window.
  • Browse through the help topics that are accessible when you choose Help > Help Contents:
    • To learn conceptual information, read the About topics. A good place to start is with About Web Applications or About Visual Web Projects.
    • To see detailed steps for performing each major task, read the Working With topics. The best one to start with is Working with Web Applications.
    • To see all the tasks you can perform for each step in the development process, read the Tasks: Quick Reference topics. These topics list all the tasks in alphabetical order in a table. For example, for projects see Visual Web Project Tasks: Quick Reference.
