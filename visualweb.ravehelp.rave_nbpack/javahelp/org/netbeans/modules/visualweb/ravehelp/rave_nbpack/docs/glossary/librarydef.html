library

A library is a collection of JAR files available to a project during compile time and run time. To view the libraries you are using, look in the install-directory/lib/references directory. The IDE provides the following libraries:

  • JavaServer Faces Support
  • Java API for XML-Based RPC (JAX-RPC): provides web services interoperability across heterogeneous platforms and languages.
  • Rowset: provides cross-database connectivity to a wide range of SQL databases and access to other tabular data sources, such as spreadsheets or flat files.
  • Exception Handler: provides exception handling services for web applications.
