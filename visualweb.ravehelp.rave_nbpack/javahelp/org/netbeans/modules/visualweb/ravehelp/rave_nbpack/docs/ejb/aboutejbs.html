About Enterprise JavaBean Components in Visual Web JSF Applications

See Also

An Enterprise JavaBean component (enterprise bean) is a reusable software component that you can easily combine to create Java applications. The IDE enables you to add existing enterprise beans or use sample enterprise beans in your applications. After you add an enterprise bean to the IDE, the enterprise bean appears under the Enterprise JavaBeans node  in the . You can expand the enterprise bean-name node to see its methods.

To use an enterprise bean in a particular application, you must add the enterprise bean to the page.
