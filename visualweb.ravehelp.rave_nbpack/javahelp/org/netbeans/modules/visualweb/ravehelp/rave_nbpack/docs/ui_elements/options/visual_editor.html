Options: Visual Designer

See Also

Use this node to configure and modify general Visual Designer default settings.

You can access this node by choosing Tools > Options on Microsoft Windows or UNIX systems, or choosing NetBeans > Preferences on Macintosh systems, clicking Advanced Options, and then choosing the Visual Designer node.

Properties

  • Default Font Size. Sets the default font size for pages opened in Java EE 5 projects in the Visual Designer. The default value is 16, a common size used in web browsers. For a design experience that more closely matches the runtime rendering of your applications, you can set this value to match what you see in web browsers that your applications are likely to run in.
    • This setting affects only Java EE 5 projects, because the components used by those projects have relative fonts. Projects that use J2SE 1.4 or earlier have components that use absolute fonts, and those components are not affected by this setting.
  • Show Grid. If selected, a grid is displayed in the Visual Designer background.
  • Snap to Grid. If selected, components you add or move in the Visual Designer are positioned on the nearest grid lines. For more information, see Setting the Grid.
  • Grid Width. Horizontal distance in pixels between grid lines in the Visual Designer.
  • Grid Height. Vertical distance in pixels between grid lines in the Visual Designer.
  • Target Browser Window. Select a value from the drop-down list to use as the default size of the target browser window used in the Visual Designer. For more information, see the description of the Target Browser Window pop-up menu item in About the Visual Designer.
