A navigation rule is an individual decision rule that will be utilized by the default NavigationHandler implementation to make decisions on what view should be displayed next, based on the view id being processed.

  • Rule from View. Specifies the view identifier of the view for which the containing navigation rule is relevant. If no "from-view" element is specified, this rule applies to navigation decisions on all views. If this element is not specified, a value of "*" is assumed, meaning that this navigation rule applies to all views.
  • Rule Description. Specifies a textual description of the element it is nested in, optionally flagged with a language code using the xml:lang attribute.
