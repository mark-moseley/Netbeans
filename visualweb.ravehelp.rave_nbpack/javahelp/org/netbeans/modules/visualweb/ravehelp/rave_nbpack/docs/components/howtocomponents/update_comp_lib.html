Updating a JSF Component Library

See Also

If you have added a component library to a project as described in Importing Components, you might want to update the library to a newer version if the vendor produces one.

  • If you have an earlier version of the library installed, you can also revert to that version by right-clicking the project's Component Libraries node and choosing Replace.

To update the component library used by a project:

  1. Open the project-name node in the
  2. Open the Component Libraries node and right-click the library you want to update, and then choose Update.

    When you click the Update menu item, you might see a message saying that no newer version is installed. If you do have a newer version of the library, follow the steps in Importing Components to add it to the IDE, and then choose Update again to update your component library.
