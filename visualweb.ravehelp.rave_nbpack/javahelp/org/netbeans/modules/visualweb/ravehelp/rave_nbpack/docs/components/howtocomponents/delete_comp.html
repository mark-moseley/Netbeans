Deleting a Component From a Visual Web JSF Page

See Also | Tutorials

  1. In the Visual Designer or , select the component to delete.
  2. Either press the Delete key or right-click the component and choose Delete.
  • You can select a form component in the Navigator window and delete it. However, be sure not to delete the original form created for the page. Every page must have at least one form in it.
