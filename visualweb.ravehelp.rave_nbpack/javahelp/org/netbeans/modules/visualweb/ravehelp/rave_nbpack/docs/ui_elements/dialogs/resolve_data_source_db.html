Resolve Data Sources Dialog Box

See Also

The Resolve Data Sources dialog box helps you to resolve a data source problem that occurs when a project uses a data source that references a database connection that is not registered in the Services window of the IDE. You open this dialog box by right-clicking the project-name node in the Projects window and choosing Resolve Data Source Problem.

The Resolve Data Sources dialog box provides you with a list of data sources that do not have a corresponding database connection registered with the IDE. To resolve the problem, select a data source from the list and click Add Connection to add a database connection for the listed data source. When you click Add Connection, the New Database Connection dialog box opens and you can create and register the required database connection.
