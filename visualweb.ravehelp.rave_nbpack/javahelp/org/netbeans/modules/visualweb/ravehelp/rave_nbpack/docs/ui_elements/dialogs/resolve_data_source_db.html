Resolve Data Sources Dialog Box

See Also

The Resolve Data Sources dialog box helps you resolve database problems when you import an existing Java Studio Creator 2 project into the IDE. You open this dialog box by right-clicking the project-name node in the Projects window and choosieng Resolve Data Source Problem.

The Resolve Data Source(s) dialog box provides you with a list of data sources with no database connection. Choose the appropriate name and click Add Connection to continue to the New Database Connection dialog box.
