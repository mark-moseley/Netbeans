Faces Validator Component

See Also | Tutorials

This advanced component is a custom <f:validator> tag that registers a validator by validator id on a UIComponent object.

This tag is analogous to a Faces Converter and is useful in the IDE solely to register a custom external class that implements javax.faces.validator.Validator and has been registered under a validatorId in a faces-config.xml file.

Use this tag only for a custom validator. If you want to use one of the standard validators, open the component's Properties window and set the validator property.
