Adding a Web Service to a Visual Web JSF Page

See Also

Add a web service to the IDE to make its methods accessible. Then add the web service to a page to call the methods in your application.

To add a web service to a page:

  1. In the , expand the Web Services node .
  2. If necessary, expand the node for the web service group.
  3. Drag the web service into the page, or right-click the web service node and choose Add to Page.

    The web service appears in the Navigator window.
