About the IDE's Technology Version Support

See Also

The IDE supports the following versions for web application development. However, the IDE compiles your applications against the J2EE specification supported by your target server. For Tomcat 5.0 and 5.5 and the Sun Java System Application Server, this is the J2EE 1.4 specification.

  • Java EE 5
  • J2EE 1.4
    • JSP 2.0
    • Servlet 2.4
    • JSTL 1.1

    • For additional information on these technologies, see:

    • The JavaServer Pages Specification, Version 2.0 available at

      .

    • The Java Servlet 2.4 Specification available at

      .

  • J2EE 1.3
    • JSP 1.2
    • Servlet 2.3

    For additional information on these technologies, see:

    • The JavaServer Pages Specification, Version 1.2 available at

      .

    • The Java Servlet 2.3 Specification available at

      .
