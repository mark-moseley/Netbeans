Deleting an Enterprise JavaBean Component from a Visual Web JSF Application

See Also

If you do not want to use an Enterprise JavaBean component (enterprise bean) on a particular page, you can delete it from the page. You can also delete set of enterprise beans from the Services window. Once you delete an enterprise bean set from the Services window, you can no longer use its enterprise beans in any page.

  • Note: At this time, NetBeans IDE supports only Enterprise JavaBeans 2.1.

To delete an enterprise bean from the page:

  • In the Navigator window, right-click the enterprise bean and choose Delete.

To delete an enterprise bean from the Services window:

  • In the Services window, right-click the enterprise bean and choose Remove.
