Faces Form Component

See Also

You can drag the Standard JavaServer Faces Form component  from the Palette to the Visual Designer to create a form on a page. The IDE creates a form for you when it creates a new page, so you are not required to add this component to a Visual Web JSF page created by the IDE. This form is for use by Standard JavaServer Faces components that cannot participate in virtual forms.

If you do add a form component to a page, you see the form appear as a selected box and a new form node for your page appears in the Navigator window. You can select the form in the   and then you can drag components onto the form.

You can also right-click the form and choose Properties to display its Properties window.

  • You can select a form component in the Navigator window and delete it. However, be sure not to delete the original form created for the page. Every page must have at least one form in it.
