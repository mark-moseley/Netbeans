Run Query Window

See Also

Use this window to define an SQL query to view the data in a database to test queries or to test operations performed by your application on a database. To open this window, in the , expand the Databases node and the database-name node, right-click a table, and choose View Data.

  • To define and save queries that will be run as part of your application, use the Query Editor.

Some things you can do in this window:

  • Create a new query by typing it in the Query text field.
  • Modify the existing query in the Query text field.
  • Import a query stored in a file. Right-click the Query text field and choose Import File. Select the file containing the query.
  • View previous queries and select one to run again. Right-click the Query text field and choose History. View the list of queries and click Run next to a query to run it again.
  • Determine how the query is run by setting options, such as the SQL command terminator. Right-click the Query text field and choose Run Options.
  • Run the query by clicking Run Query.
  • Change the number of rows returned by selecting a number from the Rows drop-down list.

Within a given window, the database connection is established when a query is first run. The connection stays open until you close the window. Each Run Query window has its own database connection.
