Import Component Library Dialog Box

See Also

The Import Component Library dialog box enables you to add new JavaServer Faces components to the IDE. The components must be provided in a library that the component provider has adapted for the Visual Web Pack.

You open this dialog box by choosing Tools > Component Library Manager and clicking the Import button.

Use the Browse button to find a valid .complib file that contains the components to import, and then click Open to add the library's path to this dialog box.

After you click Open in the file navigator, you see the path to the library in the top field. Below that field, you see the categories that will be used in the Palette for the library's components.

With the library and its categories displayed in this dialog box, clicking OK updates the Component Library Manager with the library information and adds the library to your user directory.
