Options Customizer Dialog Box

See Also

This dialog box enables to customize parameters for list components. Right-click the component in the Visual Deaigner to open the Opstions Customizer dialog box.

You can customize the following options:

  • Select items. Specify which items are selected by default.
  • Allow multiple selections. If the Select items checkbox is checked, you can specify multiple items that are selcted by default.
    • Note: This option is available in components that support more than one selection.
  • Display. Specify the text that appear in the component.
  • Value. Specify the value of the selection.
Click New to add new items to the list of items. Select the itema dn click Up or Down to change the position of items on the list.
