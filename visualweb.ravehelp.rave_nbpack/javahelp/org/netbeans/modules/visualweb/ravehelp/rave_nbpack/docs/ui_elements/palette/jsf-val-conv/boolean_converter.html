JSF Boolean Converter

See Also

Converts between java.lang.String values in your component properties and data types of java.lang.Boolean or the Java boolean primitive type. The converter is usually applied to the component property that is used to display values and to pick up values entered by users, such as the text property of a Text Field component.

An example is binding a JavaServer Faces component to to a database column of type BOOLEAN. When you bind a component, the IDE normally identifies the data type for you and sets the appropriate converter when you establish a binding to the value property.  However, you can also add this converter manually by setting the component's converter property.
