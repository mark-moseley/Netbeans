Setting the Grid for Visual Web JSF Page

When you create components in the they are positioned on a grid. You can determine the size of the grid and whether components move, or snap, to the nearest part of the grid when dropped in the Visual Designer. The following example shows a button snapped to a visible grid.

To set up the grid:

  1. From the main menu, choose Tools > Options on Microsoft Windows or UNIX systems, or choose NetBeans > Preferences on Macintosh systems.
  2. Click Miscellaneous and then click the Visual Web tab.
  3. Type the new values into the corresponding fields.
  4. Click OK.
    • To temporarily turn off the grid while working with components, hold down the Shift key.
To snap an item to the grid:
  1. In the select the item.
  2. Right-click and choose Snap To Grid.
