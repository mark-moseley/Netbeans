Test Web Service Method Dialog Box

See Also

This dialog box enables you to test a web service method is a web service that has been added to the IDE.

To test a web service method:

  1. In the Services window, expand the Web Services node until you see the indiviual methods that are used in the web service.
  2. Right-click a method and choose Test Method.
    The Test Web Service Method dialog box opens.
  3. Select the method from the list and enter the values for the parameters.
  4. Click Submit to test the method.
    The IDE displays the results in a separate window.
