Importing a Set of Enterprise JavaBean Components to a Visual Web JSF Project

See Also

You can import a set of Enterprise JavaBean components (enterprise beans) to a Visual Web JSF project to improve project consistency.

To import a set of enterprise beans:

  1. In the Services window, right-click the Enterprise JavaBeans node and choose Import EJB Sets.
  2. Browse for the EJB set and click Open.
    The EJB set appears under the Enterprise JavaBeans node in the Services window.
