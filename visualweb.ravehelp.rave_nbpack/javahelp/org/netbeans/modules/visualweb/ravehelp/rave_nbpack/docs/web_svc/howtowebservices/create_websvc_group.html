Creating a Web Service Group in a Visual Web JSF Application

See Also

If you are going to create multiple web services, you can categorize the web services in groups. Once you create a web service in one group, you cannot move the web service to another group.

To create a web service group:

  1. In the Services window, right-click the Web Services node and choose Create Group.
  2. Right-click the node that appears and choose Rename.
  3. In the Rename dialog box, type the new name and click OK.

To add web services to a group:

  1. Right-click the group node and choose Add Web Service.
  2. Follow the procedure for adding a web service.
