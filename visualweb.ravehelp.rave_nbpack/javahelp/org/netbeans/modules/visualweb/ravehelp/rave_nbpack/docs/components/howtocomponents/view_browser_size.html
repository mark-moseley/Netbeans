Viewing a Page at Different Resolutions

See Also

You can view how a page in your application will appear in web browsers using various resolutions. When you choose a resolution, the Visual Designer grays out all areas of the page that would require scrolling to view at that resolution. If you have content beyond the target size, the users must scroll to view that content. The exact position where a user would have to scroll is approximate because it depends on the user's web browser and desktop theme.

To view a page at a specific resolution by using the Visual Designer:

  • In the right-click and choose Target Browser Window > resolution.

To view a page at a specific resolution by using the Options window:

  1. From the main menu, choose Tools > Options on Microsoft Windows or UNIX systems, or choose NetBeans > Preferences on Macintosh systems.
  2. In the Options window, click the Advanced Options button.
  3. Scroll down and select the Visual Designer node.
  4. In the properties sheet, choose the resolution from the Target Browser Window drop-down list.
  5. Click Close.
