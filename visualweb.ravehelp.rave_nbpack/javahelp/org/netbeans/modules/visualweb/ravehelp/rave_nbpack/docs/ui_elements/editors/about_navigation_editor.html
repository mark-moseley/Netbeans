About the Page Flow Editor

See Also | Tutorials

The Page Flow editor enables you to link the pages of your application and to create new pages. To open the Page Flow editor, right-click a page in the Visual Designer and choose Page Navigation.

Use this editor to create links between pages so users can navigate from one page to the next in your applications. For example, you can enable a button in one page to display another page when the user clicks the button. To connect the button to another page, open the Page Flow editor, click the icon on the first page to dispaly its components, click the button in the page and hold down the mouse button, and then drag a link to the page the button is to open.

Some things you can do in this editor:

  • Create a link from a component in one page to another page. Click the icon on a page to display its components, and then drag from a button, hyperlink, or image hyperlink component on this page to another page to create a link to the other page. When the user clicks the button, hyperlink, or image hyperlink in the current page, the other page opens.
  • Create a link from one page to another page. Drag from the connector port  on the right of the page to another page to create a link that is not bound to a component.

    You might want to create a link from the page if you intend to use it later and do not want to attach it to a component yet. For example, you might want a button to go to different pages depending on what the user does in the current page. In that case, you could drag from the current page to each destination page to create all the page links, and then provide code in the action method of the button to perform the link based on some conditions.

  • Change the name of a link. Right-click the link, choose Rename, and type the new name of the link.
  • Create a new page. Right-click in the Page Flow editor, and choose New File. Follow the instructions in the New File wizard.
  • Create a new navigation component on a page. Right-click the page and choose Add Button, Add Hyperlink, or Add Image Hyperlink.
