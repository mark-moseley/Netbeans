Previewing a Visual Web JSF Page

See Also | Tutorials

You can do a quick preview of a page without compiling and running the full application. The preview shows the page in the default web browser as it will appear when the application is run.

To preview a page:

  • In the Visual Designer, right-click and choose Preview in Browser.
