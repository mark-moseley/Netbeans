Virtual Forms Dialog Box

See Also

This dialog box enables you to create, delete, and rename virtual forms and change their color codes. You open the dialog box by right-clicking the background of a page in the Visual Designer and choosing Virtual Forms.

  • To change the color associated with a virtual form, click the Color cell and select a color from the list.
  • To rename a virtual form, double click the Name cell and edit the name.
  • You can also create and delete virtual forms by clicking the New and Delete buttons.
