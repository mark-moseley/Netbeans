Specifying a Data Source Name in a Visual Web JSF Page

See Also | Tutorials

In the Visual Web framework, you can specify the name of a data source that you drag and drop onto a Visual Web JSF page. This can be very useful if your project has more than one data source.

To enable this feature, do the following:

  1. Choose Tools > Options.
    The Options Window opens.
  2. Choose Miscellaneous > Visual Web.
  3. Check the Prompt for Data Source Name checkbox.

After the feature has been enabled, whenever you drag and drop a data source onto a Visual Web JSF page, the Specify Data Source Name dialog box opens. Enter the name of the data source and click OK.
